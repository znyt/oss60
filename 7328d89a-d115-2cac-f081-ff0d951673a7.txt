夏夕绾随着夏邦和蓝烟上了车，离开了，这时一辆豪华保姆车停了下来，厉老夫人直接跑了出来，“绾绾？绾绾！”
<br/>
<br/>        刚才厉老夫人在车里看到夏夕绾了，虽然一闪而逝，但是她敢肯定自己没有看错。
<br/>
<br/>        真的是夏夕绾！
<br/>
<br/>        厉老夫人在四个月前听说了夏夕绾不幸去世的消息，她大受打击，连夜就带着人住进了山庙给夏夕绾超度祈福，现在因为孙女厉嫣然的订婚才回来的，哪知道刚过来就看到了夏夕绾。
<br/>
<br/>        夏夕绾没有死！
<br/>
<br/>        “老夫人，你慢一点，千万别摔了。”这时佣人匆匆的赶了过来。
<br/>
<br/>        “外婆，你怎么了，你在找谁啊？”徐少南来了。
<br/>
<br/>        厉老夫人迅速抓住了徐少南的手，“我刚才看到绾绾了，绾绾没有死，绾绾还好好的活着，而且来了帝都城了！”
<br/>
<br/>        夏夕绾来了？
<br/>
<br/>        徐少南四周看了看，没有看到夏夕绾那道纤柔的身影，“外婆，你肯定看错了，夏夕绾怎么可能来这里呢，不过夏夕绾真的没有死，她是帝都夏家小时候走丢的小女儿，前段时间被找了回来。”
<br/>
<br/>        “真的？发生了这么大的事情你竟然不跟我说？”厉老夫人责怪的看着徐少南。
<br/>
<br/>        徐少南很无辜，“外婆，我也是刚得知夏夕绾没有死，而且你在山庙里不许任何人打扰，我都见不到你的面，怎么跟你说啊，不过外婆，夏夕绾来了帝都短短一个月，可是发生了很多有趣的事情，要不要我讲给你听啊？”
<br/>
<br/>        厉老夫人双眼一亮，“那你快讲啊。”
<br/>
<br/>        徐少南将这段时间发生的故事都告诉了厉老夫人，当听着夏夕绾进了A大，不但带领着A大拿了篮球赛的冠军，还带领着A大杀进了联考第五，她本人更是750满分时，厉老夫人用力的拍了拍自己的大腿，只恨这些精彩的瞬间她不在场，“好好好，我没看错绾绾这丫头！”
<br/>
<br/>        接下来徐少南就讲了这两天发生的故事，当听着夏夕绾被几个黑衣男人给掳走了，现在还在医院里，外面各种造她黄谣的事情，厉老夫人气的愤然而起，“真是岂有此理，究竟是谁敢动绾绾丫头，是欺负绾绾丫头没人保护吗？我要认绾绾丫头当我的孙女，等绾绾丫头冠上厉姓，我要她在帝都城横着走！”
<br/>
<br/>        徐少南当即给厉老夫人竖起了一个大拇指，“不行外婆，你认夏夕绾当孙女，我又是你的亲外孙，那我跟她怎么办啊？”
<br/>
<br/>        厉老夫人一巴掌呼到了徐少南的脑门上，“绾绾丫头不喜欢你，你自己看看你的德行，我都瞧不上，以后不许打绾绾丫头的主意！”
<br/>
<br/>        在海城的时候，厉老夫人是想将徐少南推销给夏夕绾的，但是夏夕绾没有那个意思，当时厉老夫人就想认夏夕绾当自己的孙女了，听说夏夕绾要来帝都，她亲自去机场迎接的，谁知道只听来了夏夕绾的噩耗。
<br/>
<br/>        现在得知夏夕绾没有死，厉老夫人立刻就要将这个孙女给认回家。
<br/>
<br/>        徐少南摸着自己的脑袋，他再一次怀疑，他不是亲生的！
