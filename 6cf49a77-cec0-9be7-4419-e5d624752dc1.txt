　　第229章
<br/>　　倒在地上的屈九江一个鲤鱼打挺跃起，他手中紧紧攥着刀柄，强忍着喉头的鲜血，以及心中不可遏制的怒火。
<br/>　　他很想冲过去，与舒雷再战三百回合，哪怕打不过被对方杀死，死，死又何惧。
<br/>　　可他明白，败了就是败了，身为帝国子民，败也要败得有风度，败也要败得体面。
<br/>　　他抱拳拱手，道：“阁下高才，某甘拜下风。”
<br/>　　可舒雷不是帝国人，他既不知道何谓风度，也不在乎什么是体面。他骑在马上哈哈大笑，用铁弓指着屈九江。
<br/>　　“这就是帝国的武状元么，哈哈，这就是帝国的武状元么。”
<br/>　　在场所有人心里都窝着一口气，尤其是那些年轻的士子，恨不能此刻冲上来击败舒雷。可他们也明白舒雷的厉害，但顾及帝国的体面令他们口出恶言都不肯。
<br/>　　“他不是帝国状元，武状元在此。”
<br/>　　唯一有资格动手的是柳轻名，他已经拍马迫来，手中枪直直的扎向舒雷。
<br/>　　“看枪！”
<br/>　　势大力沉，又有奔马助力，这一枪仿佛携雷电之威，即便舒雷也知道不可力敌，掉转马头，拍马就走。
<br/>　　逃，给了柳轻名自信，他迫得更紧，只想追上舒雷，然后击败他。
<br/>　　所有人的心都为柳轻名揪起，大家攥着拳头，心中轻轻说：追，追上他，打败他，甚至……杀掉他。
<br/>　　此时，就算真的杀了舒雷，怕明帝也不会惩罚，最多也只是表面上呵斥两句，心里还会给你记一大功吧。
<br/>　　两匹马围绕着演武场绕圈，柳轻名距离舒雷越来越近，大家惊讶的看到，舒雷竟然在马上站了起来。
<br/>　　他双脚立在马鞍上，身体随着奔马而起伏，战马在狂奔，但舒雷的双脚却踩的很稳。
<br/>　　众人忍不住惊呼：世间竟真有如此骑术。
<br/>　　呼延力抿嘴笑：“一般人才。”
<br/>　　此时两匹马已经迫得近了，马头衔马尾，舒雷竟从马鞍上跃起，扑向柳轻名，直接将柳轻名从马上撞了下来。
<br/>　　他驰马狂奔，在演武场转圈，柳轻名脸色黑得发紫，这时候他真恨不得找个地缝钻进去。
<br/>　　舒雷口中发出呼勒勒的声响，口中笑道：“我夺了帝国状元，我夺了帝国状元！”
<br/>　　呼延力等戎族人面有得色，此刻脸上浮现温和的微笑，呼延力甚至不忘安慰明帝。
<br/>　　“手下人不是很懂规矩，下手不知轻重，还望陛下海涵。”
<br/>　　可这安慰令明帝的心情更加糟糕，包括他，以及文武官员，围观的士子都十分难堪。今天本来是想在戎族面前演演军武，让戎族也知道帝国的实力，然而现在，是帝国知道了戎族的实力。
<br/>　　看来今天的武状元是封不成了，还封什么封，封出来拿出去丢人么。
<br/>　　“呔，蛮人休狂，长安不是你撒野的地方！”
<br/>　　一声高喝，从围观的士子中跃出一人，他快步奔来，从演武场上的兵器架上抽出一支铁枪，以决绝之势冲向舒雷。
<br/>　　明帝微微皱眉，他以为是那个士子看不得舒雷撒野，于是挺身而出。其实场上能敌过舒雷的人并不是没有，明帝身边的高手是多的，甚至可以说大有人在。但是，以他们的身份和舒雷动手并不合适，从士子中出来一人和舒雷战斗是合适的。
<br/>　　可是，连屈九江和柳轻名都不是舒雷动手，此人又能做得什么，上场不还是丢人么。
<br/>　　但程大雷看出来了，这个人不是普通的士子，而是……林少羽。
<br/>　　他果然没走，是啊，以他这样的性格怎么会走呢，这是一个要做大侠的男人。大侠都是不怕死的，他们死也要讨个公道。今天乔装混进来，还是要向明帝讨个公道。但他毕竟是一个帝国人，看不得戎族嚣张，于是他会站出来。
<br/>　　此时金问道也发出一道轻呼：“是他。”
<br/>　　金问道仍记得林少羽给他留下的印象，林少羽当初独守牛角峰时，是他亲自带人去抓的，他惊讶林少羽的武艺，同样钦佩他的意志。这让金问道知道，帝国有些人是不怕死的，如果一个人连死都不害怕，留给敌人的就只有恐惧了。
<br/>　　程大雷已看得出来，一年时间，林少羽的等级已从优秀晋升到顶级，破浪枪练到五叠。当然，柳轻名和屈九江的等级也是顶级，可是，林少羽却是用双脚?过尸山血海的人，柳轻名和屈九江这种温室里的花朵怎么比。
<br/>　　是的，他能胜过舒雷。
<br/>　　可胜过舒雷又如何，程大雷心思电转，想如何救林少羽活下去。
<br/>　　林少羽奔向舒雷，舒雷也纵马奔来，二人相错的时候，林少羽刺出了一枪。
<br/>　　“下马！”
<br/>　　舒雷用手中铁弓砸向林少羽，借着马的冲力这一记力大无比。可林少羽的枪尖就像是长着眼睛一般，刺中了他的手腕。
<br/>　　紧跟着长枪甩在马头上，战马吃痛昂起头，发出一声响彻天地的长嘶，将舒雷掀落马下。
<br/>　　舒雷像一只发怒的老狗扑向林少羽，可一瞬间他就迎来林少羽的层层枪锋，像风吹大山，卷起林海，层层不休。
<br/>　　破浪枪的关键就在力竭处生力，一旦施展出来，就像大海波涛般无休无止。
<br/>　　“舒雷，退，你不是少侠对手。”金问道突然喊了一声。
<br/>　　舒雷撤身抽离战斗，恭恭敬敬的弯下腰。
<br/>　　“你很厉害，我打不过你。”
<br/>　　戎族不是不懂体面，也不是没有风度，只是他们的体面和风度只给他们尊敬的人，也就是……胜利者。
<br/>　　明帝紧皱的眉头舒展开来，脸上有了笑意，这一下面子岂不是都找回来了。
<br/>　　“少年，你叫什么名字，这身好功夫是和谁人学的？”
<br/>　　大家都明白了，明帝是要封赏这少年的意思，日后他借此一步登天也不是没有可能。可程大雷却心急如焚，只想林少羽先蒙混过关，找机会快些走。
<br/>　　可同时程大雷也明白，他既然来了，为什么又要走呢。
<br/>　　果然，程大雷猜的不错，只见林少羽走到高台下，屈膝跪倒，双手举过头顶。
<br/>　　“草民林少羽特来替家父喊冤。”
