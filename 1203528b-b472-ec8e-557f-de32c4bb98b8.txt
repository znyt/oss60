　　尸体！？
<br/>　　罗昊在听到李然话的一刹那，仿佛头顶上炸开了一道响雷，他的耳朵被震得嗡嗡作响。
<br/>　　“你说他们死了！？”
<br/>　　罗昊盯着李然，声音中带着颤抖。
<br/>　　李然默然，轻轻点了点头。
<br/>　　罗昊只觉得自己眼前所看到的一切天旋地转起来，紧接着眼前一黑，失去了知觉。
<br/>　　接连遭遇到“战斧”导弹变轨后的攻击，影子部队和反政府武装势力的追杀围困，罗昊的身体本来就已经达到了极限，现在李然带给他的这一则消息，成了压倒他的最后一根稻草。
<br/>　　……
<br/>　　“尼米兹”级核动力航母，联合行动司令部。
<br/>　　罗昊再次苏醒过来，已经是两天以后的事情了。
<br/>　　罗昊睁开眼睛打量着自己所处的房间，房间内，一个迷彩倩影察觉到他的醒来，立刻快步走到了床边，看着躺在床上的罗昊，脸上顿时露出一抹笑容来，“你醒了！？”
<br/>　　“小熙！？”
<br/>　　罗昊看清楚眼前的佳人后，脸上充满了诧异。然后再次打量了一下自己所在地方，外面有海浪的声音，这里的确是‘尼米兹’级核动力航母的某一间船舱，不是龙隐部队，也不是在京城的别墅。
<br/>　　在云熙的搀扶下，罗昊挣扎着从床上坐起来，对云熙问道：“你怎么会来这里？”
<br/>　　“是凤凰姐让我来的，以这次联合行动的情报小组副组长的身份。”云熙回答道。
<br/>　　罗昊明白，云熙会来这里，一定是宋萱和李然担心自己的情况，才特意通知了云熙让她赶来。如果说这个世界上，还有谁能在这种时候安抚自己的话，除了云熙以外，不会再有其他人。
<br/>　　在外，在部队，罗昊是战术中队队长，是“龙刺”，是行动中队员的中心；但是回到了家，家中那个温柔似水的女人，就是他的港湾。
<br/>　　“先喝点水。”云熙把一杯温水递给罗昊，说道：“你昏睡了两天，刚醒不能直接吃东西，先喝点水，然后再去吃东西。”
<br/>　　罗昊接过云熙递给自己的水杯，捧在双手间，眼睛看着地板，沉默着没有说话。跟罗昊夫妻多年，云熙自然是明白罗昊此时心中在想什么。
<br/>　　良久，罗昊在停下搓动手中水杯的动作，低着头，用低沉的声音对云熙问道：“他们……他们三个，真的，死了吗？”
<br/>　　云熙坐到罗昊身边，柔声说道：“遗体已经运回国了，承影会亲自主持他们的追悼会。”
<br/>　　啪！
<br/>　　罗昊手中的水杯掉落在地上，摔得粉碎，扑在云熙怀中，放声大哭着。
<br/>　　此时他不再是龙隐部队战术中队的队长，也不是龙隐部队的“龙刺”，只是一个在行动中失去了自己战友的军人。这个有着铮铮铁骨，为了自己兄弟可以搭上前程，甚至姓名，不管遇到什么事情都不会轻易落泪的男人，此时却哭得像一个孩子一样。
<br/>　　云熙没有安慰，一只手轻拍着罗昊的后背，另一只手轻拭掉自己眼眶中的眼泪。男人，想哭就哭吧。
<br/>　　清理掉房间里的玻璃碎渣，云熙又给罗昊倒了一杯水，说道：“你两天没吃没喝，先喝点水。”
<br/>　　罗昊接过云熙递给自己的杯子，喝了一口水，对云熙问道：“你到这边来了，家里怎么办？”
<br/>　　“你是说悠儿和小战？”云熙微笑着说道：“他们两个都快成混世小魔王了，谁敢惹这两个小祖宗？他们不去惹别人就算是好的了。”
<br/>　　“小悠儿也跟着小战调皮捣蛋去了？”
<br/>　　罗昊对云熙的话有些意外，在他印象中，女儿云悠总是乖巧的印象，没事就自己拿着魔方在玩，怎么也跟着罗战瞎胡闹去了？
<br/>　　“他们俩啊。”云熙微笑中又透着无奈，说道：“现在一个是预备参谋长，一个是预备大队长；一个出谋划策，一个奉命执行，他们还把萱儿给拉到了队伍中。”
<br/>　　“预备参谋？预备大队长？”罗昊不解地看着云熙，问道：“这又是什么？”
<br/>　　“小战说了，他早晚是要接你的班，不过我们部队没有他那种年龄的队员，所以这小子就自封了一个预备大队长，把悠儿任命成预备参谋长。”云熙笑着解释道：“你还真别说，现在基地里面的人看到他们俩，还给他们敬礼问好，他们这俩小子也不客气，一副领导的样子像模像样。野狼上次来基地视察，看到这姐弟俩，开玩笑让小战做一遍战术演练，结果这小子要跟警通连的进行战术对抗！”
<br/>　　“野狼就从警通连里挑了五个人陪小战进行战术对抗，悠儿负责当参谋，结果你猜怎么样？这姐弟俩愣是把警通连的五个人赢了。野狼一高兴，当即给他们特批了两个小臂章，现在他们俩没事就戴着小臂章到处晃悠。”说起家里那两个孩子，云熙仿佛有说不完的话，有说不完的趣事。
<br/>　　罗昊闻言，哑然失笑，说道：“那是警通连的放水了吧。”
<br/>　　“放水是肯定的。”云熙说道：“但是他们就两个人，即便是在放水的情况下，能赢也是很了不起了。野狼当场把警通连批了一顿，我出来的时候，警通连还在搞体能呢。”
<br/>　　“对了，你刚才说，你是以情报组副组长的身份来这儿的，这又是怎么回事？”罗昊皱眉对云熙问道。
<br/>　　云熙也知道，这件事情是瞒不住罗昊的，所以也就不再隐瞒，说道：“这次我们的人，除了你们五个，还有牺牲的虎鲨、电光、冰枪以外，其余人到目前为止还下落不明，初步怀疑是被西伦科特抓了。不仅是我们的人，其他国家的特种部队也遭到了影子部队的袭击，也都有死伤，这次联合行动可以说非常失败，美国国防部震怒，要求彻查，汉密尔顿被要求回国接受调查，并且让太平洋舰队司令官詹姆斯接替汉密尔顿，对后续的搜索、营救行动进行协调和指挥。”
<br/>　　“这次的事情背后，肯定有内鬼！”罗昊十分肯定的对云熙说道。
