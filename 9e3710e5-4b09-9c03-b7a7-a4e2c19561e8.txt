　　苏沫虽然比之前强大了许多，但而今的叶少阳，早已经不是昔日的他，苏沫根本不是对手，交手了几个回合，被叶少阳一掌弹开，勾魂索也摆脱了柳枝的纠缠，朝上方一挑，啪的一声打碎了一盏灯笼。
<br/>　　“拦住他！”苏沫咆哮，带着一干人又冲了上来。
<br/>　　叶少阳施展茅山凌空步，跟他们玩起了捉迷藏，实在躲不掉的就一鞭子过去，这些人中，不乏梨山和星宿海门下的二代弟子，比实力在人间额算是一流，但跟叶少阳比就差远了，至于那些人间弟子，更不值一提，叶少阳一个人左冲右突，不断将灯笼打灭，期间还伤了好几个人。
<br/>　　不一会工夫，二十四盏灯笼尽数熄灭。
<br/>　　结界的强度一下子降低了许多。
<br/>　　小九带领大家一起发力，总算将结界撕开一个大的缺口，率先跳了进来，这时候叶少阳已经被苏沫一干人逼到角落，小九跳到他身边，一声长吟，喷吐出的妖气有如一阵飓风，苏沫等人连连后退，其余实力差一点，直接被吹飞了，撞进了庙堂里。
<br/>　　苏沫看到小九，也只能恨恨地站住，冲叶少阳怒道：“你若是个男人，就与我单独一战！”
<br/>　　“单挑啊，我来陪你！”
<br/>　　一朵莲花，从裂缝中飘了进来，紧接着只听轰的一声，结界尽数破碎，捉鬼联盟成员们都冲了进来。
<br/>　　苏沫大惊失色，她虽然打起架来像个疯子，但现在还没到拼命的时候，她也自忖不是对手，纵身朝山里逃去，碧清紧追不舍。
<br/>　　“碧清！”
<br/>　　叶少阳试图追过去，林三生冲上来拦住他，道：“大事要紧，先去破了他们的仪式。”
<br/>　　叶少阳咬了咬牙，朝着还在作法的梨山老母一干人冲过去。
<br/>　　梨山老母突然双眼一睁，抽身站起，两袖扇动，动作有点像是京剧里的青衣在抖水袖，突然狂风大作，从她袖子里飞出了漫天花瓣，暴雪一般飞舞起来。
<br/>　　幻灵花海！
<br/>　　黎山老母一出手就是绝招。
<br/>　　霎时间，铺天盖地都是雪花，挟带着一股股的强风，将捉鬼联盟成员尽数卷在中间，花瓣落在身上，如刀割一般疼痛。
<br/>　　“她在拖延时间！冲过去！”
<br/>　　叶少阳看不到人，只听到林三生的声音。但这幻灵花海端的厉害，纵然叶少阳作法护住周身，一时间也摸不到方向，有种寸步难行的感觉。
<br/>　　“少阳！”
<br/>　　小九喊了他一声，尾巴一甩，将他拖到自己背上，朝着花瓣深处飞奔过去，体表妖气笼罩，花瓣落在上面，立刻就枯萎了……
<br/>　　“抱紧我脖子！”
<br/>　　小九嘱咐道。
<br/>　　好熟悉的感觉。
<br/>　　叶少阳搂紧她的脖子，把脑袋埋在她柔软的羽毛里，用力吸了一口气，道：“你身上有一股味道。”
<br/>　　“什么？”
<br/>　　“有点像小猫，我小时候在茅山上养过一只猫，味道就跟你身上差不多。”
<br/>　　小九嗔道：“讨厌，你说我是你的宠物吗？”
<br/>　　“哈哈，我可没这么说。”
<br/>　　突然间，所有花瓣都消失了，两人闯出了花海，小九看到黎山老母就在前面，本想直接撞过去，又看到摇光仙子和秋明子站在边上，虎视眈眈的，自己以一敌三，肯定赢不了，干脆就地站住，叶少阳也从她背上跳下来。
<br/>　　身后小白小青等人也都追上来，站在两人身后，为了闯出幻灵花海，大家都消耗了不少修为。
<br/>　　“九尾天狐，连你也来了，好，好的很！”梨山老母望着小九，冷冷说道，转身看着林三生，道：“林大帅，老身当年力排众议，捧你上位，而今你却恩将仇报，今日打断我祭祀作法，真乃卑鄙小人！”
<br/>　　林三生本想憋着不说话，到时候给他来个死不认账，但听到黎山老母指名道姓地骂自己，道义上站不住，叶少阳等人都不好看，干脆亮出身份，对着黎山老母躬身一拜，道：“圣母恩情，没齿难忘，但圣母当时所为，是为了空界，而今我之所为，又何尝不是？”
<br/>　　黎山老母喝道：“巧舌如簧！”
<br/>　　“圣母，晚生为了空界，鞠躬尽瘁，若不是我，而今空界差不多已尽归尸王之手，这点你可承认？”
<br/>　　黎山老母不说话，这事情实在没法反驳。
<br/>　　林三生接着道：“既如此，晚生并没有辜负圣母的提拔，之后我任联军大元帅，但军帐中却是你们诸位坐镇，什么都是你们说了算，我好不容易发展点自己势力，也是为了维持空界局面，驱除尸族，你们处处掣肘，而今又祭祀朝拜东岳帝君，不也是为了逢迎帝君，好将我赶出空界？”
<br/>　　摇光仙子冷笑道：“便是如此，难道错了？我等在空界多年，一向和平主事，凭什么让你来横插一脚，若你势力做大，还有我们什么好事？”
<br/>　　林三生笑道：“因此说来，大家都是为了自己，也没什么道义上的高下了。诸位放心，今日我们只破坏仪式，不伤人，今日之后，为了空界和平，大家还是要团结……”
<br/>　　说完给叶少阳使了个眼色，意思让他动手。他自己的实力不够，在这三位阐教金仙面前讨不到好，只能让叶少阳动手。
<br/>　　“得罪了啊！”
<br/>　　叶少阳拔出七星龙泉剑，朝着庙堂里的香案冲了过去，目标是庆云金灯，只要打翻这个东西，仪式额就算完了。
<br/>　　黎山老母三人一起上来阻拦，小九抢在前面冲过去，被黎山老母架住，斗起法来。
<br/>　　一边小青也冲过去，挑战摇光仙子。
<br/>　　小青虽然是捉鬼联盟中实力靠前的，但摇光仙子好歹是东岳大帝嫡传弟子、阐教金仙，在空界也是一方宗主，绝不是吃素的，几个回合下来，小青明显不支，小白也冲过去，二对一，一时战局平衡。
<br/>　　其余人，都被逍遥飞和慈心师太这几个大佬架住，慈心师太一边打架，一边嘴里还叨叨着骂着叶少阳，叶少阳真想一巴掌拍死她，但人已经朝秋明子冲了过去。
<br/>　　秋明子手拿一支判官笔，一只手捏诀，胡子眉毛都飘起来。
