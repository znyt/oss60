　　尚九娣的呼喊没有让白允儿有任何反应，而是做出了让所有人都没料到的举动。
<br/>　　她在退开短暂的捂住腹部不让其流血过多后，便又一次猛攻而上，赶在丧尸王木天靠近之前，忽然朝着一击得逞的异种王木叶狠厉的一掌劈下！
<br/>　　两人在猎者联盟总部打过一场，与新交手的丧尸王木天不同，白允儿对异种王木叶的战力和习惯有一定了解，战斗为本能的情况下，她的反击得心应手。
<br/>　　木叶初尝到神秘家族血液的香甜，贪婪让他并没有想过白允儿会在这时候反击，毫无准备之时，再加上白允儿手中那诡异的手套所带的属性，让他哪怕速度再快也避开的尤其困难。
<br/>　　速度好像被放慢了无数倍，只能眼睁睁的看着白允儿的那一劈，一下子让他的胸前裂开一个狭长的伤口！
<br/>　　噗！
<br/>　　一股味道与人类并无太多差异的红色血液流淌而出！
<br/>　　木叶快速后退着，双眼冒着凶狠和报复感，他很清楚如果不是因为白允儿本就一心三用又受伤过重，他和木天任何一人都难以战胜全盛时期的白允儿。
<br/>　　木叶立即逃离的毫不犹豫，身后的木天也在当下快速上前支援。
<br/>　　但这时候白允儿却根本没有对异种王木叶追击，而是毫无预兆的回头，一直没有太多使用的左手，忽然带着呼啸的强大能量。
<br/>　　没有再用她最擅长的速度和最习惯的锁喉，而是换上了最为简单直接的攻击，一拳狠狠朝着丧尸王的胸前轰出！
<br/>　　轰！
<br/>　　一拳打出，娇小的拳头看起来毫无力量，却偏偏让丧尸王冲上去的魁梧身躯猛然顿住，再也难以往前一步。
<br/>　　他的面容一下子扭曲起来，神色表达出的意思也简单，那就是痛苦！
<br/>　　于是在打退了异种王木叶之后，丧尸王木天也被白允儿一拳轰的步步倒退，但白允儿眼神冷峻异常，在木天的双脚第一次往后倒退的刹那。
<br/>　　轰！
<br/>　　又是一拳打出，这一回她用的是右手。
<br/>　　紧接着，双拳的交替，接连不断的轰击在丧尸王木天的胸口。
<br/>　　每一次都用上了十成力！
<br/>　　白允儿的爆发，让两兄弟难以招架，也根本摸不准她忽然变化的战斗风格。
<br/>　　多次的挥拳之后，白允儿终于停下，而此时的丧尸王木天，则已经退到了极其远的地方，与木叶站在了整个平地的最边缘地带。
<br/>　　此时众人只见到，不远处的异种王木天胸口伤势触目惊心，木天的胸口则被白允儿打的黑紫一片淤青，怕是把肉都轰碎了！
<br/>　　两兄弟都在远处喘着粗气，他们不是没有一战之力，只是相对白允儿，没有殊死一战的爆发。
<br/>　　他们都很清楚，那绝对是白允儿仅剩的最后一丝力量！
<br/>　　在白允儿爆发期间，尚九娣正被好几名狼牙防守队成员死死按住，因为她忽然激动异常的想要跑上前，口中不断呼喊着白允儿的名字，泪水更是浸·湿了衣衫。
<br/>　　防守队成员很清楚尚九娣上前就是送死，当下他们只能如此做，只是尚九娣的反应实在太过于反常。
<br/>　　顾凉晨察觉到一丝不对劲，快速走过来问道：“怎么了？”
<br/>　　尚九娣的声音都开始颤抖：“白允儿受伤了。”
<br/>　　“我知道。”顾凉晨有些没明白：“她现在受伤很重，不能再继续下去，我们得想办法……”
<br/>　　“她伤到器官了！”尚九娣忽然打断顾凉晨的话，声音凄厉甚至有些尖锐。
<br/>　　顾凉晨皱起眉，这时候他才察觉到有什么事情不对。
<br/>　　两人的对话并没有可以压低声音，木天和木叶都听得一清二楚，两人在等待着白允儿能量耗尽之时，不约而同的朝着尚九娣看了一眼，而后一抹意味深长的神色便浮现而出。
<br/>　　“人类的繁殖，终究有太多限制。”丧尸王淡淡开口，说出的话一点都不符合他种族的残暴无脑，竟然带着一丝怜悯。
<br/>　　异种王则是阴笑一声，好似报复：“你们这些弱小的种族。”
<br/>　　白允儿在将两人打退后就一直站在原地，此时她抬头看了看身后，那是已经离她颇远的木屋。
<br/>　　她右手颤抖的抬起，捂住了在刚才战斗时又一次裂开流血的小腹，第三处伤口是唯一重伤后不可恢复的地方，哪怕她的恢复能力再好，几个月后小腹表面再如何看不出伤痕，内在却已经完全被废。
<br/>　　现场大部分人也还未反应过来，在这末世纪元时刻都有可能丧生的时代，并没有太多人关心将来以后，能活在当下已经不易。
<br/>　　唯有尚九娣一直心存希望，认为他们一定能等到和平年代那一天，也才能在这时候注意到这不经意间的细节。
<br/>　　人类无论男女除了心脏和脑袋这两大致命部位之外，还有着诸多其他可以导致直接死亡的致命点，但却还有一处地方受了伤不至于死亡，却又与其他部位不同的再也难以恢复如初。
<br/>　　那便是生育！
<br/>　　刚才那一击，木叶非常巧合的破开了白允儿肚脐下方的位置，他本想直接一抓刺穿白允儿整个腹部，让其直接死亡。
<br/>　　无奈白允儿的速度太快，在受伤的第一时间就躲开了后续刺入的攻击，但木叶还是深深刺入到了其内脏器官，也是身为人类女性生命，唯一受伤后无法治愈的部位。
<br/>　　这是末世纪元中的又一新法则，加之本就大大降低的出生率，导致人类的数量急剧减少，每年出生的新生儿少之又少。
<br/>　　拥有神秘家族血脉的白允儿，在血脉限制之下，本就属于生育率极其低下的一类，这时候又在战斗时不慎被伤到。
<br/>　　也就是说，白允儿从此便失去了生育能力，再无有后代的可能……
<br/>　　哪楚涵身边的这几个女人，一直相互间暗斗个不停，尤其是步纱还杀到了白允儿面前一较高下，但在大事上她们却又诡异的想法一致。
<br/>　　白允儿以一己之力拦住了这么强大的敌袭，一而再再而三的拼死抵挡，重伤不退也要死守木屋，哪怕已经没有继续战斗的能力，也还是只身一人挡在两大强敌面前。
<br/>　　甚至还付出了如此严重的代价！
