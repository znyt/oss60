　　悟剑峰中。
<br/>　　楚风眠从北邙宫之中回来，便是一直在悟剑峰中，参悟剑术。
<br/>　　在北邙宫中的一战，尤其是跟夜皇等人的交手，对楚风眠受益匪浅。
<br/>　　这还是楚风眠第一次跟如此实力的强者交手，对楚风眠的剑术领悟，有着极大的帮助，还有剑皇殿之中记载的无数剑术，也都被楚风眠所领悟，融入到九域剑术之中去。
<br/>　　一日，两日，三日……
<br/>　　在这悟剑峰之中，楚风眠一座便是已经度过了七日的时间，在这七日时间中，楚风眠一直都是在潜心修炼这他的剑术。
<br/>　　在楚风眠身上那爆发的气势，剑意，也是已经达到了顶点，逐渐的开始沉寂了下来。
<br/>　　虽然剑意沉寂下来，但是现在楚风眠身上气势，比起刚刚，还要恐怖的多。
<br/>　　“少主，您苏醒了？”
<br/>　　楚风眠刚刚睁开双眼，在楚风眠的身边，那金色灵剑，剑皇殿的器灵云剑，便是浮现了出来，言语之中还是有些震惊的询问道。
<br/>　　“真没想到，少主您居然还活着，而且您的样子，也完全变了，如果不是您身上的剑意，没有变化的话，就连我都没有认出来是您。”
<br/>　　之前在北邙宫之中，事态紧急，云剑便是没有在多询问，现在安定下来，他的心中便是已经对楚风眠，无比好奇起来。
<br/>　　当初万年之前的劫难，就连剑道之主，都是陨落，而且就算是没有陨落，像是楚风眠的大师兄，虽然逃脱活了下来，但是也身受重伤，不久之后也依然陨落了。
<br/>　　可楚风眠，在当时只是一个没有任何力量的废人，别说是可以从这劫难之中活下来，就算是活下来，也不可能在活万年之久。
<br/>　　“而且少主，您居然可以修炼灵力了？”
<br/>　　云剑诧异的开口道，这才是他现在最为震惊的一点，前世之中楚风眠对剑道的领悟，可以是以完美无缺。
<br/>　　不过那个时候，楚风眠却是九绝之体，穷其一生都无法修炼任何一点灵力，只能够是成为一个无法动用灵力的废人。
<br/>　　不止是剑道之主，当时这云剑，对楚风眠也是极为惋惜，这样一尊剑道天才，居然是无法修炼灵力。
<br/>　　就算是楚风眠对剑术的领悟在高，没有实力最为支持，也没有任何的意义，无法发挥出来，随意一名武者都足以将楚风眠毁灭。
<br/>　　可如今这一次在遇楚风眠，可是已经发生了翻天覆地一般的变化了，一切都改变了。
<br/>　　“其实，我在万年之前，已经陨落了。”
<br/>　　楚风眠看着云剑，缓缓的开口道，对云剑，他倒是不需要有着任何的顾忌，可以畅所欲言。
<br/>　　“不过，就在几年前，我却是突然重生到了这具身躯之中，不知道到底是发生了什么……”
<br/>　　楚风眠便是将这几年发生的事，全部告诉了云剑。
<br/>　　“转世重生？这，怎么可能！”
<br/>　　云剑听到楚风眠的话，大吃一惊道。
<br/>　　转世重生的传说，在九域之中，在每一个时代之中，几乎都有过这样的传闻。
<br/>　　可是就算是当初的剑道之主，这种九域无敌的人物，在提到转世重生的时候，都是摇了摇头，表示绝无可能。
<br/>　　就算是这种巅峰人物，都觉得转世重生，是虚无缥缈的事情。
<br/>　　去询问九域之中的任何一位强者，都只会摇了摇头，表示这绝不可能。
<br/>　　可是现在，楚风眠就这样，明晃晃的出现在了云剑的面前，让云剑也不能够不相信，这转世重生一说了。
<br/>　　“你可知道，你为何会转世重生？”
<br/>　　云剑不由的开口道。
<br/>　　按理说，如果真的有转世重生的话，那也是那些九域巅峰的强者，才有可能做到才对。
<br/>　　前世的楚风眠，身为九决之体，甚至就连修炼都无法做到，又怎么可能做得到转世重生？
<br/>　　“不知道，我自从跳下天绝峰后，便是失去了一时，一直是这万年之后，我才苏醒过来。”
<br/>　　楚风眠摇了摇头道。
<br/>　　他到底为何转世重生，楚风眠自己也一直是想要弄个清楚，可是事到如今，他也依然是一无所知。
<br/>　　“也许是偶然吧，在九域之中，时长也会出现一些天地异象的，也许我的转世重生，也是天地异象之一。”
<br/>　　楚风眠开口道。
<br/>　　“算了，也许等我的实力，足以踏入九域巅峰的一刻，就能够弄清楚这到底是怎么回事了，现在猜测，也没有任何意义。”
<br/>　　楚风眠清楚，九域远远是比起他想象之中的，还要神秘的多，尤其是这九域，并非是整个世界，九域之外，还另有乾坤，这一世他接触到星玄，便是更知道一件事。
<br/>　　那边是人外有人，天外有天，在九域之外，到底是隐藏这什么，现在的楚风眠也远远不知道。
<br/>　　“以你的资质，在这一世之中，的确也有着极大的可能，可以踏上九域巅峰了。”
<br/>　　云剑打量了楚风眠一眼，突然开口道。
<br/>　　“你吃了九转夺天丹？”
<br/>　　云剑打量了一眼楚风眠的身躯，便是感觉的出，楚风眠身躯的资质，简直是只能够以恐怖来形容，比起他说见到过的任何天才，都要妖孽。
<br/>　　当初剑皇殿之中，楚风眠的师兄师姐，哪一个不是天才，妖孽，绝对是比起当初七大宗门的那些天才，都要妖孽的多，却是没有任何一人，足以跟现在楚风眠的资质所媲美的。
<br/>　　“不过，我遇到了青鸾，他给了我九转夺天丹。”
<br/>　　楚风眠点了点头道。
<br/>　　要是没有这九转夺天丹，现在的楚风眠，也没有如今的成就。
<br/>　　“主人，真的将九转夺天丹给了青鸾，然后又被你所得到了？”
<br/>　　听到楚风眠的话，云剑有些吃惊，更是喃喃自语起来。
<br/>　　“莫非主人真的知晓了，这万年之后的变故？”
<br/>　　“怎么回事？”
<br/>　　楚风眠听到云剑的话，立刻询问道。
<br/>　　“主人，在劫难前几日的时间，似乎是察觉到了什么，那个时候他叫来青鸾，便是将这九转夺天丹，交给了青鸾，让他带走。”
<br/>　　云剑缓缓的开口道。
<br/>　　他身为剑皇殿的器灵，这所做一切，都是在他的注意之下进行。
<br/>　　当时的云剑只是觉得有些奇怪，剑道之主为何要如此做，可现在看见楚风眠，云剑却是突然察觉到了一丝异样。
