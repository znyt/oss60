　　等级跟吸血蛇差之八级，但涂小安输什么也不能输了气势，他的雷电之力，天谴于人，不信劈不死这条邪蛇。
<br/>　　降妖除魔，替苍生解决掉这条邪蛇，这话有点太大义凛然，涂小安现在其实就想吃了这条蛇，让自己的实力暴涨，得到它的火焰之力。
<br/>　　不管怎样，今日，两蛇只有一条可以活着离开蛇谷。
<br/>　　“嘎嘎...来吧，本王会将你化为灰烬”
<br/>　　吸血蛇火焰滚滚缭绕，大火焚烧一切，血盆大口一张，让人如入黑暗深渊。
<br/>　　“火龙焚天”
<br/>　　只闻蛇谷中一声长啸，霎时间，一片天变得红火起来，火海满天横流，下一秒，火海变幻莫测起来，凝固成一起，一条巨大的火龙露出了峥嵘的一角，火龙翻腾，吞噬便是虚无。
<br/>　　巨大的火龙带着焚天之势，无比狰狞的朝着金银蛇吞噬而去，所到之处，催古拉朽，焚烧一切。
<br/>　　这种可怕的异象很难相信会出现在当代现实中，比科幻片还要科幻。
<br/>　　好在这里是蛇谷，佘山的最深处，如果在佘山的外围，那么难逃世人之眼。
<br/>　　涂小安看到这条火焰形成的巨大火龙，反而露出了释怀的笑，这是他的宿命之战，失败了一次，岂能失败两次，他是那里跌倒，就要从那里爬起来。
<br/>　　这条无比凶煞的蛇，他做梦都想杀了。
<br/>　　金银蛇的条纹鳞片熠熠生光，有惊雷不断窜起，只见涂小安此时眼中出现了疯狂之色，体内的雷电之力一下子被抽空，他歇斯揭底的怒吼一声。
<br/>　　“雷神之怒”
<br/>　　霎时间，金银蛇所处的这片天爆发无比耀眼的光芒，宛如星辰，宛如高阳，这是眼睛所不能承受的光芒，太过耀眼。
<br/>　　轰...．
<br/>　　一声惊天巨响，蛇谷的整片空间绝然的颤抖起来，满天横流，电闪雷鸣，顷刻间飞沙走石，天地变色，如星空轰塌。
<br/>　　激烈的震动，无法无天的火焰跟雷电相互缠绕，相互纠缠，火与电映照整片天空
<br/>　　转念后，三十米巍峨的吸血蛇被一片雷海洗礼，一条不过一米的金银蛇葬身火海。
<br/>　　蛇谷的异象已经可以用翻天覆地来形容，不管是涂小安，还是吸血蛇，都刻意的远离那座长满蛇莓的小山丘。
<br/>　　不让小山丘受到破坏，它们之所以来到蛇谷，究其原因，还是被蛇莓所吸引，这种天材地宝要是在战斗中被破坏。
<br/>　　那是不可原谅的。
<br/>　　两股异象迸发的激烈不知道维持了多久。
<br/>　　久久之后，蛇谷的天，才慢慢的恢复平静，拨云见雾的变成一潭清澈近乎见底的蓝天，没有似火的景象，没有惊雷乍现，远处几朵白云仿佛带着害怕不敢靠近，但又慢吞吞的游来，像是在试探什么一样。
<br/>　　地面一片狼藉，到处都是裂痕跟坑坑洼洼，三十米巍峨的吸血蛇浑身麻木，周身还缭绕了一股电光，上下流转的发出电流声。
<br/>　　显然吸血蛇快被雷电给电的快小儿麻痹了。
<br/>　　不远处，一条金银蛇的小蛇浑身冒着一股焦味，灿灿生辉的蛇鳞片几乎被烧焦，样子看上去十分的狼狈。
<br/>　　一时半刻，这一大一小的两条蛇，都难以站起来。
<br/>　　足足一个小时后。
<br/>　　吸血蛇一声咆哮的扶摇直上，气息凛然，好像要与天试比高，只是火红鳞片不在迸发火焰，变的黯淡无光，但它依然凶煞无比，一双铜铃般的蛇目嗜血非常，深深的锁定住瘫在地上的金银蛇，倏然一抹贪婪闪过。
<br/>　　霎时间，涂小安的天一黑，只见吸血蛇高高在上睥睨的望着他，显然是想一口吞了他。
<br/>　　他这一米的小体型，让吸血蛇用吞都有点夸大其词。
<br/>　　“你输了，本王才是真正的蛇王，吞了你我更名正言顺”
<br/>　　吸血蛇发出如怨似泣的声音，嗜血的光幽然的冒着。
<br/>　　“是吗”冷淡通透的戏谑声悄然响起，只见瘫在地上的金银蛇，骤然消失，下一秒悬浮在高空上，冷淡无波的道：“吸血蛇，你在白日做梦吗”
<br/>　　刚才的惊天一击，两股属性力量对撞，换来的结果就是两条蛇各打五十大板。
<br/>　　涂小安体内的雷电之力被清空，无法使用，但他看吸血蛇的样子，显然也没有比他好上多少。
<br/>　　已然也不能在动用火焰力量了。
<br/>　　竟然如此，那么接下来，就只能肉身相博。
<br/>　　涨...涨...涨...
<br/>　　金银蛇猛然通体发光发亮了起来，顿时间，地面剧烈震动，只瞧他娇小的蛇形在已肉眼可见的速度在暴涨，与此同时，一道骇人磅礴的气息也随之迸发出来，顷刻间飞沙走石，掀起一阵阵狂风四溢。
<br/>　　“吸血蛇，我们的战斗才刚刚开始”
<br/>　　暴涨到二十米的涂小安浑身散发恐怖气息，要是别的生物面前，早就瑟瑟发抖，匍匐与地。
<br/>　　但他面对的是更加庞大的吸血蛇，不会给与他任何惊讶的回馈。
<br/>　　吸血蛇通灵摄厉，早就急不可耐，身形一弓，一展，似一条蛟龙凶煞暴戾的朝着对立的金银巨蟒极速的横冲直撞过去。
<br/>　　它张开血口，巨大的嘴连一头大象都能轻松的吞进去，其中四颗巨大而尖锐的毒牙触目惊心，能咬断一切。
<br/>　　“现在就看我们谁能咬死谁了”
<br/>　　涂小安热血沸腾，杀意骤射，面对比自己还要大上十米的巨蟒，毫不畏惧的迎了上去，与吸血蛇正面硬刚。
<br/>　　所谓生当作人杰死亦为鬼雄，这句话用在涂小安的身上，最为合适。
<br/>　　他死后化为了家蛇，如今跟命运宿敌再战高下。
<br/>　　老天是善待他的，当今世上，谁能死了后，还有重新来过的机会。
<br/>　　还能亲自手刃仇敌。
<br/>　　顶多叫嚣一句，劳资做鬼都不会放过你。
<br/>　　所以涂小安是无比幸运的，他更加不能辜负这份幸运。
<br/>　　蛇谷的天恢复平静，有朵朵白云惬意的飘过，但蛇谷的地却承受灭顶之灾。
<br/>　　不得不说两条蛇的生命力异常的强大，好像还越战越凶，它们携带惊天之力，激战在一起，一个蛇尾横扫出去，何止千斤。
<br/>　　哪怕是一栋房子，都要轰然倒塌，比挖掘机还要恐怖如斯。
<br/>　　战，继续战，两条巨蟒战的惊天动地，不能动用属性能力的它们，如今更加残暴嗜血。
