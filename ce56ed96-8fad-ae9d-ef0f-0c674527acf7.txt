　　　　杀戮剑术的大成，便是要凝聚出远古杀神的形体来。
<br/>　　以远古杀神的力量加持，凝聚出真正杀戮之道来。
<br/>　　不过想要将杀戮剑术大成，凝聚出真正远古杀神的形体来。
<br/>　　在整个九域，能过做到的，都是凤毛麟角，不超过一手之数。
<br/>　　想要凝炼，不知道是要吞噬多少的血气才行，像是楚风眠，有着大比财富，购买了如此之多的圣者精血。
<br/>　　在加上从夺花尊者手中夺取而来的，全部圣者精血加起来，才只凝聚出了远古杀神躯体中最为弱小的一只左手来。
<br/>　　但是这凝聚的艰难，倒是也要楚风眠的意料之中，若是如此简单就可以凝聚到打成的话，杀戮剑术也不可能会有着如此的赫赫威名。
<br/>　　越强大的力量，想要得到付出的代价也就越多，像是楚风眠，虽然实力强大，但是想要突破，也需要千难万难。
<br/>　　起码如今楚风眠还没有感觉到，他有着任何一丝机会可以突破，还必须要寻找更多的奇遇才行。
<br/>　　“远古杀神！力量加持！”
<br/>　　楚风眠冷喝一声，在他的背后，远古杀神的左手，便是凝聚出来。
<br/>　　一道道世间最为强大的杀意，纷纷凝聚而出，一时间整个人间，都好像化为了一片血海一样。
<br/>　　杀神一出，遍地血海。
<br/>　　就连寒月漓在楚风眠的身边，都是不由的凝聚出了自己的灵力来，想要抵挡。
<br/>　　不得不说，这杀戮剑术的确是强大无比，无愧于九域中最为强大的剑术之一。
<br/>　　单单是凝聚出了这其中一只最为弱小的左手，都拥有这如此可怕的力量。
<br/>　　若是可以再进一步，凝聚出远古杀神的右手，甚至是将远古杀神全部凝聚出来的话。
<br/>　　杀戮剑术还可以变得更加完善，强大。
<br/>　　感受到杀戮剑术如今的力量，楚风眠便是喘了一口气，将背后的远古杀神的虚影散去。
<br/>　　在楚风眠的身上，那一股可怕的杀意，也终于是消失了。
<br/>　　感受到楚风眠身上的杀意消失，寒月漓在敢靠近过来，不由的感叹道。
<br/>　　“这剑术，真是强大无比，就连我都感觉到了莫大的压力。”
<br/>　　刚刚在楚风眠全力催动杀戮剑术，凝聚杀意的一刻，就连寒月漓都不敢贸然靠近楚风眠。
<br/>　　就算是她明知道楚风眠无论如何也不会对她出手，但是在她的感觉中，若是贸然靠近楚风眠，便是就会可能直接被楚风眠斩杀。
<br/>　　远古杀神的杀意加持，的确是恐怖到极点的一件事。
<br/>　　“想要再进一步，还必须要得到更多的圣者精血才行，四百缸？五百缸？六百缸？至少需要成倍的数量，才可以再度凝聚出远古杀神的其他部分。”
<br/>　　楚风眠停下来，平静开口道。
<br/>　　如今这一只最为弱小的左手，便是耗费了楚风眠足足两百缸圣者精血的血气。
<br/>　　要是想要在进一步，只能是付出成倍的数量才行。
<br/>　　“甚至必须要去寻找一些妖神精血了，妖神精血的血气，比起圣者精血要浓郁的多，对我更是有着大用。”
<br/>　　楚风眠出声说道。
<br/>　　妖神精血的灵力，也许跟圣者精血相差不多，但是妖神精血的血气，却是要远远超越圣者精血。
<br/>　　毕竟一尊妖神浑身的血气，到底有着多么恐怖？只怕是要超越人类圣者数十倍。
<br/>　　单单依靠圣者精血想要再度突破，不知道还需要耗费多少，利用妖神精血的话，倒是会省力很多。
<br/>　　“走吧，去参加晚宴去吧。”
<br/>　　楚风眠苏醒过来，开口道。
<br/>　　他如今倒是也要看看这晚宴，到底虚家打的什么主意。
<br/>　　“好。”
<br/>　　寒月漓点了点头。
<br/>　　旁边房间的拓跋鼎，也苏醒了过来，走了出来，便是跟随楚风眠，离开了房间中。
<br/>　　以三人的实力，便是一瞬之间，便是来到了这虚家的一处巨大无比的宫殿前。
<br/>　　这宫殿附近，有着无数虚家护卫守护，极为森严。
<br/>　　一尊尊强大武者，便是纷纷出来，前来了这宫殿附近。
<br/>　　“快让开！王家的人来了！”
<br/>　　楚风眠刚刚来到宫殿前，便是有人突然大喊道，随后周围的无数人，都是如同洪水一般，纷纷退让开来。
<br/>　　留出了一道宽阔无比的大道来，让开了。
<br/>　　随后一辆马车，便是从中走了过来，这马车，居然是一件地级灵器，利用的种种手段炼制而成的，在这马车中，寻常圣者，都无法威胁到其中的人。
<br/>　　而这拉马车的马，也不是寻常马匹，乃是一种有着一缕真龙血脉的马匹，赤麟马，一身赤麟，相传有着真龙的血脉，一日万里，不成问题。
<br/>　　这样一匹赤麟马，只怕都价值一亿九鼎丹左右，这一辆马车，足足是有着八匹这样的赤麟马。
<br/>　　在加上马车，这一辆马车，少说价值十亿财富。
<br/>　　“这气派可够大的啊，王家，看起来也是一大家族了。”
<br/>　　楚风眠心中暗自想到，便是灵识一动，观察着。
<br/>　　马车在宫殿前，缓缓停下来，从其中便是走出了一名白发男子。
<br/>　　这白发男子，样子也就二十岁出头的样子，不知什么原因，却是一口白发，他一下马车，扫视了一眼众人，便是一种发自内心的不屑眼神。
<br/>　　“此人是王傲，乃是我们北邙学院中的一尊内门弟子，不过据说已经是快要晋升为种子弟子了，力量指数逼近十，距离种子弟子只差一步之遥，在内门弟子中，近乎无敌的存在。”
<br/>　　那拓跋鼎，看到这白发男子，突然小声对着楚风眠出声说道。
<br/>　　“王家的势力，比起我背后的拓跋家还要大，在北邙学院中举足轻重，这虚家的人，居然是跟王家有所关系？”
<br/>　　拓跋鼎的声音中，也有些诧异，似乎也并不知道这一层关系的存在。
<br/>　　“居然还有北邙学院的弟子来了。”
<br/>　　楚风眠嘴角露出一抹笑容，这倒是越来越有意思了。
<br/>　　“是王傲。”
<br/>　　“王家的人，怎么会在我们斗玄国度中。”
<br/>　　周围的无数人，看到王傲，似乎心中也都是惊讶无比，一个个有些不可置信。
<br/>　　不清楚这王傲，为何会出现。
