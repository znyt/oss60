　　“轰隆隆！”
<br/>　　雷霆之中，雷龙破碎，化为能量，灌注元始元神之中，不一会儿，雷霆就消失不见了。
<br/>　　这大吞噬术是元始参悟的吞噬法则所创，吞噬法则极为强悍，在三千大道法则之中能排进前三十，吞噬之主也在当年的混沌之战中战死，他的魔神真身有幸被元始收集到。
<br/>　　经过多年参悟，元始借助元始印已经参悟了多重法则，但仅仅只限于皮毛，法则也不是那么容易就参悟透的。不过，虽然只是些皮毛，但能排进前三十的法则，对于小小的化形之劫，也足够了。
<br/>　　大吞噬术已经被修炼《神元九转》的元始融入到神魂之中，能够做到瞬发而出，为了对付天劫，元始也是拼了。
<br/>　　轰隆隆！
<br/>　　天劫也被元始的此种行为所激怒了，虽然他是盘古元神所化，有着大气运，但它还是决定给他一个难忘的教训。第二波雷劫闪烁着赤黄绿白黑五彩光芒，交织在一起，这个昆仑天空都被五彩天劫覆盖，轰然之间，朝着元始元神轰来。
<br/>　　大五行天劫！
<br/>　　五行天劫是众多天劫中排名靠前的几种，能够逆乱人体五行，毁人道体，灭人神魂，威力惊人，这才第二波天劫，元始的就遇到这种天劫，真不知道是元始幸运，还是他倒霉。
<br/>　　“哈哈哈！来吧，五行天劫？那又如何？”
<br/>　　元始狂笑，笑声之中，透露着不惧生死的豪迈，他周身气势惊人，迎向雷劫，毫不畏惧，元始意志可憾天！
<br/>　　“杀！”元始再次冲入雷劫之中，让狂暴刚烈的雷霆轰击在元神之上，阴神转阳神，造化天地。
<br/>　　“五行雷劫吗？好！五行印，给我出——！”
<br/>　　元始暴喝一声，双手结印，一个闪烁着白绿黑赤黄五色的转轮旋转而出，金木水火土，五色五行，缓缓旋转，相生相克，吞噬同化着苍穹之上的五行力量，慢慢壮大。
<br/>　　“五行对五行，最好不过了。”元始看着不断吞噬五行之力壮大己身的五行印，微笑着说道。
<br/>　　五行印！
<br/>　　五行印，是元始为混沌魔神是就创出的一个神通，分为大五行印与逆五行印、五行分印，威力无穷，经过与盘古开天第五式定五行相结合，从此威力更胜。
<br/>　　哗啦！
<br/>　　雷劫消散，五行印化为五道流光飞灰元始身体之中，分别钻入元始五脏之中，滋养着元始五脏。
<br/>　　轰隆！
<br/>　　第三道雷霆如约而至，雷霆化作青紫两色，相互旋转，相生相克，互相增补。同时，天地狂风呼啸而起，天地变色，日月颠倒。
<br/>　　无相罡风雷！
<br/>　　风雷结合，如阴阳相合，风借雷势，雷助风威，威力暴增。风雷滚滚，闪电呼啸，劫云翻滚，透着一股强烈的杀意轰下。
<br/>　　刹那间，万千条青紫色雷龙，就包围在元始元神四周，封锁住元始的去路，不让他逃走。
<br/>　　“罡风？罡雷？风雷吗？既然如此，那么……”元始若有所思道，“阴阳印，给我出！”
<br/>　　一瞬间，黑白两色圆印旋转而出，一条黑鱼，一条白鱼，首尾相连，负阴抱阳，渐渐化为一个巨大的太极阴阳图，围向风雷。
<br/>　　轰！雷龙狂轰滥炸，不顾一丝情面，带着毁天灭地的威力，轰然而下。阴阳印不急不慢，像一个黑洞，将所有击向它的雷龙轰碎，吞噬，同化。
<br/>　　阴阳印！
<br/>　　阴阳印，是元始参悟盘古开天时第四式割阴阳，并依照自己的水火印，领悟而出，目前还不太完善，威力不显，但元始相信以后一定会威震诸天。元始时常在想，要是自己当时得到太极阴阳图就好了，自己的阴阳印也可以更好的完善。不过，那也只是想想，现如今谁知道太极图在哪里。
<br/>　　闪电游走，劫云轰鸣炸响，元始却丝毫不惧，不断地破灭一道又一道雷劫，被阴阳印同化吸收。
<br/>　　轰轰之声惊天动地，雷劫一波波的被消耗，雷龙咆哮，但在元始的凶残轰击之下，纷纷悲鸣而起，化为最纯净的造化之力，被元始吞噬，同时，元始的身体，也在这一波波雷霆之中，变得更加凝实，隐隐有化形的意味。
<br/>　　第四波……
<br/>　　第五波……
<br/>　　……
<br/>　　整整三个月过去了，苍穹仍在轰鸣，雷霆仍然不止。在这三个月的时间中，元始的元神多次被轰碎再重组，意志不灭，灵魂则不灭。在破灭中重生，在寂灭中觉醒，于毁灭之中造化，于雷霆之中再生。
<br/>　　元始的元神一次次地凝实，身体之中，生机渐渐多了起来。连带着元始对雷霆之道的领悟也在快速增加，想要领悟雷霆之道，就必须经历过雷霆，于天劫之中领悟雷霆，没有什么比这更好的了。
<br/>　　“轰隆隆隆！”
<br/>　　天地在轰鸣，随着元始度过第四十九道天劫，天地灵气瞬间暴涌而来，天地在这一刻仿佛在沸腾，灵气跳跃，霞光万兆，瑞气祥云，整个天地仿佛在为他庆祝新生。

<br/>　　只见昆仑之上，苍穹之巅，风云倒卷，灵气风暴不由自主的汇聚而来，灌入元始的身体之中，修复着他已经快要破碎的身体，同时，天空降下一道道白色光柱，那白色光柱照耀在元始身上，元始如沐春风，一阵阵舒爽从身体之中传来。周围的灵气更加疯狂的汇聚而来，天空一团五彩之光落下，造化之光包围元始，改造着他的身体。

<br/>　　光芒散去，露出其中一个身无寸缕的男子，只见这男子不过三十岁左右，面色威严，透露着一股高贵之气。男子身如白玉，一头黑发披散而下，直到臀部，双目紫光闪烁，但很快隐去。
<br/>　　“呃？！”元始注意到自身情况，哭笑不得，“没想到我也裸奔了，我可没有这习惯啊？”
<br/>　　如美玉般的双手一挥，一身白玉色道袍出现在身上，绣袍之上金色花纹环绕，尽显贵气。
<br/>　　“一元复始，万象更新，洪荒之内，吾为玉清元始！”
