　　楚风眠看着白萱，笑了笑道。
<br/>　　“这次回来，倒是轰动非凡了。”
<br/>　　他们二人如今这次回来，毫不掩饰，当然不是为了嚣张，还是有着几分炫耀武力的意思。
<br/>　　北邙学院的种子弟子，不是需要考核，而是需要北邙学院的长老亲自挑选。
<br/>　　只有在内门中表现出色的弟子，才会被邀请晋升种子弟子。
<br/>　　所以想要晋升种子弟子，也必须要展现出自己的实力才行。
<br/>　　白萱看了一眼楚风眠开口道。
<br/>　　“以你的身份，一会肯定会有不少不长眼的家伙过来，你还是做的不要太过分为好，毕竟这里是北邙学院，闹出人命来不好交代。”
<br/>　　“自然。”
<br/>　　楚风眠点了点头道。
<br/>　　“那些蝼蚁，我只会给他们一个教训，不会直接杀了他们的。”
<br/>　　楚风眠可还没有跟北邙学院撕破脸皮的打算，现在闹出人命来，谁都无法下场。
<br/>　　不过若是有人敢招惹楚风眠，楚风眠自然也会给他们一个惩罚才行。
<br/>　　两人正在交谈之际，两道遁光，突然飞到了白萱的洞府周围。
<br/>　　这遁光，明晃晃的闯入这洞府附近，简直就是在挑衅。
<br/>　　“你们看！是燕古师兄跟牧野战师兄！”
<br/>　　“他们二位之前被长老收为弟子，最近才回到北邙学院之中，如今居然出现了。”
<br/>　　“他们二人现在都已经比得上地榜前十的地步，这还是隐藏了极大的实力，若是真的全力，只怕是地榜前三，都可以争上一争。”
<br/>　　“快看，果然他们二人出手了！”
<br/>　　一尊尊北邙学院弟子，纷纷开口道。
<br/>　　楚风眠跟寒月漓的关系，他们自然知晓，同时也明白，这燕古正在追求寒月漓。
<br/>　　如今寒月漓无法对付，自然要向着楚风眠开刀。
<br/>　　“楚风眠，滚出来！”
<br/>　　一声怒喝，突然响起。
<br/>　　只见燕古，凌空而立，在他的身边，周围的空间便是凝聚出了一道道晶体屏障。
<br/>　　这晶体屏障，仿佛是周围的空间所化，无时无刻的都要守护着燕古的身躯。
<br/>　　“这是玉镜神护决！是北邙学院特有的一种极为玄妙的护身秘术，一般只有种子弟子才能接触的到的，眼前这燕古居然练成了！”
<br/>　　一尊北邙学院弟子，看着燕古身边的晶体屏障，喃喃开口道。
<br/>　　这燕古身边凝聚出的玉镜屏障，足以无时无刻守护他的身躯，是一种绝对强大的护身秘术，除非是可以打破这玉镜，不然燕古便是立于不败之地。
<br/>　　显然是这岩谷，被长老收为弟子后，也是得到了极大的好处，实力又进一步。
<br/>　　“玉镜神护决？以此人的身份，应该是不可能有资格接触到的，看来是他的师尊，玉归长老，赐予他的。”
<br/>　　白萱缓缓的开口道。
<br/>　　“看来这玉归长老，倒是极为看中他，居然是不惜冒犯门规，赐予他这等法决。”
<br/>　　“能够短短时间成长到如此地步，背后的培养，是绝不可少的。”
<br/>　　楚风眠随意开口道，丝毫没有将这燕古放在眼中。
<br/>　　若是说刚刚进入北邙学院，考核的时候，楚风眠对着燕古还是有些忌惮的。
<br/>　　但是现在，楚风眠得到祖龙至尊剑，在虚空海中得到了无数机遇之后，这燕古在楚风眠看来，便是如同蝼蚁一般的人物。
<br/>　　就连让楚风眠正视他的资格都没有。
<br/>　　“怎么？楚风眠，你要当一个缩头乌龟吗？躲在女人的洞府里，真是可笑，就凭你这样的人，也配得上寒月漓？”
<br/>　　燕古看着楚风眠没有回话，气焰更加嚣张起来。
<br/>　　“你不滚出来，我就在这里等着你，看看你到底什么时候出来！”
<br/>　　“哦？我配不上，难道你就配得上吗？”
<br/>　　楚风眠冷笑一声，听到燕古的话，脸上都是露出几分嘲笑。
<br/>　　先不要提楚风眠配不配得上寒月漓，就这燕古，跟寒月漓都是云泥之别，寒月漓可是可是种子弟子，北邙学院重点培养的人物。
<br/>　　这岩谷虽然是长老弟子，但是就连种子弟子都不是，相差实在再远。
<br/>　　“该死！”
<br/>　　燕古听见楚风眠的话，脸上猛然露出几分怒色，楚风眠提到的，简直是令他气愤不已的事。
<br/>　　本来在考核中，燕古看到寒月漓，便是有着想要将寒月漓收复的想法，可现在他跟寒月漓的地位，已经是天差地别。
<br/>　　楚风眠的话，正好是戳到了他的痛处。
<br/>　　燕古怒声吼道。
<br/>　　“楚风眠，就算是我燕古配不上，你楚风眠缩头乌龟一样的家伙，也一样配不上，在不滚出来，我就砸了这洞府！”
<br/>　　燕古说着，他的一身灵力猛然暴起，一层层的玉镜，层层交织在一起，其中蕴含透过蕴含的光芒，都是恐怖至极的力量。
<br/>　　“想要让我出来？我不过是想要让要多玩你一会罢了，既然这么想要教训，那我就教训教训你！”
<br/>　　楚风眠看着燕古，冷笑连连，身形一动，直接离开洞府，凌空而立。
<br/>　　“楚风眠，你终于出来了！不当缩头乌龟了？”
<br/>　　看着楚风眠出来，燕古脸上的嚣张气焰更甚。
<br/>　　“什么？想要教训我？可笑，就凭你？一个小小的外门弟子，就凭你一个都没有被长老挑选中的人，也敢说要教训我？真是可笑！”
<br/>　　“燕兄，出手轻一点，这里毕竟是北邙学院，杀了他总归是不好的，废了他的修为，让他滚出北邙学院便是。”
<br/>　　在燕古身边，那牧野战也开口了。
<br/>　　“楚风眠，我跟你并无冤仇，不过听说过你一来北邙学院，便是嚣张无比，废了不少弟子的修为，今日我们废了你，也是为北邙学院除一个害！”
<br/>　　“废了他的修为？让他滚，也未免太便宜他了！”
<br/>　　燕古舔了舔舌头，脸色狰狞无比道。
<br/>　　“我要在他的身体中种下禁制，让他成为我的一条狗，带他爬到寒月漓的面前去，看看寒月漓还会不会喜欢这个废物！”
<br/>　　“什么北邙学院的规矩，不过是废了一个区区外门弟子，北邙学院还敢处罚我？”
<br/>　　这燕古，牧野战开口，好像是已经将楚风眠当成了死人，可以随意拿捏一样。
<br/>　　“你们说够了没有？”
<br/>　　终于楚风眠的脸上露出了几分不耐烦。
<br/>　　“说够了，你们就跪下吧，现在跪下，是你们唯一的机会。”
