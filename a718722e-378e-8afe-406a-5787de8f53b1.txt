　　第242章
<br/>　　七月十三夜，子时，青叶佛出现在天牢中，二人目光相对，相顾无言。
<br/>　　青叶佛打开牢门进来，又将他身上层层枷锁打开，手镣，脚铐，身上缠了好几圈的铁链，沉重的木枷。
<br/>　　程大雷第一次这么轻松，下意识伸展身体，骨节发出噼啪响声。
<br/>　　“跟我走。”青叶佛起身离去。
<br/>　　程大雷愣了愣，警惕的盯着周围，明日就是行刑的日子，今夜叫自己出来做什么，是等不及今夜就要杀，还是说正式行刑前先彩排一下，可死刑犯上刑场前还有碗砍头酒呢。
<br/>　　“你有一个时辰的时间，一个时辰内长安对你不设防。”青叶佛看到程大雷没有跟上来，回头唤道。
<br/>　　程大雷赶紧跟过去，鬼鬼祟祟盯着周围，他忽然发现，今夜的牢房一个守卫都无，空空荡荡，如同寂寥的鬼蜮。
<br/>　　咚！
<br/>　　程大雷手指敲在青叶佛光亮的脑门上。
<br/>　　“你干嘛？”青叶佛本来不会反应不过来，关键是没想到。
<br/>　　“疼嘛？”程大雷认真问：“我看是不是我太想出去了，所以在做梦？”
<br/>　　“梦你XX”青叶佛深呼吸，深呼吸，默念阿弥陀佛，佛不打诳语，也包括脏话。
<br/>　　我佛啊，赶紧把这妖孽收了吧，和他在一起，很影响人设。
<br/>　　天牢洞开，走出后明月在天，清风拂面，就连大门都是敞开着。
<br/>　　走到长街上，程大雷有种感觉，自己好像走入一副画卷，画卷里的一切都是静止的，只有自己和一席白衣的青叶佛行走其中。
<br/>　　正想着，前方忽然走过来一人，见到程大雷时怔了怔。程大雷一机灵，此人他认得，正是常忠伯百里胜。
<br/>　　程大雷眼底浮现杀机，正想要不要杀掉此人灭口。可是紧跟着，百里胜与自己擦肩而过，那神情完全是没看到自己的样子。
<br/>　　飞天蛤蟆一直躲在天牢外，监视着天牢的动静，今夜见程大雷出来，心里却不知发生什么事，立刻飞身而起，快速回去禀报消息。
<br/>　　在程大雷和青叶佛走在大街上时，两旁的墙壁房顶上，探出一个个脑袋，像是等着老猫打盹儿，出来偷米吃的老鼠。
<br/>　　想来想去，其实没想到什么法子，于是只有铤而走险，来一个劫法场，杀人放火大闹长安城。
<br/>　　所以这几天诸多人潜入长安城，只等着明日动手。
<br/>　　程大雷身后跟着的人越来越多，猜不透青叶佛的用意，也就在暗处躲着，如果程大雷有危险，随时准备杀出来。
<br/>　　来到西门，城门紧闭，有两个卫兵守在门口。见到青叶佛过来，也不吭声，只是把城门打开一道缝。
<br/>　　当程大雷和他们擦肩而过时，他们又露出百里胜一样的表情，把程大雷当空气。
<br/>　　今夜，长安对程大雷不设防。
<br/>　　宫里已经发出消息，就连城门的两名卫兵也是从禁卫军中临时调了两名百夫长过来。
<br/>　　今夜没有人放程大雷走，当他也不在天牢，至于他如何不见的，嘿，你别问我，因为我没见过他。
<br/>　　在程大雷走出长安城时，两名百夫长准备将城门关上。
<br/>　　“喂，等一下，等一下。”
<br/>　　徐神机忽然跑出来，他虽然琢磨不明白是怎么回事，但看上去程大雷好像已经没危险。既然程大雷已经出城，那么自己也赶紧撤吧，最近城里查得可严，明天想混出城就不容易了。
<br/>　　“不好意思，两位兄弟辛苦辛苦，俺们是一起的，再见，再见。”徐神机很热情，看上去很想和他们来个男人间的拥抱。
<br/>　　两名百夫长看着徐神机、秦蛮、徐灵儿，一时被搞得有些懵，不知该不该放他们走，以询问的眼神看着青叶佛。
<br/>　　青叶佛假装没看见，于是他们也只好假装没看见。
<br/>　　“等下，等下，都是一起的。”
<br/>　　飞天蛤蟆跃出来，大步跑过来。
<br/>　　“都是兄弟，都是兄弟，有时间一起喝酒。”阿喜。
<br/>　　“这夜里还出来，你们也够辛苦的，兄弟，以后去满庭芳喝酒，直接报我的帐，管用。”阿苦有种二把手的牛X。
<br/>　　“一起的，一起的。”
<br/>　　两个严肃的百夫长彻底懵圈，前前后后，有数百人从黑暗中跳出来。
<br/>　　他们都很热情，有人与他们称兄道弟，有人说以后在长安城出事了，可以提他们的名字。有人看上去很懂古董，要和他们聊一聊二人手上的玉扳指。
<br/>　　“软玉不行，玩几次就毁了，射箭得用硬玉，回头去我那儿，我有一箱子。”
<br/>　　他们失落在神经病的汪洋大海，无助的眼神瞧向青叶佛。
<br/>　　青叶佛却忽有种自己踩到狗屎，然后看到别人也踩到狗屎的舒爽感。
<br/>　　终于，人都撤光了，二人失魂落魄的将门关上，感觉脑袋晕晕乎乎的。
<br/>　　“大哥，您没事吧，那些狗官有没有给您用刑？”一群人围着程大雷，捏手捏肩捏头发。
<br/>　　“没事儿！”程大雷一挥手：“爷是什么人，狼行千里吃肉，狗行千里吃屎，咱爷们不管走到哪儿都是吃独份儿和独份儿。”
<br/>　　青叶佛想捂脸，果然是有怎样的老大，就有怎样的手下，而有怎样的手下也会孕育出怎样的老大。
<br/>　　众人走出城门十里外，方才停住脚步，周围空荡荡，只有清风明月。
<br/>　　青叶佛递给程大雷一个包袱，程大雷打开看到里面是官服、印信、还有自己的匹夫剑。
<br/>　　官凭上写着一句话：牛三斤，面黄无须，高七尺，封七品安边校尉，守琴川关。
<br/>　　程大雷将东西收起，问：“这是要放我走，还封我一个官？”
<br/>　　“没有人放你走，但今夜贼人劫了刑部天牢，一个时辰后会有人发现，然后天下通缉。封的是牛三斤，你可以选择去，也可以选择不去。不过贫僧觉得，你会去的。”
<br/>　　“为何？”
<br/>　　青叶佛略顿一顿，目光看向西方：“因为西北有王气。”
<br/>　　程大雷的目光也看向西方，群山叠嶂，黑幕中如同怪物模糊，他愣了半晌，道：“是呐，西北有王气。”
<br/>　　琴川就在西北。
<br/>　　只是不知道这次的基本盘会如何。
<br/>　　“我还有一个弟兄陷在大牢里，我要带他一起走。”程大雷转向青叶佛。
