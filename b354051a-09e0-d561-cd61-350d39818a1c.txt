　　这一位荒古门的半步天人境强者，是从虚空深处，被其中狂暴的剑气，硬生生给逼迫出来的。
<br/>　　一尊半步天人境的强者，一旦是隐藏起来身形，楚风眠想要察觉到，都并不容易。
<br/>　　毕竟现在楚风眠哪怕是突破了，其实还是九劫古帝的境界，距离半步天人境，在力量上也许是没有了差距，可是境界上，还有着几分差距。
<br/>　　所以楚风眠在跟这一尊半步天人境强者交手的时候，一抹灵识，也是深入到了虚空之中，在探察这这一位半步天人境强者的所在。
<br/>　　自己在明，敌人在暗，对楚风眠来说可不是一件好事。
<br/>　　如今催动剑气，硬生生是将他的身形，从虚空深处给逼了出来，楚风眠也更好从容应对。
<br/>　　这一道身影，从虚空之中飞遁而出，便是显现出了人影，这是一名中年男子，眉头之上，有着一道伤疤，看起来年龄也就三十岁出头的模样。
<br/>　　不过半步天人境的强者，寿命悠长，比起九劫古帝的寿命要长的多，所以眼前这一尊中年男子的真正年龄，可是比起那枯叶还要老的多。
<br/>　　楚风眠在看清楚这中年男子真正样貌的一刻，嘴角也是浮现出了几分诡异的笑容。
<br/>　　“好手段，能够察觉到我的身形所在，看来你的真正实力，已经是跟真正的半步天人境，没有任何的区别了。”
<br/>　　那一尊中年男子，从虚空之中被逼迫出来，看向楚风眠的眼神之中，也是微微有些吃惊。
<br/>　　他隐藏身形的手段，可以说是无比玄妙，哪怕是同样为半步天人境的武者，都难以察觉到他身形所在。
<br/>　　更何况是一尊九劫古帝。
<br/>　　如今楚风眠能够逼迫出他的身形来，已经是代表这，单单是在灵识方面，楚风眠已经不再半步天人境之下了。
<br/>　　这一尊中年男子，看向楚风眠的眼神之中，也露出了几分惊疑之色，打量着楚风眠，似乎是无比好奇。
<br/>　　“有意思，没想到剑道门在万年之后，居然还出现了这样一尊人物！”
<br/>　　话音一落。
<br/>　　在他的目光瞳孔之中，流露出了几分金光。
<br/>　　似乎是想要将楚风眠的所有底牌，都给看穿。
<br/>　　被这目光扫过的一刻，仿佛是任何的底牌手段，都无法瞒过他的双眼。
<br/>　　鎏金瞳！
<br/>　　鎏金神体的特性，这鎏金神体，也是位列先天神体之一。
<br/>　　不过鎏金神体，却是没有任何的战斗能力，而是会赐予一双足以看穿一切的双眼，便是这鎏金瞳。
<br/>　　同境界的武者，甚至可能是比起这鎏金瞳主人境界还要高的武者，都会被这一双眼睛所看穿，一切秘密，都将无从遁形。
<br/>　　不过当这鎏金瞳，看向楚风眠的时候，却是发现了一片虚空，楚风眠隐藏底牌的手段，可所谓的无穷无尽，纵然是鎏金瞳，也难以看出楚风眠的玄妙。
<br/>　　而且楚风眠感觉到，冥冥之中，楚风眠也都有着一种可以遮掩窥探的能力。
<br/>　　仿佛是重生一来，楚风眠便是成为了一种独特的存在，足以屏蔽他人的窥探，只要楚风眠愿意，任何一位半步天人境强者，都不可能看穿楚风眠。
<br/>　　二世为人，带给楚风眠不止是拥有二世的道心，更是有着其他的玄妙。
<br/>　　任由这中年男子，催动鎏金瞳如何窥探，依然是无法从楚风眠的身上，看穿任何一点秘密。
<br/>　　最终这中年男子，也是放弃了，他也知道，在窥探下去，也没有任何的意义。
<br/>　　除非是楚风眠愿意让他看到，不然的话，他如何窥探，都将无功而返。
<br/>　　“没想到，一个小小的九劫古帝，都可以瞒得住我的鎏金瞳，看来你这小家伙，只怕是得到了惊天的奇遇，也怪不得短短时间内，崛起的如此之快。”
<br/>　　这中年男子看向楚风眠，嘴角浮现出几分冰冷笑容道。
<br/>　　“不过无妨，给将你擒下来，带回荒古门之中，严刑拷打一番，你的一切秘密，我们依然可以知晓。”
<br/>　　这中年男子，看向楚风眠的目光，如同是看向一个死人，他哪怕是被楚风眠逼出身形来，也没有将楚风眠，放在眼中。
<br/>　　毕竟以他的半步天人境的境界，想要对付楚风眠一个区区九劫古帝，不是手到擒来。
<br/>　　只是当这中年男子，即将出手的时候，却是看到了楚风眠脸上，那一抹诡异笑容。
<br/>　　看到这一抹诡异笑容的一刻，中年男子的心神之中，都露出了几分愤怒，冷喝道。
<br/>　　“小子，你笑什么？”
<br/>　　“哦？我只是没有想到，居然还有跟你见面的一日，荒鎏金，没想到你的命倒是不小，居然还没死？”
<br/>　　楚风眠看向这中年男子，也就是这荒鎏金，突然开口道。
<br/>　　“看来你命还不小，之前偷到剑道门来，被放逐到了无尽海沟之下，居然还活了下来，看来你能够踏入半步天人境，也是在无尽海沟之下，得到了几分奇遇吧。”
<br/>　　眼前这一尊半步天人境强者，名为荒鎏金，先天神体，算是当年荒古门之中的天才人物。
<br/>　　不过此人修行的，却是一直隐藏身形之法，天天偷窃各大宗门，盗取了无数宝物，后来一直是偷到了剑道门之中，被楚风眠的大师兄剑凌抓到，扔到了无尽海沟之中。
<br/>　　无尽海沟，是这九域边缘之海之中的一处绝地，是一道深不见底的海沟，其中隐藏无数危机，被扔下去的人，九死一生。
<br/>　　没想到这荒鎏金，还活了下来。
<br/>　　看到这荒鎏金的一刻，楚风眠就想起了万年之前，他被扔下无尽海沟的样子，不由的笑了起来。
<br/>　　“你是谁？你怎么知道的这件事？”
<br/>　　听到楚风眠的话，那荒鎏金脸色大变。
<br/>　　这对他来说，可是一生的耻辱，被扔下无尽海沟，历经千辛万苦，他才从其中爬了出来，那副凄惨的样子，他绝对不允许任何人知道。
<br/>　　“荒鎏金，你从剑道门之中，偷走的那一尊剑鼎，交出来吧。”
<br/>　　楚风眠看向荒鎏金，没有回答他的问题，而是平静的开口道。
<br/>　　“念在你万年前没有攻打剑道门的份上，交出剑鼎，今日我不杀你。”
