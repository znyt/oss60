　　两座山的中间，有一条河，从远处一直流淌出来，大伙沿着河边进入峡谷，贴着山脚下往前飞行，在群山脚下环绕，偶尔能看到几个在山中清修的生灵，也都远远地避开，绕过了几座山，猛然间看到一座山的山顶挂着彩虹，从一头延伸道另一头，看上去像一座桥。
<br/>　　彩虹边上，有层层叠叠的祥云，也被彩虹染成了七种颜色，景象之美，让大伙看得都有点出神。
<br/>　　“这里让我想到了鹊桥。”芮冷玉一脸向往的说道。
<br/>　　“这应该就是彩云峰了，”叶少阳望着山峰说道，“据说这种山峰的主人，是七大长老中的彩云仙子的道场，我们小心一点，不上山，山下有一个洞穴，星月奴的肉身，就在里面。”
<br/>　　“没人把守吗？”芮冷玉问。
<br/>　　“肯定有，所以我们要冲进去。”
<br/>　　大伙一口气飞到彩云峰山下，躲在树丛里，能看到山的一面好像斧头劈过似的，完全笔直，有一道泉水哗哗流下来，形成一道水帘，将彩虹的光反射下来，五彩缤纷，绚烂壮观。后面隐约可见一处洞穴。
<br/>　　这风景之美，放在人间绝对是5A级的。
<br/>　　“应该就是这里了，似乎没人把守啊。”叶少阳上上下下地望着。
<br/>　　道风的目光，却落在洞穴两边的草丛里。他看到了两块石碑，立在洞穴两边的乱草中，从这边看过去，也看不到碑上有什么，但道风却从中觉察到了一丝不寻常的气息。
<br/>　　“那是什么？”杨宫梓也发现了石碑的异样。
<br/>　　道风轻轻摇头，他也不知道这是什么，只是觉得这石碑很不对劲，似乎透着一股神秘力量。
<br/>　　连道风都感到神秘的力量，那自然是非同寻常。
<br/>　　一时间，大伙目光都落在这座石碑上。
<br/>　　“这石碑有古怪，道风，我估计是个什么法阵，还须当心啊。”
<br/>　　道风听到最后几个字，心中微微一动，更加强了自己的判断，没等开口，杨宫梓先说道：“若是遇到伏击，我跟邪神挡着，你们几个进去，撑得一时是一时。”
<br/>　　道风看了她一眼，道：“我们一起挡着，少阳你自己进去。”
<br/>　　“这……”
<br/>　　没等叶少阳说完，道风已经飞了过去，落在洞穴外面。
<br/>　　杨宫梓和上古邪神紧随其后，落在他身边。
<br/>　　三人背靠着背，面对着不同方向，四下张望起来。
<br/>　　“没有守兵。”杨宫梓道。
<br/>　　“你跟我一起。”叶少阳不由分说，一把拉起芮冷玉，飞身上前，从三人身边越过，进入了山洞。
<br/>　　就在他们穿过水帘、进入山洞的一瞬间，一道明光，从两边的石碑上闪过，道风看得分明，石碑上那些原本暗淡的字迹，突然明亮起来，纷纷从石碑上飞起，首尾相接，形成了一个圆环，将道风三人围在中间。
<br/>　　“果然有埋伏。”道风喃喃道，表情却没有变化。
<br/>　　“没有守兵，光靠这个法阵，怕是不够吧。”
<br/>　　“人是有的，但这法阵，也足以抵挡千军万马了。”道风转头看着杨宫梓，目光闪动，声音十分柔和地说道：“宫梓，你保重……”
<br/>　　杨宫梓一怔，没等开口，道风突然袖子一挥，展开了血海万魔幡，在她面前用力一抖，像一团黑色火焰，将她裹在中间，瞬间收了进去，口中念咒，朝着轩辕之门的方向掷了出去。
<br/>　　旗杆凌空穿梭，本来空无一物的长空之中，突然现出了许多道人影，试图伸手去抓，但都被血海万魔幡上强烈的怨气所逼退。
<br/>　　并没有真正的强者出来阻拦。
<br/>　　一眨眼的工夫，血海万魔幡已经飞过了轩辕之门。
<br/>　　道风脸上露出微笑，转头看着上古邪神，一只手搭在他肩膀上，微微发力，上古邪神脸上带着恶魔一般的面具，眼神却是深邃无比。
<br/>　　“今日一战，必有死伤，我与你并肩。”
<br/>　　上古邪神双脚微微分开，抬头朝山顶上方的虹桥望去。
<br/>　　一阵悠扬的乐声，从上方传下来。
<br/>　　铮铮作响，极富节奏。
<br/>　　是琵琶的声音。
<br/>　　虹桥之上，大块大块的云彩，汇聚在了一起，随着乐声的节奏，不断切割成了无数小块，堆砌在一起，远远看去，却像是一丛丛的花朵，七色绚烂，繁复至极，簇拥着一道人影，怀里抱着一只琵琶，纤细的手指在上面不断地弹动起来。
<br/>　　是一个头戴花环，容颜绝美的姑娘，身穿一件古风的百花长裙，眉心处点着朱砂。
<br/>　　这身打扮，再加上身边那七色繁花，看上去就像是一个童话世界里的公主。
<br/>　　不知道她是什么时候出现的。
<br/>　　她没有去看道风二人，脸上带着陶醉痴迷的表情，静静地弹着怀中的琵琶。
<br/>　　突然，她身边那些云朵都缓缓移动起来，围绕着她，像春风，像流水，更像是一群小小的舞者，在跟着音乐的节奏跳舞。
<br/>　　整个画面，如梦似幻，但道风二人，却从中嗅出了危险的气息。
<br/>　　一边弹琴，姑娘微微睁开眼睛，朝道风二人俯瞰了一眼，接着朱唇轻启，缓缓唱到：
<br/>　　“天上白玉京，十二楼五城。仙人抚我顶，结发受长生。误逐世间乐，颇穷理乱情。九十六圣君，浮云挂空名……”
<br/>　　声如天籁，缓缓吟唱着，仿佛一点点勾着人的心神。
<br/>　　道风心神一凛，冷笑道：“彩云仙子？”
<br/>　　彩云仙子不答，继续弹唱，突然，身边那些原本聚集在一起的云朵缓缓分解，朝下方飞来，在空中又组成了七朵巨大的花，悬浮在空中，将道风二人圈在中间。
<br/>　　那些碑文在花朵的里面，也在不断旋转。这样一来，就形成了一道双层结界，强如道风，也感受到了强大的压迫，含蓄待发。
<br/>　　“道风，你果真来了。”
<br/>　　一道紫色长衫的人影，从空中飞落，落在对面一座半山腰的山岩上，背着双手，面带微笑地望着道风。
<br/>　　北斗真人清长风！
<br/>　　“你的伤好了？”道风冷笑着说道。
<br/>　　清长风面上闪过一丝尴尬，上次也是在这轩辕山，两人单挑决斗，最后清长风重伤败北，虽然有客观原因，影响了战斗结果，但他内心严重不服。
