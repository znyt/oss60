　　宝库实在太大，宝物的数量也太多。
<br/>　　要想将所有东西一一看个遍，起码需要半月以上。
<br/>　　半天时间，要想找到适合自己的宝物，只能靠机缘和运气。
<br/>　　萧逸快速地掠过一件件宝物，期间找到不少高级的武技。
<br/>　　玄阶中级剑技，五本；玄阶高阶剑技，三本。
<br/>　　玄阶巅峰掌法，两本；玄阶巅峰刀法，一本。
<br/>　　只可惜，这些都不是他想要的，毫不犹豫地掠过，继续寻找着。
<br/>　　他只能得到三件宝物，必须慎重选择。
<br/>　　剑技方面，他有玄阶中级的崩山斩，再要其他同一级别的剑技，根本无用，反而还会浪费时间去修炼。
<br/>　　其他武技方面，他有噬火百刃，虽然只是玄阶低级，但只要炼至大成，加上自己的融合武技，百道火刃融合，威力丝毫不亚于玄阶巅峰。
<br/>　　所以玄阶巅峰的掌法、刀法，他也不需要。
<br/>　　思来想去，自己现在最需要的，只有玄阶巅峰的剑技。
<br/>　　也不知是不是他运气太背，他已经快速掠过了上万件宝物，却始终没找到。
<br/>　　半天时间，很快便过去。

<br/>　　众人都沉寂在挑选宝物的兴致中，闻言，也只能遗憾地点了点头。
<br/>　　十人中，有八人，都选择了功法武技。
<br/>　　他们倒不像萧逸那般要求多多，只要是玄阶巅峰武技，他们都看得上眼。
<br/>　　北山郡，地阶武技只有寥寥几大势力拥有。
<br/>　　玄阶巅峰武技，已经足以让大部分武者欣喜若狂了。
<br/>　　剩下的两人，分别是萧逸和顾长风。
<br/>　　顾长风没有选，他的机会，必须给予萧逸。
<br/>　　萧逸面露纠结之色，半晌，还是指了指数个木台上的东西，那是妖兽精血，都是四级的。
<br/>　　这已经是他暂时能找到最好，也是最适合自己的东西了。
<br/>　　这四瓶妖兽精血，乃是斩杀四头洞玄九重妖兽所提取，内含极庞大的力量。
<br/>　　“妖兽精血？”易老皱了皱眉头，提醒道，“萧逸，我知道你的武魂吸收天地灵气很慢，故你修炼起来也很慢。”
<br/>　　“选妖兽精血来提升自己的修炼速度无可厚非，不过，这随便一瓶就够你用许久的了，无需三瓶这么多。”
<br/>　　妖兽精血，普遍来说，用途只有两个。
<br/>　　一个是炼药师用来做药引，或者炼药材料。
<br/>　　一个是普通武者修炼，用来药浴。
<br/>　　妖兽精血，内含妖兽的力量。但武者是不敢服用的，只能用水稀释，通过药浴的方式变相去吸收。
<br/>　　一般，一个大浴盆的水，滴上一滴妖兽精血，就够武者修炼几天的了。
<br/>　　修炼速度，比之正常速度快数倍以上。
<br/>　　其他武者，得到妖兽精血，几乎全都是用在提升修炼速度之上。
<br/>　　很显然，易老是误会了，以为萧逸的武魂是控火兽，修炼速度极慢，所以才需要妖兽精血来提升自己的修炼速度。
<br/>　　萧逸摇摇头，道，“谢易老提醒，不过我就选这三瓶妖兽精血行了。”
<br/>　　易老点点头，他和萧逸非亲非故，刚才只是随口提醒一句，并没有再多说什么。
<br/>　　“你还有一次机会，要选什么？”易老问道。
<br/>　　这次机会，自然是顾长风的，不过现在归萧逸了。
<br/>　　“三火傀儡球。”萧逸沉声道。
<br/>　　这是外门宝库中最珍贵的东西之一了，属于亚灵器，论珍贵程度，仅次于灵器。
<br/>　　乃是炼药师通过天材地宝、珍惜矿石，以及奇特的手法炼制而成，炼制的难度极大。
<br/>　　三，代表数量和级别；火，代表属性。
<br/>　　三火傀儡球，便是里面拥有三只先天九重实力的火焰傀儡。
<br/>　　战斗时，可释放出三只火焰傀儡为自己战斗。
<br/>　　传闻，裂天剑派药堂的那位长老，还炼制过一个四水傀儡球，厉害得很。
<br/>　　四水傀儡球，便是里面拥有四头洞玄九重的水属性傀儡。
<br/>　　言归正传，三火傀儡球，由于是亚灵器，所以里面存在着力量。
<br/>　　一旦三只火焰傀儡被释放出来，在战斗中被杀，或者能量耗光，它们就会自动消散，不复存在。
<br/>　　三火傀儡球，也会彻底成为一个空壳。
<br/>　　要想继续使用，必须重新注入能量，重新凝聚三头火焰傀儡。
<br/>　　火焰傀儡的实力，以注入能量的武者修为而定。
<br/>　　不过，三火傀儡球的能量承受有限制，最多只到洞玄一重。
<br/>　　萧逸想着，若是自己往里面注入紫炎，必可使这三头火焰傀儡成为半步洞玄的实力。
<br/>　　日后自己在外战斗时，也能给予自己很大的帮助。
<br/>　　“三火傀儡球。”易老点点头，道，“你倒运气好，竟找到这个东西。”
<br/>　　说罢，易老一一为众人取出禁制内的宝物。
<br/>　　“好了。”易老看向十人，沉声道，“如今，第一轮考核的奖励，你们已经得到。接下来，我会领你们去寒冰池。”
<br/>　　“你们将在寒冰池内待上七天，该疗伤的疗伤，该修炼的修炼，做好准备。”
<br/>　　“七天之后，第二轮考核便会开始。”
<br/>　　说罢，易老带着十人离开外门宝库。
<br/>　　外面，其余的四百九十个武者，早已在等候。
<br/>　　五百人，浩浩荡荡，在易老的带领下，来到位于后山的寒冰池。
<br/>　　寒冰池，乃是整个北山郡都有名的存在，萧逸也耳闻过。
<br/>　　进入后山，众人立刻感受到，一股滔天的浓郁天地灵气扑面而来。
<br/>　　前方百米外，有一巨大的冰池，一阵阵氤氲之气从池上飘出，极为梦幻。
<br/>　　以冰池为中心，方圆数百米范围，成了一个天地灵气极度浓郁的地方。
<br/>　　“在此地修炼，一日之功，可抵外头十日苦修，甚至更多。把握好你们的时间，七天之后，我会回来领你们离开。”
<br/>　　易老沉声说罢，转身离去。
<br/>　　寒冰池，本是天地间自然而生的一个神奇之池。
<br/>　　池水奇冷无比，哪怕是洞玄境武者在里头泡一泡，都会瞬间冻成一块冰坨。
<br/>　　冰池周围，寒气刺骨，人畜勿近。
<br/>　　后经裂天剑派长老，往里头加入大量珍贵的火属性天材地宝，才压制下这庞大的冰冷气息。
<br/>　　冰池的冰冷气息，与火属性天材地宝中的药力气息相冲，才融合成那些氤氲之气。
<br/>　　数百米范围内，四面被一大型聚灵阵封锁。
<br/>　　氤氲之气无法外泄，加上不断聚拢而来的天地灵气，便成了如今这一块修炼宝地。
<br/>　　一众武者，没有浪费时间，立刻盘膝坐下，修炼起来。
<br/>　　为防彼此修炼时吸收了别人身边的天地灵气，故每人都隔开一段距离。
<br/>　　萧逸也寻了一个空旷的地方，开始修炼。
<br/>　　澎湃的天地灵气，排山倒海般向他体内涌去，让他心中蓦然一笑。
<br/>　　他想起了萧家的紫云洞。
<br/>　　不过，紫云洞内的灵气浓郁程度，远远比不上这里，差了百倍都不止。
<br/>　　紫云洞，顶多够凡境武者修炼，对后天武者已经作用很小了。
<br/>　　可是这里，连先天境乃至洞玄境，都效果极大。
<br/>　　两者的差距，可想而知。
<br/>　　“对比外门宝库的四件宝物，这寒冰池，对于我来说，反倒是更大的奖励。”萧逸心中欣喜至极。
<br/>　　这里的其他武者，武魂顶多绿色，修炼一日，可抵外头十日苦修。
<br/>　　但以萧逸紫色武魂的吸收速度，这里修炼一日，怕是比外头修炼百日还多。
<br/>　　“对了，我的血戮剑还需要庞大的真气去温养。”萧逸忽然想了自己的血戮剑。
