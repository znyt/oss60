　　K国，医院。
<br/>　　数量黑色防弹吉普快速驶过医院大门，进入医院中。
<br/>　　第一行动小队还是比中方大使馆抢先一步抵达了医院，行动队长Guram对着跟在后面的一辆吉普车打了手势，车队最后那辆黑色防弹吉普停在医院大门口，四名行动队队员挎着M4A1战术步枪走下吉普，在医院大门口拉起警戒线，暂时接管了医院大门的控制权。
<br/>　　吱——！
<br/>　　吱——！
<br/>　　吱——！
<br/>　　三辆黑色防弹吉普刹停在停车场前，留在地面上的黑色轮胎刹车印，说明着这三辆黑色防弹吉普冲进医院时的高速。
<br/>　　十二名穿着防弹战术背心，挎着M4A1战术步枪的行动队队员走下吉普，以防弹吉普作为掩体，端着M4A1战术步枪瞄准了停车场中的赵天宇和罗昊。
<br/>　　从这些行动队队员的行动动作就能看出，他们接受过专业的军事训练，并且应该有着一定的实战经验，这些行动队队员绝对不是之前那些第九处探员所能相比的。
<br/>　　“事情不妙啊。”罗昊扫了眼堵在停车场前的行动队队员，嘀咕道。
<br/>　　救护车中的司机，这个时候已经是完全不能保持镇定，慌乱的推开车门，跌跌撞撞的往停车场外跑去，嘴里喊着，“不关我的事，不要杀我！不要杀我！”
<br/>　　救护车司机跑到停车场出口处的时候，立刻就被两名行动队队员给保护了起来，送到了安全地带。
<br/>　　“我是K国安全局行动处第一行动队队长Guram，赵将军，我不想跟你为敌，希望你能配合我们，跟我们回去协助调查。”Guram趴在一辆防弹吉普的引擎盖上，他手中M4A1战术步枪上，ACOG瞄准镜中的十字刻线锁定着赵天宇。
<br/>　　“同样的话，我不想多次重复。”赵天宇回答道：“很抱歉，我无法跟你们完成配合。”
<br/>　　“那我们就要采取强制措施了！”Guram说道。
<br/>　　“妈的，要是给我一把56-1，老子打得连他们妈都不认识！”罗昊怒声道，这样的局面对罗昊来说实在是太憋屈了，看了眼身边的赵天宇，问道：“师父，咱们就这么被他们带走？到了他们的地盘，白的都能被他们说成是黑的，无罪都变成有罪了。”
<br/>　　“跟他们走！”赵天宇咬了咬牙，说道。
<br/>　　“师父！？”
<br/>　　“一旦发生交火，我们可以自保，但是救护车里的小熙怎么办？”赵天宇看了眼罗昊，说道：“如果她不小心被流弹击中，你会一辈子背负着负罪感，无法原谅你自己。”
<br/>　　“嘎吱！”
<br/>　　罗昊紧紧地握了握拳头，最后还是松了开来，心中做出了妥协。
<br/>　　医院大门，挂着中国大使馆车牌，车头插着一面小国旗的轿车被两名行动队队员拦截下来。
<br/>　　一名身穿黑色西装，西装上别着一个金灿灿国徽男子走下汽车，走到两名行动队队员面前，说道：“我是中方大使馆大使李昌建。”
<br/>　　左手边那名行动队队员伸手挡在李昌建面前，说道：“这里目前被K国安全局行动处接管，无关人员不得入内！”
<br/>　　“我再说一遍，我是中方大使馆代表李昌建，我现在需要进入医院，将我国三名现役军人带离这里！”
<br/>　　李昌建重申了一遍自己的身份，并且说明了自己此行的目的，而他说话的语气比起之前也是要严肃了许多。
<br/>　　站在医院大门口的四名行动队队员脸色变得有些难看，他们已经从李昌建话语中感觉到了他的怒意，作为K国安全局行动处的一员，他们的确是有着一些特权，但是对比起李昌建这个中方大使，差得太远太远了。
<br/>　　就在这个时候，罗昊驾驶着救护车，在四辆黑色防弹吉普的“护卫”下朝着医院大门口驶来，双方毫无悬念的在大门口发生了对峙。
<br/>　　Guram带着第一行动小队的队员下车来跟李昌建进行交涉，但是双方带着不同的目的，这样的交涉必然是没有任何结果的，双方之间从交涉变成了对峙。
<br/>　　K国安全局行动处第一行动小队这边，依旧是以防弹吉普作为掩体，端着M4A1战术步枪瞄准着中方大使馆车队；而中方大使馆这边，清一色的95-1自动步枪，同样对准了第一行动小队这边。
<br/>　　Guram通过无线电向指挥中心请示行动指令，但是却迟迟没有得到回应。
<br/>　　K国安全局指挥中心，全权负责此次行动的第九处处长科尔此时紧握着对讲机，眉头紧皱。科尔无法对医院中的Guram下达新的行动指令，原本他是想让第一行动小队抢在中方大使馆的人抵达之前，把三个人带回安全局的；可是现在情况完全发生了改变。
<br/>　　准确的说，科尔此时不是无法下达新的行动指令，而是不敢下达新的指令！
<br/>　　只要第一行动小队这边打响第一枪，事情的性质就完全发生了改变，这就意味着K国向一个常任理事国正是宣战，这是K国无论如何都无法承受的。
<br/>　　一旦双方交战，在中方海空火力的配合下，K国的机场、舰队和地面装甲部队会在二十四小时内沦陷，紧接着中方的空降部队和海军陆部队会进入K国，可能连三天都不需要，就能攻入K国首都！
<br/>　　而K国方面作为挑起战争的一方，甚至连争取联合国帮助的机会都不会有，招惹了一个常任理事国还想要去联合国讲道理，别开玩笑了，其他国家不过来多踩几脚就算不错了。
<br/>　　“嗬，来的还不算晚。”
<br/>　　救护车内，罗昊松开紧握的方向盘，脸上变得轻松起来，靠在椅子上，看着外面正在对峙的双方。
<br/>　　“现在该轮到K国方面头疼了，他们现在是骑虎难下。”赵天宇说道：“鸡没偷到反而惹了一身骚。这件事背后，K国肯定是得到了支持，不然就算借他们十个胆子也不敢这么做；但是现在情况不一样了，不管K国是不是按照原来的计划去做，它都已经惹来了我们国家的怒火！”
<br/>　　“不管怎么说，反正现在没有我们什么事情了。”罗昊双手枕在脑后，说道：“我们只需要看戏就行了。”
