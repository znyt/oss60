　　几人在大海之上，不停的飞行，几个时辰的时间，他们已经击杀了二十余名魔甲人。
<br/>　　死亡的魔甲人的储物戒，只要没有毁灭，便都被五人收取了。
<br/>　　不过，储物戒大多数都被廖云剑得了去，苏莫三人得到的储物戒寥寥无几。
<br/>　　苏莫只得到一枚储物戒，其内只有少量的魔晶，所谓的魔晶与灵晶差不多，只不过其内蕴含的主要是魔气，灵气并不多。
<br/>　　当然，这也是一种资源，只要炼化那些魔气，就能留下精纯的灵气。
<br/>　　因为廖云剑等人在，苏莫也没有了吞噬魔魂的机会，让得他已经心生去意了。
<br/>　　“三位，我们现在去一处宝地！”飞行之中，廖云剑对苏莫三人说道。
<br/>　　“什么宝地？”聂乾好奇的问道。
<br/>　　“这处宝地是一处墓穴，魔甲人强者的墓穴，我以前偶然发现，只不过当时实力低微，没有闯进去，现在以我们四人之力，应该不成问题！”廖云剑说道。
<br/>　　“廖兄，那这墓穴中有什么珍贵的宝物吗？”韩盈柔声问道。
<br/>　　“当然，据我所知，这墓穴中葬着一名魔甲人的先祖，其内有很多陪葬的宝物。“廖云剑说道。
<br/>　　苏莫闻言也是来了兴趣，虽然他此行的目的是魔魂，但若是能得到大量宝物，他自然不会拒绝。
<br/>　　毕竟，他以后提升修为，会需要海量的资源。
<br/>　　宝物，既是他的资源。
<br/>　　“既然如此，那我们还等什么！”聂乾同样来了兴趣，略显兴奋的说道。
<br/>　　“嗯！那我们这就过去！”廖云剑眸中异色一闪，点了点头，顿时加快速度，向远处急速飞去。
<br/>　　四人又飞行了两个时辰之后，来到了一片荒凉的海域。
<br/>　　这片海域没有一座岛屿，甚至连礁石都没有一块，海水也显得较为平静。
<br/>　　“墓穴就在海底，我们下去！”
<br/>　　廖云剑招呼一声，便立刻身形下沉，潜入了漆黑的海水之中。
<br/>　　苏莫三人没有犹豫，紧随廖云剑之后，潜入了海水之中。
<br/>　　海水中蕴含强烈的魔气，漆黑无比，伸手不见五指，好在几人都是高手，目力超强，影响倒也不大。
<br/>　　四人足足下潜了五千多里，才来到了海底。
<br/>　　而此刻，他们的眼前，是一片一望无际的海底山脉，仿佛蛰伏在海底的巨兽，恐怖无比。
<br/>　　“墓穴的入口，就在前方的山谷中！”廖云剑一指远处的一处小山谷，随即急速向山谷飞了过去。
<br/>　　很快，四人便飞进了山谷中，山谷不大，只有方圆一里左右。
<br/>　　而在山谷中，一处岩壁之下，赫然有一个方圆两丈左右的洞口。
<br/>　　黝黑的洞口，非常的幽深，其内还有一缕缕魔气冒出，仿佛是通往幽冥地狱。
<br/>　　看到这个洞口，苏莫心中不禁微颤，他有一种莫名的心悸之感，好像此洞之内有无比恐怖的存在。
<br/>　　“廖兄，我感觉此洞内非常的危险！”聂乾开口了，声音中有些惊颤之色。
<br/>　　“我也有这种感觉！”韩盈面色凝重的说道。
<br/>　　“此洞是通向墓穴的入口，洞穴倒是没有危险，但墓穴里面的确有些危险，不过我曾经探过一次，只要我们小心谨慎，以我们的实力，就不会有什么事！”廖云剑笑道。
<br/>　　聂乾和韩盈闻言，稍稍沉默，两人有些迟疑了。
<br/>　　宝物虽好，但若是没有命享用，那也是白搭。
<br/>　　“苏兄，你觉得呢？”聂乾看向苏莫。
<br/>　　“可以一探！”苏莫略一沉吟，便点了点头，他虽然也看出了此地危险，但还是想要一探。
<br/>　　富贵险中求，想要宝物自然会有风险。
<br/>　　当然，他还是有些底气的，毕竟，廖云剑曾经探过此地，都能全身而退，而他的实力比现在的廖云剑，都要稍强两分，自然也不怕。
<br/>　　“他都不怕，你们两人难道还怕了？”廖云剑闻听苏莫之言，顿时笑着对聂乾和韩盈说道。
<br/>　　“那我们就进去一探，若是有危险立刻返回！”聂乾沉声道。
<br/>　　“嗯！”韩盈也点了点头。
<br/>　　“那好，我们进去！”廖云剑见此，一步踏出，率先走进了洞穴之中。
<br/>　　聂乾和韩盈紧随其后，进入了洞穴，苏莫最后才进入。
<br/>　　四人漫步在洞穴之中，速度并不快，缓步前进。
<br/>　　洞穴中非常的阴暗，通道是斜斜通向下方，并且越是前进空间越是宽敞。
<br/>　　四人均是小心警惕，足足走了一个多时辰，才走到了通道的尽头。
<br/>　　而此刻，苏莫四人前方出现了一座古老的石门，石门紧闭，高达十数丈，其上刻画着繁杂的纹路。
<br/>　　而在石门的上方，书写着四个魔气森森的大字——阎魔地宫！
<br/>　　“阎魔地宫！”苏莫呢喃自语，面色凝重不已。
<br/>　　“这地宫便是那名魔甲人先祖的墓穴，不过想进入墓穴的核心墓室，则要通过两关！”廖云剑沉声说道。
<br/>　　“哪两关？”聂乾问道。
<br/>　　“第一关便是迷宫，不过此关已经被我破解，而第二关便是凶灵之地，要挡住凶灵的攻击，才能进入核心墓室！”廖云剑说道。
<br/>　　“什么凶灵？有多强？”韩盈问道。
<br/>　　“这凶灵是一种类似于魔魂的凶物，应该是由魔气中诞生，实力倒是不强，只相当于普通的武帝境二、三重武者，但胜在数量不少。”廖云剑说道。
<br/>　　“凶灵？”苏莫闻言心中一动，吞噬武魂应该也能吞噬吧！
<br/>　　无论这凶灵是一种魂体，还是灵体，亦或是魔气形成，都是吞噬武魂需要的养料。
<br/>　　不由的，苏莫心中暗暗激动了起来，若真的有大量凶灵的话，绝对能提升他武魂的等级。
<br/>　　“那我们如何对付凶灵？”聂乾问道。
<br/>　　“我已经有了计划，由聂兄你和苏莫牵制住凶灵，我和韩姑娘进入墓室，一旦取得宝物，我们立刻撤离！”廖云剑沉声说道。
<br/>　　聂乾闻言眉头微皱，他可不傻，牵制凶灵必然极为危险。
<br/>　　苏莫面无表情，目光淡淡的望着廖云剑，他发现此人并不简单，可能想将他们当枪使。
<br/>　　廖云剑见聂乾皱眉，眸光一闪，笑道：“聂兄不必担心，那些凶灵虽然有普通的武帝境二、三重的实力，但你和苏莫都是天才，联手的话，牵制片刻还是不难的！”
<br/>　　言罢，廖云剑又看向苏莫，道：“你虽然修为很低，但实力勉强还可以，只要能拖住半柱香的时间就行！”
<br/>　　“可以，我同意！”苏莫点了点头说道。
<br/>　　苏莫如此干脆的便答应了下来，倒是让另外三人有些意外。
<br/>　　廖云剑本还担心苏莫惧怕，毕竟对方才准帝之境，没想到对方立刻就同意了。
<br/>　　而聂乾和韩盈两人，也是眸光一凝，暗暗惊奇苏莫如此大胆。
<br/>　　“好吧！那就这么办吧！”少倾之后，聂乾便也点了点，苏莫都同意了，他自然也不会有什么意见！
<br/>　　而且，他虽然只有武帝境二重的实力，但真实战力足以轻易镇杀普通的武帝境三重武者。
<br/>　　“好！那我们进去吧！”廖云剑笑了笑，随即缓缓推开了石门，四人一同走了进去。
