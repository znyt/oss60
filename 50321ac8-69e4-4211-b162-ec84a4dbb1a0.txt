        “哥哥！”连星儿高兴地挥着手，然后拖着行李箱，往冷晏琛那边小跑了过去。
<br/>
<br/>        她才跑到冷晏琛的跟前，一件粉色披风，从冷晏琛的手中划过一道优美的弧度，直接落在了她的肩头上。
<br/>
<br/>        北城地处北方，比南方要冷许多。
<br/>
<br/>        这件粉色披风是他来这里的途中，路过一家女装店，看了一眼就买下来的。
<br/>
<br/>        当时，冷晏琛心想着，星儿这丫头十有八九是穿着单衣过来的。
<br/>
<br/>        现在看来，还真被他猜中了。
<br/>
<br/>        “哥哥，我来看你，你高不高兴？”连星儿嬉皮笑脸地问。
<br/>
<br/>        冷晏琛莞尔一笑，扬了扬剑眉：“我觉得你是祖爷爷派来的‘监控器’，我没说错吧？”
<br/>
<br/>        连星儿顿时忍俊不禁，向冷晏琛俏皮地吐了吐舌头。
<br/>
<br/>        冷晏琛接着从连星儿的手中拿走行李箱，一边抬手揽着她的肩头，一边带着她转身往出租车等候区那边走去。
<br/>
<br/>        “哥哥，爹地不是给你买了车吗？你怎么不开？”连星儿不解地问道。
<br/>
<br/>        冷晏琛会心一笑，反问：“你觉得，我穿着军装，然后开着一辆上千万的豪车，合适吗？”
<br/>
<br/>        “那个……呵呵——也对哦！确实不合适。”连星儿咧嘴一笑，顿了顿后，又问道，“接下来，我们去哪儿？”
<br/>
<br/>        “祖爷爷没跟你说吗？”冷晏琛反问。
<br/>
<br/>        连星儿微微摇了摇头。
<br/>
<br/>        “我们直接去相亲。”冷晏琛意味深长地笑了笑。
<br/>
<br/>        连星儿顿时一脸懵：“？！”
<br/>
<br/>        到了餐馆后，连星儿才明白“我们直接去相亲”是什么意思。
<br/>
<br/>        她自个儿坐一桌，她哥哥坐一桌，她和哥哥之间隔着绿色蔓藤隔断。
<br/>
<br/>        所以，哥哥的动静，她一清二楚。
<br/>
<br/>        对方约的时间是七点，她和哥哥冷晏琛提前了半个小时就到了。
<br/>
<br/>        所以，冷晏琛事先点好了菜，不过是给连星儿点的。
<br/>
<br/>        毕竟，饿着了自己，也不能饿着了妹妹。
<br/>
<br/>        而“未来的嫂嫂”还未到，连星儿这边就已经开吃了。
<br/>
<br/>        晚上七点差不多还差近一分钟的时间，一个女人穿着航空公司的白色机长制服上衣以及墨蓝色制服裤，推门而入。
<br/>
<br/>        在女服务员的带领下，那个女人向连星儿这边的走道走来。
<br/>
<br/>        连星儿是面对着大门那边的，所以，这女人从进门到往这边走过来时的情形，她看得一清二楚。
<br/>
<br/>        女人梳着熠熠生辉的斜刘海，没有一丝多余的发丝遗留在姣好的面容上，往下只见那精致的皮带扣出她那格外令人艳羡的A4腰格。
<br/>
<br/>        她走路恍若带风，气质高贵优雅，甚至能令一旁领路的女服务员自惭形秽。
<br/>
<br/>        连星儿顿时觉得这个“未来的嫂嫂”好帅气！
<br/>
<br/>        不对，是好美才对！
<br/>
<br/>        这真人比相片，要漂亮多了！
<br/>
<br/>        晚上七点整，女人准时走到了冷晏琛的身侧，并礼貌性地伸出了白皙修长的手，问候道：“你好，我是宋云汐，很高兴认识你。”
<br/>
<br/>        冷晏琛见状，连忙起身，十分绅士地握住宋云汐的四根手指，轻轻地摇了摇：“你好，我是冷晏琛，同样很高兴认识你。”
<br/>
<br/>        两人握手自我介绍完后，宋云汐优雅地在冷晏琛对面落座。
<br/>
<br/>        冷晏琛刚想叫服务员拿菜单过来，却不料被宋云汐直截了当地拒绝了。
<br/>
<br/>        “我在飞机上已经吃过了，我们直接进入正题即可。”宋云汐不冷不热地说。
<br/>
<br/>        坐在隔壁雅座里的连星儿，听着她的这话，差点喷饭了。
<br/>
<br/>        她倒是吃过晚饭了，可是她的哥哥为了等她，还未吃的啊！

<br/>        顿时，连星儿对宋云汐的第一印象直接在心底暗戳戳地给了差评。

<br/>        冷晏琛二话不说，微笑着点了点头。

<br/>        宋云汐这才仔仔细细地将冷晏琛打量了一番。

<br/>        按理来说，是她将来的年薪会不及她哥哥的十分之一吧？
<br/>
<br/>        怎么，现在她倒是哪来的自信，说着这么不分轻重的话？
