　　房间之中，
<br/>　　叶尘猛然睁开双眼，眼眸中闪过一抹厉色，
<br/>　　“竟然敢窥视我！哼！”
<br/>　　虽然刚刚，寿天禄只是用神念一扫，不过还是被叶尘瞬间发觉，直接将那道神念打了回去。
<br/>　　在修真界，用神念扫描别人，是一种十分无礼的举动，既然对方无礼在先，叶尘自然也不必客气。
<br/>　　“看来应该是万宝阁，某位化神级的强者……”
<br/>　　叶尘喃喃自语了一句，并没有理会，而是继续进入了修炼状态。
<br/>　　他如今，不但恢复到了鼎盛状态，修为也突破到了金丹境二重，即便是化神境初期的强者，想要找他的麻烦，他也有一战之力。
<br/>　　但如果是化神中期，甚至是化神后期，叶尘还是必输无疑！
<br/>　　“这里才只是四级星域而已，处于修真界的边缘地带，等将来进入天才妖孽辈出的中心星域，我这点修为可就不够看了！还是要尽快提升实力才行！”
<br/>　　一念及此，叶尘长袖一挥，一排排丹药，便齐刷刷的出现在面前的桌子上。
<br/>　　按照他和张凌云的约定，炼制的丹药，一半归他。
<br/>　　而这些丹药，则全是叶尘当下，对修炼最为有用的。
<br/>　　叶尘随手再次一挥，药瓶自动弹开，一颗颗四品上等丹药，纷纷被吸入叶尘的口中，
<br/>　　“乾元四气神丹，具有扩展经脉的作用……太阴凝碧丹，可以提升真元的品质……还有九鼎聚气丹，可以直接转化灵力化成真元……”
<br/>　　随着叶尘将一颗颗丹药先后丢入嘴中，体内顿时生出一股狂暴无比的能量，在他的全身经脉中肆虐起来！
<br/>　　倘若让外人看到，叶尘生吞这些四品上等灵丹，如同吃糖豆一般随意，恐怕非得大呼疯子不可。
<br/>　　如此庞大的能量，若是换成寻常金丹境修士，恐怕此刻早就已经爆体而亡了。
<br/>　　叶尘却恍若无事，又一连吞下十余颗丹药，这才双掌结印，开始进入修炼状态，
<br/>　　“吞天神功！！吞噬！”
<br/>　　呼呼呼！
<br/>　　体内那些四品上等灵药转化成的庞大能量，在吞天神功的运转之下，开始迅速被炼化吸收，化成一道道真元，融入叶尘的丹田之中。
<br/>　　四五个时辰之后，叶尘已经一口气吞食了几十颗灵丹，而他的修为，也犹如坐火箭一般，一路飙涨，竟然再次突破一重境界，达到了金丹境三重！
<br/>　　仅仅一天时间，叶尘就从金丹境二重，提升到了金丹境三重，这等修炼速度，如果让外人得知，恐怕非得惊得连下巴都掉下来不可。
<br/>　　之后的一连数日，
<br/>　　叶尘白天帮张凌云炼制丹药，晚上就用这些丹药修炼，顺便用张凌云派人寻来的灵草，来蕴养曦月的残魂。
<br/>　　两天后，叶尘的修为再次突破，达到了金丹四重。
<br/>　　又过了三天，已经飙升到了金丹五重！
<br/>　　如此恐怖的修炼速度，就连叶尘自己也不由得暗暗感叹，
<br/>　　“不愧是魔族的无上功法，若非我有八百年修真经历，心志早已经经过千锤百炼，否则修为如此冒进，必然会变得狂傲不可一世，甚至走火入魔，心志大乱！”
<br/>　　虽然这种情况并没有发生，叶尘还是果断中止了修炼，准备先积淀一段时间，将根基打牢之后，然后再行突破。
<br/>　　随手撤下周围的禁制，叶尘心念一动，发现门外，张凌云一直在踱来踱去，应该是在等他出关。
<br/>　　房门自动打开，叶尘缓缓走出房间，
<br/>　　张凌云见状，连忙快步走了上来，
<br/>　　“叶兄弟，您可算是出来了！”
<br/>　　叶尘眉头微微一皱，
<br/>　　“有事？”
<br/>　　张凌云一脸郁闷道：
<br/>　　“叶兄弟，今日天玄城来了一位大人物，父亲大人在天香楼设宴，邀请了整个天玄城的权贵，可是这位大人物，不知道从哪里听说了你的事情，点名说要见你，我怀疑是老二搞的鬼，你看这可如何是好？”
<br/>　　“大人物？”
<br/>　　叶尘眉头微微一挑，
<br/>　　“是什么样的大人物？又为何要见我？”
<br/>　　张凌云想了想道：
<br/>　　“我只知道此人的名号是百草仙人，听说来自丹域，是一名五品炼丹师！”
<br/>　　“哦？”
<br/>　　叶尘两眼眯了眯，他现在反正也无事，正想出走走走，于是点了点头道：
<br/>　　“既然如此，那就去见见吧！”
<br/>　　张凌云知道叶尘性格古怪，原本还担心叶尘会拒绝，到时候没有办法向父亲交代，没想到叶尘竟然答应的如此干脆，顿时大喜。
<br/>　　……
<br/>　　两人出了张府之后，沿着宽阔的街道走了十几里，来到一座华美大气的酒楼前，上面牌匾之上，是金光闪闪的三个大字：天香楼！
<br/>　　此时楼阁之内，早已经是宾客满座，而且来的全都是整个天玄城的最上流的人物。
<br/>　　“万宝阁四公子到了！”
<br/>　　叶尘随着张凌云踏入其中，立即有人高呼了一声。
<br/>　　唰唰唰！
<br/>　　楼阁里，所有人纷纷朝门口的方向望了过来，
<br/>　　“快看！凌云公子旁边的那位少年，好像就是天玄城最近人气很旺的叶大师！听说那此人的炼丹术神乎其神，一日便可炼制上百枚四品上等丹药！”
<br/>　　“不是吧！竟然这么年轻？该不会是搞错了吧？”
<br/>　　“或许是驻颜有术呢？听说那些传说中的高阶驻颜丹，足以以假乱真，让人看不出真实年纪！”
<br/>　　“也对！说不定都已经几百岁了吧？”
<br/>　　“也不知道这位叶大师，和百草仙人相比，两人的炼丹术，谁强谁弱？”
<br/>　　……
<br/>　　对于众人的议论声，叶尘浑不在意，只是一脸淡然的跟在张凌云的身后。
<br/>　　楼阁内部很大，而且座位的排列，很显然是根据身份地位来划分的，越靠近门口的位置，地位就越低。
<br/>　　万宝阁作为天玄城三大势力之一，张凌云的地位自然也不会太低。
<br/>　　很快，两人来到一个比较靠前的位置坐了下来。
<br/>　　“叶兄弟，这位百草仙人的面子可不小！待会不光我父亲和三大长老会来，赤阳府主和红莲教主应该也会来捧场，这可是很多人，一辈子都见不到的大人物啊！”
<br/>　　入座之后，张凌云满脸兴奋道。
<br/>　　叶尘却不以为意，神情平淡无比，
<br/>　　区区一些四级星域的地头蛇罢了，若是在前世，连给他当奴仆的资格都没有！
<br/>　　心里正想着，旁边忽然传来一道充满戏谑的声音，
<br/>　　“哟！这不是大名鼎鼎的叶大师吗？今天终于肯露面了！啧啧！”
