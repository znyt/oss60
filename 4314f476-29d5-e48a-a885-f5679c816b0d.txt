　　竖日清晨，天刚蒙蒙亮，两人就早早的起床，简单的洗漱一番后，客栈仆役就送来早餐，同时还有一份前往枫城的地图。
<br/>　　用餐时，慕清霄微微一愣，只见袖口中探出一个美丽的小脑袋，正是从沉睡中醒来的七彩吞天蟒。
<br/>　　只见美丽的小家伙吐着蛇芯，淡紫色的大眼睛望着慕清霄，眼瞳中满是哀求，好似是再说：“主淫，宝宝好饿喔。”
<br/>　　“好可爱啊！”
<br/>　　坐在身旁的小医仙，自然也注意到了七彩吞天蟒，美眸扑闪扑闪，刚想伸手去抚摸，吞天蟒身躯就是一缩，巧妙的避了开来。
<br/>　　它可是高贵的七彩吞天蟒，虽然认慕清霄为主，但也不是谁都能摸的，又不是宠物。
<br/>　　慕清霄嘴角则抽了抽，可爱是可爱，若小医仙知道这条小蛇是美杜莎女王，不知道会露出怎样精彩的表情。
<br/>　　微微摇头，取出一瓶伴生紫晶源，递给小医仙。
<br/>　　握着伴生紫晶源，小医仙有些不明所以，只是见小蛇淡紫色瞳孔可怜楚楚的望着她，甚至游过来蹭了蹭她，完全没有刚才的警惕的模样后，方才理解。
<br/>　　“它食用伴生紫晶源吗？”
<br/>　　“你也太小看它的胃了，只要稍微高级一些的灵药，它估计都能吃。”
<br/>　　小医仙微微颔首，将玉瓶打开，递到吞天蟒面前，小家伙摇了摇尾巴，蹭了蹭她，随后兴奋的舔-舐起来。
<br/>　　用过早餐后，七彩吞天蟒就回到手腕，慕清霄则带着小医仙径直离开客栈。
<br/>　　行走在黑印城大街上，路上行人口中议论纷纷，谈论的话题，自然就是血宗少宗主，惨死在森林中的事情。
<br/>　　甚至，有些不怕事的修炼者，猜测起凶手是谁。
<br/>　　最让慕清霄感到意外的是，现在血宗竟与黑骷墓硬钢上了，就因为黑骷墓昨日埋伏地点，正是范凌惨死的森林百米开外所在。
<br/>　　血宗宗主范崂，直接将所有仇恨算在黑骷墓头上，为了阴阳玄龙丹，杀死他的儿子，理由非常的充分。
<br/>　　黑骷墓是百口莫辩，不是他们杀的，也是他们杀的，慕清霄神色有些古怪，没想到还有黑骷墓给自己背锅。
<br/>　　血宗与黑骷墓最后会如何，慕清霄丝毫不感兴趣，在黑印城已经逗留了两天，也是时候启程去枫城了。
<br/>　　周围空间一阵扭曲，两人就消失的无影无踪。
<br/>　　......
<br/>　　......
<br/>　　枫城，位于黑角域中心地带，也是最繁华的城市之一。
<br/>　　这座城市原本并不叫枫城，只不过由于韩枫的关系，这里最后改名为枫城，而韩枫的府邸位于枫城最中央的地方。
<br/>　　进入枫城后，慕清霄与小医仙依旧是披着黑袍，最后走进一家酒店，刚进入酒店，耳边就传来一阵议论声。
<br/>　　“喂，你们听说没，黑印城里出大事了。”
<br/>　　“听说了，听说了，好像是血宗的少宗主被黑骷墓的人给杀了，两大势力正准备开战呢。”
<br/>　　“何止，听说拍卖会出现了七品丹药，阴阳玄龙丹，最后被天蛇府拍获，但天蛇府的人也在半路被人劫杀，全都死光了，府主震怒，也在追查凶手，现在黑印城是乱成了一锅粥啊。”
<br/>　　听到周围的议论声，慕清霄微微一愣，没想到消息传播速度倒是挺快的。
<br/>　　正当两人愣神之际，店小二走了上来，见两人身穿黑袍，也不在意，微笑道：“两位客官，吃饭还是住店啊？”
<br/>　　“住店，给我开一间上等客房。”
<br/>　　“好嘞，两位客官请跟我来。”
<br/>　　说罢，店小二就带路，向着二楼走去，随后领着两人进入一间非常干净整洁的客房，道：“客官，就是这里了，有什么事，两位尽管吩咐。”
<br/>　　慕清霄递过去五十金币，道：“将你们店里最好的酒菜端上来，多余的赏你了。”
<br/>　　闻言，店小二愣了一下，旋即就是激动无比，他在黑角域这么多年，还是第一次碰到如此阔绰的大人。
<br/>　　“好的，好的，大人您稍等，小的这就让厨房给您准备。”接过金币，店小二态度变得恭敬无比，随后转身就快步朝一楼厨房走去。
<br/>　　约莫五分钟后，店小二就恭恭敬敬的将店中好酒好菜端了上来，道：“两位客官请慢用。”
<br/>　　待店小二离去，慕清霄设下结界，两人才将黑袍取下。
<br/>　　小医仙问道：“待会要我陪你去吗？”
<br/>　　“不用，我自己去就行了，我若是杀不了他，你去了也无济于事，乖乖在这里等我，等事情结束，我们就去迦南学院走一趟。”
<br/>　　吃饭的时候，慕清霄在心中思索。
<br/>　　韩枫与魂殿有联系，说不定枫城中就隐藏着魂殿的人，他的实力在这里虽算是顶尖层次，但也要避免阴沟里翻船。
<br/>　　随着时间的推移，夜幕降临，皎洁的月光倾洒而下。
<br/>　　望了一眼修炼中的小医仙，周围空间一阵扭曲，慕清霄悄无声息的离开，径直向枫城中央，韩枫的府邸而去。
<br/>　　来到枫城中央处，站在高楼顶端，俯视着远处灯火通明的豪华院落，此处就是韩枫的府邸。
<br/>　　“这韩枫倒是谨慎，不过他做梦的想不到，今晚就是他的死期。”
<br/>　　慕清霄能清晰的感觉到，院落中设有许多暗哨，外部有五强烈的波动，内部四个方向各有一股更加强烈的气息波动。
<br/>　　从气息上来看，是五名斗王，四名斗皇，在院落最深处，还有一道非常隐晦的气息，最起码是四星斗宗。
<br/>　　拥有斗宗的实力，看来是魂殿，不会有错。
<br/>　　慕清霄并没有特地去仔细查探，魂殿对灵魂感知极其敏感，稍微小心恐怕就会引起他们的注意。
<br/>　　想杀韩枫，必须一击必杀，否则定会遭到围杀，想到此处，慕清霄就将气息完全收敛起来，开始入侵。
<br/>　　斗王与斗皇级别，对现在的他来说，根本就没有威胁，隐匿气息后，直接穿过重重包围圈，向着中央的府邸而去。
<br/>　　中央院落中种着几颗树木，枝繁叶茂，倒是隐藏身形的好地方，悄无声息的落在树梢上，视线就在院落中扫视起来。
<br/>　　中央院落倒是没有暗哨，唯有高楼中有一股气息，七星斗皇，慕清霄眼底闪过一丝杀意，住在最中央处，有斗皇的实力，也就只有韩枫了。
