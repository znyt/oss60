　　十天一面的改造速度在时限只有两个月的紧急状况下并不算快，但在众人看来狼牙工程队的可怕程度，已经不亚于狼牙战团，光是这推翻重建的架势，无论是胆量还是手法，都在此震惊众人。
<br/>　　也让文奇胜等人对狼牙的想法重新刷新，或许他们从来都没有真正的了解过这个组织！
<br/>　　原来狼牙强的不仅是战团，它内在的一切都强大无比，简直就像是开了挂！
<br/>　　接下来的二十五天里，狼牙工程队就将南署基地剩下的三面墙全部改造完毕，与第一面墙一样的改造方式，并且还提前了五天完工，当段氏基地前来送物资的人看到时，哪怕早已经有了心理建树的他们，也不得不叹服。
<br/>　　先不说这大动干戈的改造是否能增强冲击受力，单凭这改造过后的内墙和外墙结合，就让人轻而易举的看到了狼牙面对这次大尸潮的有备而来，尤其是外墙内部犹如迷宫般的小型隧道，则是让很多人倍感好奇。
<br/>　　狼牙战团将会如何利用这些隧道？
<br/>　　当四面墙全部改造完毕后，和上的科技部就一批又一批的大箱子送往了南署基地，而后在那些段氏基地军官目瞪口呆的目光中，将各式各样不知道有何作用的漆黑玩意儿，安装在了南署基地的外墙之内。
<br/>　　那些段氏基地的军官根本来不及询问，只来得及看一眼就不得不立即离开，因为苏行的工程队又开始动工了！
<br/>　　改造完四面城墙之后，他们就将目标放在了基地内部，临近城墙的一座座房屋都被强行拆掉，而后变废为宝将其快速砌成了一座座高台，这些高台的数量极多，近乎每隔几米就有一个，又将内墙之内围城了一个圈。
<br/>　　这回不用解释，很多人都看懂了这些高台的作用，显然就是为射击队准备的！
<br/>　　这些高台的改造又耗时十天，这时候距离楚涵预计的大尸潮来临时间，只剩下了十五天。
<br/>　　并且谁也不确定尸潮会在哪一天来临，提前也说不定……
<br/>　　这时候的苏行工程队还在对南署基地做最后的改造，他们在底下打通了很多条隧道，又将很多房屋分拆或是链接而起，如果有人能够乘坐直升机从高空往下看，则会发现这座已经是空城的南署基地，已经逐渐变成了一个军事要塞！
<br/>　　并且，还是呈现出迷宫形式的军事要塞！
<br/>　　只是眼下改造还未全部完成，在大尸潮来临的时间越来越近时，段氏基地余留下的那些人，已经开始人心惶惶。
<br/>　　这些没有能力参与战斗的人数量也不少，全部被鲁初雪调配到了后勤亦或是其余岗位上，新兵在训练，他们也没有闲着。
<br/>　　而那些已经训练了一个多月的五万新兵，在杨天的警戒队步步带领，及刘玉定的军法处时刻紧盯的状态下，则是稳步又快速的成长起来。
<br/>　　新人类的数量在短短一个月的时间里增长到了三万，剩下的两万人中有一半在接下来的半个月中也步入了新人类，剩下的一万人则是将进行最后冲刺，争取在两个月期限到达之际成为新人类。
<br/>　　那些早已步入新人类的士兵，狼牙也依照了他们每日体能的增长幅度，开始有计划的分配等阶提升剂。
<br/>　　当时间到了五十五天，距离大尸潮预定来临的日子只有五天之时，这批五万人队伍终于全体步入了新人类，并且不少人都在等阶提升剂的帮助下增长战力。
<br/>　　这时候的五万人对于狼牙战团来说依然是新兵，但比之两个月前，却是有着天翻地覆的变化。
<br/>　　段氏基地的军官之前还对狼牙的训练模式存在质疑，但当他们亲眼目睹了这五万人的进步，将狼牙的训练模式尽数收入眼底的时候，才不得不佩服这个战团的可怕和厉害之处。
<br/>　　并不是单纯的极限训练让人喘不过气，而是有一定依据和合理安排的，将这些人的潜力全部激发出来。
<br/>　　能做到这一点的，那些分配到每一个十人小队的狼牙警戒队成员，功不可没！
<br/>　　每一名小队长都牢记自己队伍内的每一个队员情况，五千警戒队成员竟然一次差错都没出，愣是完美的将这五万菜鸟带了出来，并且在不断实战的训练中，这些小队成员之间已经形成了一定默契，团队作战越来越令人期待。
<br/>　　对比这五万人由经警戒队训练，另外一批狼牙三万新兵队伍，成长的程度则就只能用恐怖来形容！
<br/>　　他们与五万新兵的情况不同，从分界线一路狂奔而来的旅途本就艰苦无比，而带领他们翻山越岭的人不是一直在狼牙地界守护的警戒队，而是真正至始至终战斗在前线的狼牙战团。
<br/>　　三战队，射击队，侦查队，防守队，精英队，这些队伍每一批人都闯下过令人震惊的战绩和荣耀。
<br/>　　狼牙三战队之间更是一直存在竞争，在训练新兵的同时，他们更是不忘本，一路上相互较量的那叫一个精彩绝伦！
<br/>　　于是乎这跟随着狼牙战团一路飞奔而来三万新兵，不仅全体在一个月内就步入了新人类，甚至还在接下来一个月不到的时间里，全部经历了等阶提升。
<br/>　　每个人的基础不同，所以有着差距，有些人只经历了一次等阶提升，到达了二阶新人类，而有些人则是一跃好几阶，成长速度惊人到都能跟三战队中的最弱者斗上几场。
<br/>　　这三万人的行径路程不仅段氏基地的人在不断关注，文奇胜等人更是好奇无比，谁都想知道狼牙战团究竟会将这三万人训练成什么样。
<br/>　　但当他们看到这批队伍一路飞沙走石，走着一条笔直的直线道路直奔段氏基地，屠了一个又一个充满丧尸的村庄城镇，甚至还穿过了好几座有着大几百万丧尸群的城市之后，文奇胜等人的心情就从一开始的不可置信，渐渐变成了麻木。
<br/>　　分界线到段氏基地，段氏基地再到南署基地，愣是被这三万新兵开出了一条路！
