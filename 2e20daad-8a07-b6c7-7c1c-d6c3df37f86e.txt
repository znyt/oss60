　　啊……
<br/>　　杀杀杀……
<br/>　　巨人身上无数面孔嚎叫，发出嗡嗡无比嘈杂之声。这声音能影响人的心神，普通人只要听一下，立刻就会精神崩溃而死。
<br/>　　就连掌握巨人的比杨德，也被这杂乱吼叫弄的无比心烦。眉头微皱，比杨德陡然大喝：“闭嘴！”
<br/>　　轰！
<br/>　　巨人原地倒塌，陡然散开成一圈黑烟。眨眼间消失在比杨德背后。
<br/>　　“吸收岩浆热量为自己所用，非常不错的能力。如果学会，足可以应对大多数高温环境。可惜了……”比杨德喃喃自语。
<br/>　　能够在岩浆中泡澡，这招秘术威力强大。比杨德不想出意外，不仅采取偷袭策略。而且一出手，就用上了最强一击。
<br/>　　魔像是比杨德模仿自己父亲，猎人协会会展涅特罗的招式。
<br/>　　尼特罗的百式观音，出手时身后便会出现观音巨像，威力惊人。是他个人意志与体魄高度凝聚而成的产物。
<br/>　　他没有尼特罗十年如一日的坚持锻炼，他不是那种追求极致的武者。
<br/>　　不过比杨德也有自己的办法，吸收天地负面情绪，借助他人的精神凝聚自己的魔像。成就他自己的招式——魔神之像！
<br/>　　这是他的最强单体攻击！
<br/>　　“重力水晶是拿不到了。”比杨德颇为遗憾。不仅失去了学习那种秘术的机会，而且重力水晶也没拿到。
<br/>　　杀了揍敌客家的小子，却没得到什么好处，有些亏。嗯，可以从卡金帝国身上找回一些损失。毕竟是他们要求杀人的，当然需要付出报仇。
<br/>　　比杨德转身准备离开。
<br/>　　砰！
<br/>　　一股巨大力道骤然击中他后心，比杨德身体好似炮弹一般被打飞。笔直的射入不远处黑红地面。
<br/>　　轰！
<br/>　　地面烟尘滚滚。
<br/>　　呼！
<br/>　　一股紫黑色烟气骤然扩散，巨大的魔神雕像快速汇聚。
<br/>　　“还没死？”看清对面身影，比杨德瞳孔一缩。
<br/>　　黑红色圆形光膜笼罩全身，隐约可见里面人影。眉心处火焰纹路越发鲜亮，好似要真正烧起来一般！
<br/>　　没人比他更清楚自己一击的威力。
<br/>　　他的魔像虽然走了捷径，力量上比不得百式观音的凝练。但他同样另辟蹊径，给魔像配上草雉剑，杀伤力骤然上升。
<br/>　　多年来，凡是与人交战，只要施展出魔像抚斩，敌人无不被他斩成两半！苏重竟然完好无损！
<br/>　　那光膜的防御力竟然那么强？
<br/>　　“一剑杀不死，那就多砍上几剑。”比杨德脸色恢复平静，心里不禁没有失望，反而有些兴奋。
<br/>　　没死才好，这说明光膜技能很强大。自己学过来，实力能更近一步，就能适应高温环境。黑暗大陆上，可是有很多露在地表的岩浆河流。有了那黑红光膜一样的能力，他就能探索跟多的地域！
<br/>　　“这老家伙是哪根葱？”苏重一肚子的火气。
<br/>　　他钻进地下岩浆湖，不断吸收地热修炼魔龙七段。经验值蹭蹭蹭的往上窜，身体素质不断发生蜕变，朝着某种不可思议的境地进化。
<br/>　　虽然疼痛，但效果惊人。他正痛并快乐着呢，当头就被人砍了一刀。像是个皮球一样被砸进岩浆深处。他如何不生气。
<br/>　　上来就给了这老家伙一下狠得，没想到对方竟然完好无损？
<br/>　　“那紫黑巨人是什么东西，怎么有些眼熟？”苏重盯着对面巨人看，他想到了百式观音。
<br/>　　旋即苏重就发现了巨人身上的人脸，扭曲狰狞的面孔，陡然散发出一股邪恶意念。饶是苏重精神强大，突如其来之下，被晃的愣了一下神。
<br/>　　比杨德眼中精光骤然一闪，好机会！
<br/>　　毫不犹豫，腰间长剑骤然出窍。
<br/>　　魔像·千斩！
<br/>　　嗡！
<br/>　　在这一瞬间，比杨德不知道挥出了多少刀。身后魔像紧随他的动作，十多米长刀挥舞地快若闪电！
<br/>　　身前空间内，骤然多出密密麻麻紫黑色刀光。凌厉凛冽的刀锋，铺天盖地的冲向苏重。
<br/>　　叮叮叮……
<br/>　　刀光好似瞬移一般，凭空出现在苏重身边，把他团团围后当头斩下。火星骤然绽放。
<br/>　　本来半径有两米的气魄圆球，被狂暴攻击压的快速收缩。眨眼间就缩小了一半还多，露出了里面全身光秃秃的苏重。
<br/>　　他没穿衣服，他没有头发，就连眉毛也没有！
<br/>　　苏重刚开始修炼的时候，全身的衣服就被烧光了。他为什么顶着气魄光膜不收，没穿衣服啊！
<br/>　　这下好了，全曝光了！
<br/>　　“老家伙，给我去死吧！”
<br/>　　暗杀拳·崩山！
<br/>　　嗡！
<br/>　　气魄光膜好似吹了气，猛然胀大。前端迅速凸出一点尖，好像有人正从内戳气球。下一刻，光膜陪戳开一个口子。
<br/>　　呼！
<br/>　　狂暴的震荡力量，夹杂着黑红气魄，猛然爆发出来，笔直射向比杨德。
<br/>　　黑烟迅速汇聚成一面圆盾，死死挡在比杨德身前。四周烟气缭绕如火焰，正中处一个恶鬼脸狰狞大笑！
<br/>　　魔像·八尺镜！
<br/>　　轰！
<br/>　　黑红色暗杀拳和紫黑色圆盾悍然撞击在一起。一股波纹状冲击波猛然扩散开去，整个地下空间内突兀刮起一阵飓风。
<br/>　　“去死吧，暗杀拳！”一声大吼，道种震颤，苏重震劲再次爆发。
<br/>　　轰！
<br/>　　圆盾轰然破碎，劲风毫无阻碍的压向比杨德。他脸颊被吹得如布匹般起伏，眼中满是惊骇。自从他凝聚魔神之像，还是第一次被人正面击溃！
<br/>　　砰！
<br/>　　好似皮球被大锤轰击，比杨德笔直倒飞而去。
<br/>　　落地之后，双腿肌肉以肉眼可见的速度变粗，肌肉血管吐出好似要裂开。

<br/>　　坚硬的岩浆地面猛然炸开，借着反震之力，比杨德身影瞬间消失在原地。他没攻击苏重，而是转身就逃！

<br/>　　当他快速奔跑是，气魄竟然能够减小风阻？
<br/>　　空气中就好像有了一个无形通道，他迈出一步，就挤入那条通道，再出来是已经是遥远的另一端！
