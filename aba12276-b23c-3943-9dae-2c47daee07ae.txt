　　药尘本是远古八大种族之一药族的族人，因被人陷害而驱逐出族。
<br/>　　流落中州后，结识风闲，闯荡大陆，凭借自己的努力，打出中州第一炼药师的名号，且实力达到九星斗尊巅峰，期间帮助风闲改善身体，成为一代强者。
<br/>　　药尘也因此出名，成为了药尊者，创立了星陨阁，成为阁主，后去远古遗迹发现一个孤儿，取名韩枫并收为徒。
<br/>　　因喜好自由，独自游历斗气大陆，被徒弟韩枫陷害，被魂殿追杀，失去肉体，寄存于萧炎母亲的遗物戒指之中沉睡。
<br/>　　慕清霄伸出手来，道：“把你手中的戒指给我。”
<br/>　　萧炎沉吟片刻，有些举棋不定，目光落在这枚黑色古朴的戒指上，难道这是什么宝物不成，看面前的青年也没有出手抢夺的意思。
<br/>　　可能是因为两人都是来自地球，萧炎选择了信任，于是慢慢摘了下来，递给慕清霄后，道：“难道这枚戒指有什么特殊的吗？”
<br/>　　“自然特殊。”
<br/>　　话语刚落，慕清霄掌心就窜出九霄琉璃焱，将戒指包围，道：“你是想隐藏到什么时候，难道是想让我把你的灵魂也焚烧殆尽吗？”
<br/>　　“小娃娃，你这是什么火焰，赶快收回去。”
<br/>　　周围凭空冒出一道苍老的声音，萧炎顿时吓一跳，只见古朴的黑色戒指上空，一道透明的苍老身影飘了出来，看向慕清霄的神情明显有些惊惧。
<br/>　　看到药尘的灵魂，慕清霄一愣，他也是第一次看到灵魂状态的人类，看来修为达到一定的层次，灵魂果然能够脱离肉身生存啊。
<br/>　　“你是什么人，年纪明明不大，却是斗皇强者，而且这火焰居然令我感到余悸，还有你是怎么发现我的？”
<br/>　　药尘刚出现，便喋喋不休的询问起来，对于面前这位神秘的青年，心中甚是好奇。
<br/>　　斗皇！
<br/>　　听到药尘的话语，萧炎惊得下巴都快掉下来，他的同乡竟然是斗皇强者，这可是遥不可及的存在啊。
<br/>　　慕清霄并没有理会药尘，虽然后者生前是斗尊巅峰，但现在却只是一缕虚弱的灵魂，只要他想，立刻就能将药尘的灵魂焚灭。
<br/>　　目光看向萧炎，道：“你现在应该知道，自己不能修炼的原因了吧？”
<br/>　　萧炎在怎么蠢，现在也知道前因后果，脸色阴沉的盯着药尘的灵魂，道：“原来，我体内斗之气莫名其妙的消失，是你搞的鬼！”
<br/>　　药尘有些尴尬，道：“小娃娃，你别生气，我也是迫不得已啊！”
<br/>　　想起家族中的羞辱，多年来的冷嘲热讽，萧炎心中一阵酸楚，那能是一句迫不得已就能了解的，当场就向破口大骂，却被慕清霄及时制止。
<br/>　　“你先别急，他并没有说谎，如果没有你的斗之气，他能不能醒来都是问题，我有些事情要与他商量，待会让他补偿你。”
<br/>　　听到补偿，萧炎心中一动，只能压下心中的怒火，点头同意，道：“好吧。”
<br/>　　而且，萧炎自己也清楚，他现在就是个弱者，在这个弱肉强食的世界中，弱者没有说话的权利，更何况慕清霄帮他解决了最大的烦恼。
<br/>　　见萧炎冷静下来，慕清霄满意的点了点头，后者的表现可谓是极其沉稳，往后他的成就与心性脱不了干系，哪怕是穿越者，没有强大的心性，想成为强者也是极难的。
<br/>　　随后，慕清霄的视线就落在药尘身上，道：“将你的骨灵冷火给我。”
<br/>　　闻言，药尘瞳孔一缩，当慕清霄白天在萧家大厅出现的时候，他就有种不好的预感，察觉到他的存在后，这种预感更加强烈。
<br/>　　现在看来，面前的青年知道很多东西，根本就是冲着他的骨灵冷火来的，药尘眼中满是凝重，道：“想要我的异火，阁下未免有些过分了吧？”
<br/>　　九霄琉璃焱再现，瞬间将药尘的灵魂包围住。
<br/>　　慕清霄面无表情，道：“我可不这样觉得，你的命都被我掌控着，我只是要异火，难道很过分吗？况且我杀了你，照样能够获得异火，何必多此一举？”
<br/>　　察觉到骨灵冷火的恐惧，药尘眼中满是惊惧，静下心沉吟起来。
<br/>　　片刻后，他觉得慕清霄说的不无道理，明明能够直接杀了自己，现在却给他讲条件的机会，看来有某种原因限制着他！
<br/>　　药尘猜的不错，慕清霄就是在恐吓他，真要杀了药尘，他依旧下不去手，不是心生怜悯，而是怕引起蝴蝶效应，大幅度改变剧情的轨迹。
<br/>　　这里可是斗破苍穹世界，不是普通小世界。
<br/>　　普通小世界中，他怎样胡作非为都没事，因为一切都在他的掌控中，总而言之，还是他太弱了，斗皇强者在这里勉强能够自保而已。
<br/>　　药尘道：“你先将火焰收起来，我们可以好好谈谈。”
<br/>　　“老狐狸。”
<br/>　　慕清霄暗骂一声，最终还是将九霄琉璃焱收了起来：“有什么要求你就提吧，只要在我的能力范围内。”
<br/>　　药尘深深的看了一眼慕清霄，眼底闪过一丝赞赏，年纪不小，心性就如此可怕，真是长江后浪推前浪啊！
<br/>　　“我的要求也不高，以你的能力，或者说等你在成长一些，便能够轻松办到。”
<br/>　　慕清霄也不说话，依旧是面无表情，情绪完全不表现在脸上。
<br/>　　药尘感到一股深深的无力感，道：“第一件事，帮我杀掉欺师灭祖之徒，韩风。”
<br/>　　慕清霄想也不想就点头答应：“可以。”
<br/>　　药尘一愣，接着心中就是一喜，原来这小子这么好说话，于是就将韩风的信息告诉了他。
<br/>　　其实不用药尘说明，慕清霄也知道来龙去脉，这个要求不管药尘提不提，他都会去找韩风，因为后者手中有他想要的异火‘海心炎’。
<br/>　　“咳咳。”
<br/>　　药尘干咳两声，道：“至于第二件事情，就是助我复活。”
<br/>　　在原著中，药尘收萧炎为徒弟，初时肯定是抱着某种目的的，却也是别无选择，无奈只有在萧炎身上赌一赌。
<br/>　　药尘毕竟人老成精，当初受到最信任的徒弟杀害，心中自然更加谨慎，选择萧炎为徒，自然要事先清楚他的性格。
