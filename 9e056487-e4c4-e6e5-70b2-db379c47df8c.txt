　　“没什么意思，第七株，易某出价20万两，顾家主想要的话，尽管竞拍便是。”
<br/>　　萧逸淡淡地说着。
<br/>　　“你...”顾家家主脸色彻底黑了下来。
<br/>　　他刚才之所以强硬地说要第七、第八株，就是想以15万两的最低价，直接得到两株八宝玲珑参。
<br/>　　“紫炎，卖我顾家一个面子可否？”顾家家主稍稍放缓了语气。
<br/>　　“你也是闻名北山郡的天才之一，应当知道我儿顾长空，乃裂天剑派剑堂首席。他日若有机会，我引荐你们结识一番，也是好的。”
<br/>　　顾家家主，嘴上说得好听，实则，只是想用顾长空的名声来压萧逸。
<br/>　　这时，萧逸身旁的白家主冷声道，“顾家主此言差矣。”
<br/>　　“据我所知，顾长空十年前加入裂天剑派，苦修十年，方成剑派首席。”
<br/>　　“十年前，不过是北山榜12，也好意思说出来显摆。”
<br/>　　白家主知道顾家家主话中意思，自然不会让萧逸吃亏。
<br/>　　顾家家主闻言，脸色冰冷，不再说什么，直接报价，“25万两。”
<br/>　　“30万两。”萧逸毫不犹豫地出价，而后继续道，“顾家主尽管出价，高于30万两，易某便不与你争了。”
<br/>　　“你...”顾家家主再次脸色一冷，这个价格，其实他已经不想要了。
<br/>　　但，他本想继续加价，他知道，易霄会继续出价的，他想让易霄多亏些钱。
<br/>　　可是，萧逸的这句话，却直接打断了他的念头。
<br/>　　他不知道萧逸的话是真是假。
<br/>　　他不敢赌。
<br/>　　半晌，他只能愤愤地坐下，停止了叫价。
<br/>　　一层的宾客，也没有出价。
<br/>　　他们很清楚，这是顾家和紫炎，或者顾家和白家间的争斗；他们没资格参与，也更不想得罪这两大势力。
<br/>　　起码为了一株八宝玲珑参，不值得这样做。
<br/>　　毕竟八宝玲珑参是要付钱的，又不是白拿。
<br/>　　最终，毫无疑问，第七、第八株，皆被萧逸以30万两的价格收入囊中。
<br/>　　第九株，同样是30万两的价格。
<br/>　　九株八宝玲珑参，加起来恰好是二百多万，萧逸身上的钱，一次花了个精光。
<br/>　　但，能将九株八宝玲珑参全部得到，绝对是值得的。
<br/>　　一旁的白家主，看着萧逸一次次给钱，脸色的惊讶之色一闪而过。
<br/>　　“易霄小兄弟，看不出来阿，你有钱到这个地步。”
<br/>　　“二百多万两，就算是顾家这种大势力，若一次拿出来，也要伤筋动骨。你倒好，眉头都不见皱一个的。”
<br/>　　“呵呵。”萧逸笑笑。
<br/>　　拍卖会，有条不紊地进行着。
<br/>　　接下来的物品，每一件开始竞拍时，一楼的宾客，都会下意识地抬头看一眼贵宾席上的萧逸。
<br/>　　他们真怕易霄忽然又蹦出个高价来，他们根本抢不过。
<br/>　　萧逸苦笑一声，后面的拍卖品，他一件都没有出价。一来是不感兴趣，二来是没钱了。
<br/>　　见状，一层的宾客才算放心，纷纷露出了满意的笑容。
<br/>　　在他们看来，萧逸还是会做人的，前面将九株八宝玲珑参悉数买了，后面的拍卖品则一件都不争，也算是给面子。
<br/>　　期间，有一亚灵器，倒是稍稍引起了萧逸的注意。
<br/>　　名为神行铠甲，防御不俗，且可短暂地在半个时辰内将自己的速度提升至洞玄五重。
<br/>　　对于低级别的武者来说，乃是相当不错的保命、逃命宝贝。
<br/>　　最终被一层的一位家主买去。
<br/>　　时间一分一秒过去。
<br/>　　拍卖会，已经持续了数个时辰。
<br/>　　数十件珍贵的拍卖品，也悉数被竞拍下。
<br/>　　不多时，终于到了最后一件压轴宝贝，也彻底将拍卖会的气氛推至最高。
<br/>　　拍卖师，有着他自己的拍卖手段。
<br/>　　第一次气氛的高潮，乃是玄阶巅峰武技的出现；第二次，乃是九株八宝玲珑参；第三次，便是这最后一件拍卖品。
<br/>　　“有请最后一件拍卖品。”
<br/>　　拍卖师刚说完，二层五个贵宾席中的大人物，一个个都脸色凝重了起来。
<br/>　　萧逸看了眼白家主的反应，知晓，他等的，就是这最后一件拍卖品。
<br/>　　当四个壮汉将拍卖品放到展览台，拍卖师高声道，“最后一件拍卖品，星芒石。”
<br/>　　几乎是拍卖师话语落下的瞬间，整个拍卖会，变得落针可闻。
<br/>　　所有人耳中，都回荡着‘星芒石’三个字。
<br/>　　拍卖师似是早就料到众人的反应，继续道，“星芒石，起拍价二百万两；每次加价，不得低于十万两。”
<br/>　　他高声说着，目光却是看向二层。
<br/>　　很显然，他知道整个拍卖会，只有二层的大势力有资格出价。
<br/>　　“星芒石。”萧逸此刻心头也是一惊。
<br/>　　他曾经想过五个贵宾席中的大人物，等待的物品绝非凡物。
<br/>　　却万万没想到，竟是星芒石。
<br/>　　星芒石，乃是比灵器更为珍贵的东西。
<br/>　　因为，它能让灵器进阶。
<br/>　　无论是下品灵器还是中品灵器，加入星芒石稍稍炼制一番，立马就能跃升为上品灵器。
<br/>　　简单来说，它是一种品阶非常高的材料，且又非常神奇。
<br/>　　这已经不是单纯的宝贝了，而是重宝。
<br/>　　它的珍贵程度，远超一切。据萧逸所知，它在北山郡历史上出现过的次数，不足三次。
<br/>　　算上这一次，便是第四次。
<br/>　　难怪此次的拍卖会盛况如此空前，前面所谓的数十件宝物，不过是它的点缀罢了。
<br/>　　也难怪盛宝商行要联合本土的两个强大商业家族联手举办，等闲势力，可保不住这样的重宝。
<br/>　　星芒石的底价是二百万两，但萧逸知道，最后的价格，必然是一个更高的天价。
<br/>　　果然，二层的贵宾席上，已经立刻有人出价。
<br/>　　“200万两。”
<br/>　　“220万两。”
<br/>　　“250万两。”
<br/>　　价格持续飙升。
<br/>　　一层的宾客，纯粹只是在看热闹，根本没资格出价。
<br/>　　“300万两。”这时，白家主也出价了。
<br/>　　“如无意外，星芒石的竞拍，会在这里结束。”白家主沉声说道。
<br/>　　另外四家，都是百武城内的大家族，家主乃洞玄九重修为。
<br/>　　而他们能一次性拿出手的真金白银，顶多在250万两左右。
<br/>　　白家直接叫价300万两，足以震住他们。
<br/>　　有白家主这个破玄境武者坐镇的白家，其势力和财力，自然是远超另外四家中的任何一家的。
<br/>　　场面再次静了下来。
<br/>　　白家主展露出笑意，如无意外，这颗星芒石，归他了。
<br/>　　但所谓意外，必然是意料之外的。
<br/>　　“310万两。”忽然，王家的贵宾席中，传出了叫价声。
<br/>　　五个贵宾席，除了白家外，分别是王家、赵家、钟家、顾家，皆是百武城内的大家族。
<br/>　　“310万两？”白家主皱起了眉头，自语道，“我最不想看到的情况，出现了。”
<br/>　　“他们联手了？”萧逸问道。
<br/>　　白家主点点头。
<br/>　　果然，王家叫价之后，赵家不再出言；在百武城内，他们两家的关系一直很不错。
<br/>　　“320万两。”这时，顾家出价了，钟家则没有出价。
<br/>　　显然，顾家和钟家联手了。
<br/>　　“这下麻烦了。”听着顾家和王家的叫价，白家主脸上写满了没有把握的表情。
<br/>　　“400万两。”下一秒，白家主当机立断，喝了一声。
<br/>　　拍卖，除了是一场财富的竞争外，更是心理的博弈。
<br/>　　白家主一下子以高出80万两的价格出价，希望能震住另外四家。
<br/>　　只是，星芒石如此重宝，谁都不可能轻易放弃。
<br/>　　“410万两。”王家再次出价。
<br/>　　“430万两。”顾家毫不犹豫地加价。
<br/>　　“450万两。”白家主喝了一声。
<br/>　　但萧逸明显能看到，白家主几乎是咬着牙出价的。显然，这个价格，已经到了他所能承受的最大程度。
<br/>　　450万两，已经是非常骇人的数字。
<br/>　　场面再次寂静下来。
<br/>　　萧逸也在一旁暗暗咂舌，他万万想不到，今日会出现一场如此激烈的竞价。
<br/>　　不多时，王家家主站了起来，高声道，“白家主，如果我没有猜错的话，450万两，已经是你的极限了吧。”
<br/>　　“哈哈，我王家出价460万两。我就不信还有人能比这个价格更高，若有，我王家放弃竞价便是。”
<br/>　　显然，王家和赵家的联手，也只拿得出这个价钱。
<br/>　　“该死。”白家主脸色变得无比难看。
<br/>　　450万两，确实是白家的极限了。
<br/>　　恰在此时，一道得意的声音响起。
<br/>　　“怎么，白家主不出价了吗？那么，我顾某人可就不客气了。”顾家家主得意地笑了起来。
<br/>　　“王家、赵家，你们也该放弃了。”
<br/>　　“我出价，480万两。”顾家家主的声音，响彻了整个拍卖会。
<br/>　　白家、王家、赵家，三家的人，脸色变得极为难看。
<br/>(本章完)
