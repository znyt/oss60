　　在楚涵正打算接受又一次长时间的黑暗和安静时，这间小屋却来一个出乎意料的人。
<br/>　　猎王早已察觉到事情的不对劲，在猎者联盟总部遭受攻击之后，他没有第一时间将分散在各处的广大猎者聚集，也没有回到总部查看情况，而是交代了几名高级猎者一个秘密任务，然后就这么忽然消失，出现在了狼牙基地。
<br/>　　他没有找尚九娣，而是直接找上了陆玟丞。
<br/>　　作为情报部的部·长，陆玟丞是唯一知道楚涵与猎王关系的人，当然从第一次从各方情报上察觉到不对，到被楚涵找到亲自摊牌后，陆玟丞就早已对楚涵崇拜到了最高峰。
<br/>　　暗部！
<br/>　　又一暗部！
<br/>　　谁能想到猎者联盟，竟然也是狼牙的一部分，而猎王更是楚涵一开始就拉拢的合作对象。
<br/>　　说是合作，可那三七开的经济分配足以说明，猎王根本就是在帮楚涵赚钱和收集情报而已，难怪狼牙这两年明明没有过多生意投资的往来，生产各种药剂和科技的成本也巨大，却莫名其妙的越来越有钱。
<br/>　　除了风花雪月城，还有个猎者联盟啊！
<br/>　　所以当猎王悄悄出现，陆玟丞二话不说就带他来到了木屋之内。
<br/>　　“昏迷多久了？”猎王望着床·上的楚涵，皱眉深思。
<br/>　　“半个月。”陆玟丞回答道。
<br/>　　“你先出去，我跟他聊聊。”猎王忽然说出了一句莫名其妙的话。
<br/>　　陆玟丞很想说‘你是不是在说笑’，但看到猎王认真的神色，还是点头走出木屋，给他留下了私人空间。
<br/>　　当屋内只剩下自己和昏迷的楚涵时，猎王这才放松警惕的坐下，淡笑道：“你这昏迷的真是时候，外面世界都大乱了知道不？”
<br/>　　黑暗中的楚涵睁开眼，无奈的自言自语：“聪明点的就直接汇报，最好来点刺激的事情，我实在是太无聊了。”
<br/>　　并不能听到任何楚涵内心言辞的猎王，先是自顾自的嘲笑了楚涵一顿，而后神情才渐渐严肃起来：“我也是习惯了每过一段时间找你汇报情况，虽然知道你听不到也醒不过来，但有很多事憋在心里我堵得慌啊！”
<br/>　　“首先呢，你花了大价钱改造的猎者联盟总部，被不明势力毁了。”猎王说着，好似知道楚涵会暴跳如雷一般，立即接话道：“你先别激动，咱也只是损失了一大笔钱而已，人没死多少，哦对不起，我忘了你这人最在乎钱。”
<br/>　　“其次呢就是在南部边境，有几百名猎者遭受了丧尸的针对性攻击，都死了，这是猎者联盟成立以来最大的一次打击。”
<br/>　　“两件事看起来毫无关联，很看好你的几个老人相当重视，他们分析的结果是幕后有神秘家族，也就是对猎者联盟最仇视的白家。”
<br/>　　“我一开始也是这么认为的，可总觉得有什么不对，段江伟提议让我把猎者都聚集起来，方便管理也防止消息爆出后造成恐慌，但我后来仔细想了想，没有听取他的意见。”
<br/>　　猎王的语速很快，正色道：“我暗中调动了一批猎者行动，看上去就像是接到命令的聚集，实际上却是我的一次试探，结果你猜怎么着？”
<br/>　　“那批猎者遭到了埋伏，来自大批丧尸军团的埋伏，而统领这批丧尸行动的，是十人以上的超战力异种。”
<br/>　　“当然那批猎者有死有伤，对你我而言又是一次损失，不过试探之后事情不仅没有变清晰，反而更复杂。”
<br/>　　“你觉得这像是白家指示的不？”猎王自顾自的问道，又自顾自的回答：“我觉得不是，但又想不通，因为那些丧尸数量太多了，多的不像话，甚至也根本不可能出现在那片区域，最重要的是我还观察到，部分丧尸不像是来自华夏本土，你知道的，丧尸在等阶变高之后，会慢慢的趋近于人类体态。”
<br/>　　说到这里，猎王皱起眉道：“你怎么想？我的想法比较异想天开，总感觉异种和丧尸这两大势力，似乎联手了。”
<br/>　　一直听着的楚涵并没有任何情绪暴走，虽然在听到有关猎者联盟的两则事件之后很惊讶，但随后也渐渐接受心态平复，这已经不是他所熟悉的上一世，这一世的一切因为他的重生而改变，猎者联盟会遭受如此重创也不是没可能。
<br/>　　“当然这些都是我的猜测。”这时候的猎王以及汇报完毕，他起身打算离开，忽然又折回道：“对了，还有一则噩耗，你听了可别难过。”
<br/>　　“何培源死了，白允儿没有救出他，我不在场所以不知道发生了什么，但白允儿此时在哪我也不知道，不过我有接到了步纱的密函，江凌轩命大的被雪水冲下山，到了她的小屋，被就下了。”
<br/>　　“步纱最近心情有点莫名其妙的亢奋，也不知道遇到了什么事，自从听你的话找到她，安排她隐世在那里之后，她明明就一直很暴躁来着……”
<br/>　　听到白允儿失踪楚涵一点不奇怪，这个女孩要是能被猎王找到，那她也就不是那个一直处于人类巅峰的人间杀器白允儿了。
<br/>　　只是听到步纱的名字后，楚涵诡异的沉默异常，他早就知道步纱的所在地，但却一直没有去看过她，忙于战事只是借口，更多的是楚涵不知道如何去面对。
<br/>　　那天他到底对这个年轻的女孩做了什么，他有些不敢去想。
<br/>　　只是传说中的蛇蝎妖姬美阎罗，却与上一世的状态大有不同，步纱虽毒，却在某些方面莫名的单纯。
<br/>　　他也算是，改变了一个女孩的人生？
<br/>　　猎王将事情交代完之后就离开，虽然很多事看似无所谓，但作为猎者联盟明面上的创始人，他的忙碌也不是一般人能理解的。
<br/>　　猎王走后的楚涵，则是陷入了长久的沉思，从刚刚猎王所透露的几件事来看，很明显外面的世界又要有一场狂风暴雨。
<br/>　　也不知道这个节骨眼上已经离开华夏中心，到达了西南边境的狼牙战团，到底是是福是祸？
