　　　　三人的出现，立刻被道风察觉到，但自己正在跟黎山老母死战之中，无法分心，于是喊出了一声简短的口令。
<br/>　　不远处正在厮杀的十二门徒中的几个，立刻飞身而起，试图挡住三人。
<br/>　　苏沫在空中出手，几颗红色火球一般的灵力波冲射出去，将几人短暂地困住，人影从他们身边飞快地掠过。
<br/>　　突然一道橘红色的剑气，后发先至，朝着三人射来，苏沫布置的灵力波，几乎被一瞬间击碎，剑气继续奔袭，眼看着就像刺中苏沫，她身边那个黄脸男子突然折返，双手像打太极似的在空中朝着一个方向拨动起来，双掌之间蓝色的气息喷涌而出，形成了一道屏障，剑气打在上面，震出了一道强大的反震波动。
<br/>　　红光折返，重新回到建文帝的手中，那黄脸男子看了他一眼，眼神中带着一丝冷漠，双手分别提着张云和苏沫，继续朝山下飞去。
<br/>　　建文帝手提鱼肠剑，站在山口，望着黄脸男子的背影，表情十分凝重，内心深处反复思考着一个问题：这个实力几乎不亚于道风之下的男人，到底是谁？
<br/>　　三人飞身下到山谷中，朝着芮冷玉等人杀过来。
<br/>　　钟馗正在全力推动旗杆，根本无暇顾及身后几人，听见动静，立刻让萧逸云等人前去抵挡。
<br/>　　萧逸云带着天子殿众人冲了上去，挡在三人前面，厉声喝道：“你们什么人，我是天子殿押司，敢对我动手么！”
<br/>　　黄脸男子看了萧逸云一眼，什么也没说，脚下动作不停，一抬手，一道黑色火焰从掌心喷出，如同一把尖刀，直接扎了过来，萧逸云急忙打出三道鬼符，与火焰一旦接触，只支撑了不到几秒钟，立刻被击打得粉碎，身后的几个金甲和银甲鬼武士立刻围上去，作法将火焰挡住。
<br/>　　萧逸云这才就回过神来，命令大家布阵，将三人围起来，展开进攻。
<br/>　　芮冷玉和扁头继续帮助钟馗一起推动旗杆，虽然两人实力有限，但是为了救叶少阳，也是献出了自己全部的力量。
<br/>　　老郭在这种地方，是完全没有一点用的，站在一边，默默观察着战局，心中越来越感到震惊：
<br/>　　本来看到这三个人从山上下来，并没有太过在意，毕竟自己这边人多，金甲鬼武士是上等鬼将，三等鬼首，每一个司衙最多也就配备几名，更不要说还有几个银甲鬼武士，身为天子殿押司的萧逸云，最少也是准地仙牌位。
<br/>　　这个阵容，已经十分强大，而对面只有三个人，双方混战起来之后，对方三个人中间的白面小子（张云）基本上只有自保能力，苏沫差不多跟萧逸云形成兑子，也就是说，那个黄脸男子一个人抵挡住所有金甲和银甲鬼武士的进攻，仍然不落下风！
<br/>　　老郭总算明白，为什么对方只有三个人，仍然敢发起进攻了。
<br/>　　这个家伙……到底是谁？
<br/>　　老郭皱着眉头上下打量他，脑海中猛然浮现出了一个名字，心中咯噔了一下，原来……是他！
<br/>　　“老大，有人来救我们了！”橙子躺在叶少阳怀里，透过灵力网朝外面看去，虽然看不清许多，但是能看到一杆大旗，正在缓缓朝阵心靠近，几乎已经伸到面前，只剩下最后一丝距离。
<br/>　　“小天师，你那边加把劲，只差一点！！”
<br/>　　叶少阳很想回答，但是现在的他，已经连嘴巴都张不开了——头顶上悬挂的那一张最强大的灵力网，正在缓缓收紧，几乎快压缩到他头顶上空。
<br/>　　这还是他拼尽全力抵挡的结果，现在的他，连体内所有的戾气差不多都用尽了，真正的灯枯油尽，仅仅凭着一丝不屈的的意念在支撑着。
<br/>　　“老大……”橙子看着他惨白的面孔，感受到身体越来越强烈的抽搐，她知道，虽然获救的希望就在眼前，但是叶少阳撑不到那时候了。
<br/>　　“老大，我真的很没用。”两行清泪，从橙子脸上滑落，用力咬了咬嘴唇，表情随后淡然起来，从他膝盖上缓缓爬了起来，冲他笑了笑，道：“也挺好的，总算是有机会为你做点事情了。”
<br/>　　什么？
<br/>　　橙子反常的表现，让叶少阳近乎混沌的意识清醒了一些，一边苦苦支撑着不让灵力网落下，一边朝橙子看过去。
<br/>　　她早就耗尽了修为，连移动都困难，她还能干什么？
<br/>　　橙子站了起来，张开嘴，一股红色的灵气，从她口中吐了出来，化作一条鲤鱼的形状，甩着尾巴，跳到头顶上空。
<br/>　　橙子双手交握，不断变化着手印，闭着眼睛，口中念念有词，突然张开眼睛，那条灵气组成的鲤鱼甩了一下尾巴，形状散开，化作了一层结界，在两人头顶笼罩下来，瞬间挡住了灵力网的下落，虽然结界立刻摇晃起来，但看上去至少能支撑片刻。
<br/>　　叶少阳顿时感到压力轻松了许多，双腿一软，跌坐在地上，几乎脱力。
<br/>　　“什么，这是什么法术！”叶少阳心中浮起不详的预感，挣扎着爬起来。
<br/>　　“不要过来，老大，如果你碰到我，封印一破，我立刻就魂飞魄散了……”
<br/>　　橙子定定地望着他，嘴巴没用，在用神念对他说话。
<br/>　　“什么意思，什么？”叶少阳用最大的力气嚷起来，但脱力的他几乎没发出声音来。
<br/>　　“老大，你还记得鱼玄机吗？”
<br/>　　鱼玄机，巡游司主？
<br/>　　往日的一些记忆又浮上心头。叶少阳不明白，在这关头，橙子为什么提起她来？
<br/>　　“鱼玄机往生修罗道之前，把一身灵骨拆了给我……这件事你还记得吗，因为我跟她算是同类，之后我修为没有增长，是因为我的神识，与她的灵骨还没能完全融合，我一直都在祭炼灵骨，总算是祭炼差不多了，将我的元神融入了进去……”
<br/>　　将元神与鱼玄机的灵骨融合？这的确是修炼的正道，只有这样，才能把灵骨变成自己的，方能够一点点释放鱼骨中蕴藏的修为和记忆……叶少阳心中咯噔了一下，猛然想到了什么。
