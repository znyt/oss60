　　　　
<br/>　　这一路下来，曾书书也知晓，慕清霄与陆雪琪貌似不太喜与人接触，于是介绍两人时，都是一言带过。
<br/>　　经过一番交谈后，法相几人都知道，张小凡他们昨夜便被困在蝙蝠群中，直到几分钟前方才脱困，居然没有受伤，心下有些佩服。
<br/>　　李洵冷哼一声，眼中不屑之色极为明显，昨日的蝙蝠群他们也看到了，就算是他碰到也只能狼狈逃离，这群青云门的小子，怎么可能脱身，想必是在说大话。
<br/>　　燕虹也没有说话，但表情已经表现在脸上，显然也不信他们说的话。
<br/>　　法相是和尚，没有这么多计较，微笑道：“这次空桑山一事，我们三派长老本就要我们年轻一辈受些历练，如今人数已经到齐，不过青云门诸位师兄远来幸苦，不如我们先休息一日，明日一早，再进万蝠古窟查探如何？”
<br/>　　不等张小凡几人说话，慕清霄语气平淡道：“不必多等，即可启程吧。”
<br/>　　既然已经来到空桑山，他自然不想多等，天书就在眼前，而且来到空桑山的可不止他们，迟则生变。
<br/>　　李洵瞥了眼慕清霄，视线落在空灵清绝的陆雪琪身上，见她神情虽冷，可凛然中却自有俾睨众生，飘逸出尘的清丽，一时间有些呆了，他从未见过如此绝色。
<br/>　　法相一愣，却见张小凡几人都默认，目光诧异的看了一眼慕清霄.
<br/>　　面前的青年除了相貌神骏外，看不出任何特殊之处，没想到青云门却是以他为首，当真人不可貌相，海水不可斗量。
<br/>　　收回目光，法相说道：“既然如此，我们就即可启程吧。”
<br/>　　行走在空桑山上，张小凡几人心中都保持着警惕，慕清霄则与陆雪琪肩并肩行走，简直跟逛后花园一样，对此根本没放在心上。
<br/>　　边走边谈，众人也从法相口中得知，空桑山‘万蝠古窟’中那些蝙蝠乃是当年魔教畜养的异种，凶蛮残忍，性好吸血，本为魔教帮凶。
<br/>　　八百年前，魔教在此地据点覆灭后，仍有少数蝙蝠残存下来，天长日久，居然繁衍旺盛，直到有了今日的庞大规模，每出掠食都是遮天蔽日，把这方圆五百里内搞的荒无人烟。
<br/>　　不过这些蝙蝠好似都畏惧阳光，所以都只在夜间活动，白日都栖息在万蝠古窟中，昨晚张小凡他们正好碰巧遇上，若是白天上山就不会向昨晚一样危险。
<br/>　　听到此处，慕清霄剑眉挑了挑，道：“那些畜生都在万蝠古窟中，我们又如何进去查探？”
<br/>　　虽然蝙蝠对他来说没有危险，但他不得不为陆雪琪考虑，这妮子显然害怕蝙蝠这类生物，昨日吓的身子都软了，而且他也不想暴露太多修为。
<br/>　　法相迟疑了一下，青云门明显以慕清霄为首，于是道：“慕师兄，根据小僧这些日子的观察，这些畜生在白日都只倒悬于古窟洞顶，并未活动，我们或许可以试一试也不一定。”
<br/>　　听到待会还要面对蝙蝠，陆雪琪原本就白皙的脸蛋更白了一些，慕清霄左手悄无声息的放在她的腰间，轻轻拍了拍以示安慰。
<br/>　　曾书书哑然，张小凡却忍不住道：“那就是说法相师兄你也没有把握了，说不定那些家伙看到我们进洞就扑过来，那可如何是好？”
<br/>　　法相看向张小凡，眼中似乎隐隐有什么光芒闪烁，但神态依然温和，道：
<br/>　　“正是如此，小僧也没有十成把握，但师门授命，总是要去做的，试一试也无妨，不行我等就退出来，到时候另想办法，今日我与法善师弟，焚香谷两位施主本向进去打探一番，没想到正巧遇上诸位，如此也好，人多好照应。”
<br/>　　“哼！”
<br/>　　走在左侧的李洵冷哼一声，张小凡几人忘了过去，李洵却是丝毫不惧，只是当他看向陆雪琪眼神看过来的时候，神情多少才有些变化。
<br/>　　慕清霄剑眉挑了挑，筑基七层的修为也来哗众取宠，这种心性也能成为焚香谷的天才弟子，真是可笑。
<br/>　　来到空桑山中央地区，依旧是满山荒芜，沙石满地，偌大一座山上，竟连普通的鸟鸣声也听不见，怕是早已成为了蝙蝠的点心，或者早已从此地迁移。
<br/>　　半个时辰后，慕清霄等人已经找到了万蝠古窟所在，当下众人跟随在法相身后，一路小心翼翼来到万蝠古窟的洞口。
<br/>　　这里是一个巨大的半山洞穴，位置在山阴背阳处，微微向下倾斜，只有洞口出有些亮光，再往里面就是漆黑一片。
<br/>　　站在离洞口还有五丈远的地方，众人都能感觉到洞里阴风一阵阵的吹来，拂过脸颊，阴冷入骨，同时隐隐有些沙沙声，似低语似鬼泣，让人心头发麻。
<br/>　　张小凡多看了洞穴两眼，回头强笑一声，道：“如此，我们就进去吧。”
<br/>　　众人默然，法相点头道：“正是，不过此洞内危险难测，各位最好备好仙器，以防万一。”
<br/>　　事关生死，众人都是不敢怠慢，纷纷将法宝拿在手中，唯有慕清霄两手空空，陆雪琪手握天琊剑，语气清冷道：“走吧。”
<br/>　　前脚刚踏进洞穴，浓郁的阴冷气息就包裹而来，地面四周都充满湿气，有的地方凹凸不平，若是普通人在其中，很容易就困死其中。
<br/>　　此时，众人已经深处黑暗中，对于慕清霄来说，与白昼无异，只是其他人就不好说了。
<br/>　　不过，他们也都没有逞强，各自祭起仙器，散发出道道霞光，周围总算是亮了起来，看向四周，只见各处都有一坨坨散发着恶臭东西，显然是蝙蝠群的分泌物。
<br/>　　众人脸色都苦了下来，尤其是两个女子，陆雪琪与焚香谷的燕虹，更是紧皱眉头，面色苍白。
<br/>　　慕清霄微微摇头，不着痕迹的牵起陆雪琪的芊芊玉手，片刻后，众人勉强定下心来，随着深入洞穴中，周围的沙沙声也逐渐大了起来，绕耳不绝。
<br/>　　深处这种环境下，众人心里都直突突，谁也不知道会突然冒出什么，各自都保持着百分之二百的警惕。
