        贱女人生的孩子，就是下作！
<br/>
<br/>        季思妍甩手离去。
<br/>
<br/>        季雅姗转身跑进一个胡同里，背靠着爬满了青苔的墙壁，鼻子一酸，眼泪忍不住夺眶而出。
<br/>
<br/>        她变成这个样子，是被谁害的？
<br/>
<br/>        就是她季思妍的母亲！
<br/>
<br/>        今年年初的时候，她妈妈突然脑溢血被送往医院，需要近四十万的手术费。
<br/>
<br/>        她去季家借钱，在那栋宽敞的季家别墅里跪了一天，跪到膝盖痛得发麻了，她季思妍的母亲不但不借她钱，还趁机拿起鸡毛掸子对她一顿毒打谩骂。
<br/>
<br/>        季思妍的妈妈可以凶神恶煞地对她拳打脚踢，发泄这么多年来压抑在心里的怒火，那么她季雅姗呢？
<br/>
<br/>        她妈妈插足她父亲的家庭，是她妈妈的错，可她又有什么错？
<br/>
<br/>        顶着“小三女儿”的骂名，活到现在，她心里早就支离破碎。
<br/>
<br/>        没有谁会同情她，因为在别人眼里，她不值得被人同情。
<br/>
<br/>        这一切的孽，都是她妈妈自作自受种下的。
<br/>
<br/>        季雅姗听到的最多的一句话就是“你活该”。
<br/>
<br/>        是啊！
<br/>
<br/>        她是活该……
<br/>
<br/>        命贱！
<br/>
<br/>        活该步入她妈妈的后尘……
<br/>
<br/>        只不过，她妈妈是插足别人的婚姻，还妄想上位，结果没什么好下场。
<br/>
<br/>        而她只不过是被别人包养了而已，她不想上位，只想快点离开，希望自己将来能图个好未来。
<br/>
<br/>        那个男人一次性给了她两百万，包养她五年，而且只占用她每个周末那一天的时间。
<br/>
<br/>        在她未满十八岁以前，那个男人允诺不会碰她，除了接吻，他不会对她有任何过激过分的“猥琐”行为。
<br/>
<br/>        只要等到五年后协议期满，她便彻底自由了。
<br/>
<br/>        五年后、五年后……她才二十二岁，还年轻不是吗？
<br/>
<br/>        只是被人包养过的污点，恐怕这一辈子都洗不掉了……
<br/>
<br/>        季雅姗蹲在地上，双手抱臂，十指掐得双臂通红，嘴唇微微颤抖。她一边伤心痛哭，一边自我麻木。
<br/>
<br/>        泪眼模糊的视野里，突然出现了一只拿着白色纸巾的纤瘦白嫩的手。
<br/>
<br/>        季雅姗怔了一下，讷讷地抬起头来，目光也顺着这只好看的手往上看去。
<br/>
<br/>        童筱颖莞尔一笑，温柔地问道：“你是不是遇到什么困难了？需要我帮忙吗？”
<br/>
<br/>        然而，季雅姗却并不领情，反而没好气地呵斥：“我是小三的女儿，你最好离我远点！免得你以后的婚姻，被我染了衰气，也被小三插足了！”
<br/>
<br/>        “？？？”童筱颖听得稀里糊涂的。
<br/>
<br/>        季雅姗看着她这一脸不知所以然的样子，险些忍俊不禁。
<br/>
<br/>        她知道，她是好心，所以，她接过她手里的纸巾，拭去了脸上和眼角的泪痕，淡淡地说了一句：“谢谢。”
<br/>
<br/>        童筱颖释然一笑：“不客气！”
<br/>
<br/>        季雅姗从地上站起身来，看着童筱颖，将她上上下下打量了一番。
<br/>
<br/>        童筱颖也在打量着季雅姗，见她身上还穿着校服，心里估摸着她的年龄也就十七八岁的样子。
<br/>
<br/>        “你是乡里来的？”打量完童筱颖，季雅姗毫不客气地问。
<br/>
<br/>        这女人身上的衣服没啥品味，但她身材好，人瘦，穿什么都好看。
<br/>
<br/>        童筱颖看着季雅姗，赧然地笑了笑：“算是吧！”
<br/>
<br/>        她老家西镇那种地方，确实是穷山僻壤，不过山清水秀。
<br/>
<br/>        季雅姗接着说道：“我今天心情不好，想找个人陪吃饭。今天你走运了！我请你去吃自助餐！”
<br/>
<br/>        这中午刚放学，她就遇见了那个对她不友善的姐姐，说了那么一大堆她不爱听的话，还戳痛了她的心事。
<br/>
<br/>        童筱颖见这个女生并没有什么坏主意，正好昼景今天中午不在家吃饭，她也不打算回家去做饭了，不如跟这个女生搭个伴一起下馆子好了。
<br/>
<br/>        “我们AA制，我就陪你去。”童筱颖微笑着说，她可不想欠别人的人情。
<br/>
<br/>        季雅姗自然也知道童筱颖心里的想法，所以依了她的要求，点头道：“好！AA制，就AA制吧！”
<br/>
<br/>        到了自助餐店里，季雅姗端来了一桌子的食物，童筱颖不禁干笑着问：“你拿这么多，吃得完吗？”
<br/>
<br/>        “你自暴自弃过吗？”季雅姗拿起一个生蚝，一边用勺子挑出里面的肉来塞入嘴中，一边嚼得“吧唧吧唧”地问道。
<br/>
<br/>        童筱颖微微摇了摇头。
<br/>
<br/>        季雅姗抿了抿唇，将嘴里的食物咽下肚后，接着说道：“我妈妈是小三，我一生下来就不受待见。而且，我还是在别人的唾骂中长大的。每次我心情不好的时候，就会来吃自助餐。不管自己手里有钱没钱！”
<br/>
<br/>        所以，这就是一个人自暴自弃的理由吗？
<br/>
<br/>        那她童筱颖岂不是要泪流成河？！
<br/>
<br/>        “这要看你自己用什么心态去过日子，自暴自弃，对身体不好。”童筱颖安慰道。
<br/>
<br/>        季雅姗忽然想起了什么，又问道：“你结婚了没有？”
<br/>
<br/>        “嗯，领了证，但还未办酒。”童筱颖甜甜地笑了起来，不知道为什么，她每次想到傅昼景，就会觉得心里特别甜。
<br/>
<br/>        季雅姗看着童筱颖脸上那甜得像抹了蜜糖一般的笑容，左手托腮，没心没肺地笑了笑：“那你遇上我，就注定不会婚姻幸福了！”
<br/>
<br/>        “为什么？”童筱颖不解地看着季雅姗。
<br/>
<br/>        季雅姗也直言不讳地回答道：“我从小到大，就是个婚姻衰神。我妈妈带我住谁家隔壁，谁家的老公就会出轨。”
<br/>
<br/>        “你又不住我家隔壁。”童筱颖忍俊不禁道。
<br/>
<br/>        季雅姗将勺子衔在嘴里，双手背在了身后，探入到自己的书包里摸索了一会儿。
<br/>
<br/>        当她的手重新放到餐桌上来时，手里多了一副塔罗牌。
<br/>
<br/>        季雅姗随手花样洗牌，看得童筱颖惊赞了一声：“你好厉害！”
<br/>
<br/>        “更厉害的还在后头了！”季雅姗洗好牌后，将塔罗牌朝童筱颖递了过去，“随便抽三张出来！我帮你测婚姻生活。”
<br/>
<br/>        童筱颖顿时好奇心十足，搓了搓手后，从季雅姗手中的那叠塔罗牌中，随便抽了三张出来放到了餐桌上。
<br/>
<br/>        季雅姗一张一张地翻开童筱颖所抽的这三张牌后，不禁啧啧嘴，摇摇头。
<br/>
<br/>        童筱颖柳眉一蹙，担忧地问：“有什么不对吗？”
