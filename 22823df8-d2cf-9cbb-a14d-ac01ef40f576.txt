在她进入顾家的那一刻的起，她就能听到顾伯父和顾伯母永无休止的争吵。
<br/>
<br/>        顾伯母总是在骂她妈咪，说她妈咪是个狐狸精，是个不要脸的贱人，给爹地戴绿帽子，跟顾伯父上床等等…
<br/>
<br/>        后来顾伯母也开始骂她，说她就是一个小野种，一个小贱人，长大了也是勾引男人上床，跟她妈咪是一个货物…
<br/>
<br/>        她见过顾伯父打过顾伯母，那狠狠的一巴掌直接将顾伯母从轮椅扇翻在了地上，她看着那么趾高气扬的顾伯母又笑又哭，简直像个疯子，很可恨，又…很可怜…
<br/>
<br/>        叶翎慢慢的长大，开始明白顾伯母的话，她的妈咪跟顾伯父出轨了，当年那场出轨带走了两条人命，废了顾伯母的双腿，她亲哥哥失踪了，她成了孤儿，所有人都陷入了痛苦。
<br/>
<br/>        怪不得，怪不得那场车祸后，爹地所有的亲人朋友都用异样的目光看着她，没有给予任何的帮助。
<br/>
<br/>        也许，妈咪真的出轨了吧。
<br/>
<br/>        可是叶翎真的不愿意去相信，她记忆里的妈咪那么的美丽知性温婉，不管爹地多晚回家，妈咪都会给爹地留一盏灯。
<br/>
<br/>        爹地亲手教哥哥骑马射箭，她的哥哥是大院里最优秀最俊逸的男孩子，十几岁已经执掌了海城右翼，灼华无双。
<br/>
<br/>        妈咪每天把她打扮的像个小公主，还说我家翎翎是最漂亮最乖巧的女孩子。
<br/>
<br/>        叶翎记得，真的记得，他们一家人生活的很开心。
<br/>
<br/>        可是，一切都变了。
<br/>
<br/>        究竟什么是谎言？
<br/>
<br/>        这时手机又在震动，顾夜瑾还在不停的给她打电话。
<br/>
<br/>        叶翎觉得这手机的防水功能还真是强大，竟然能抵住这一天的倾盆大雨。
<br/>
<br/>        纤白的手指按了接通键，她将手机贴在耳畔。
<br/>
<br/>        很快，顾夜瑾低醇好听的嗓音就传递了过来，“翎翎，你现在在哪里，外面下雨了。”
<br/>
<br/>        叶翎伸手往自己的小脸上摸了摸，一脸水，不知道是雨水，还是自己的眼泪。
<br/>
<br/>        叶翎勾唇，“我已经回醉玉欢了。”
<br/>
<br/>        “我不放心，我现在过去找你。”
<br/>
<br/>        叶翎，“不要过来，顾夜瑾，不要过来，求你了。”
<br/>
<br/>        那端的顾夜瑾沉默了。
<br/>
<br/>        叶翎直接挂断了电话。
<br/>
<br/>        刚才他跟顾夫人在楼下客厅里的谈话她都听到了，顾夜瑾绝对是她见过最阴暗狠戾，卑鄙无耻的男人，在她什么都不懂的年纪里，他在一步步的引诱她爱上他。
<br/>
<br/>        他是这世上最好的猎手，她就是他的猎物。
<br/>
<br/>        成人礼那一天她不知道喝错了什么，浑身发热，她进了他的房间里，摸上了他的床。
<br/>
<br/>        这个伪君子，如果当时他有丁点的不愿意，她还能真的强了他？
<br/>
<br/>        那晚他趴在她的身上，叫她夏妍妍。
<br/>
<br/>        房门推开，大家都进来时，他又给了她一巴掌。
<br/>
<br/>        所有的宠溺呵护在那么一刻都撕开了伪装，露出了虚伪而狰狞的面目，上一代的仇恨转嫁到了她的身上，他用最残忍无情的方式毁了她。
<br/>
<br/>        毁了她的身，毁了她的心。
