夏夕绾对这个结果是相当不满意的，她并不想卖给徐少南！
<br/>
<br/>        夏夕绾看向陆寒霆，他竟然喊了一个十二亿就不喊了，这一点都不符合他霸总的作风，他就应该是那种来啊，斗啊，不就是钱吗的那一种人，他不差钱的。
<br/>
<br/>        但是夏夕绾只看到了陆寒霆英挺而薄冷的背影，因为他对接下来的事情毫无兴趣，已经起身离开了。
<br/>
<br/>        ……
<br/>
<br/>        夏夕绾被送到了酒庄的豪华套房里，她等了一会儿，房间门被推开，徐少南走了进来。
<br/>
<br/>        徐少南眯起那双桃花眼上下打量了她一番，唇角的弧线邪魅慵懒，“丑丫头，原来你长这样啊。”
<br/>
<br/>        既然被认出来了，夏夕绾也就大方的承认了，她颤着俏皮的羽捷笑道，“徐少，这一次多谢你出手相助，改日我请你吃饭，我还有事，先走了。”
<br/>
<br/>        夏夕绾大摇大摆的准备离开。
<br/>
<br/>        但是徐少南挡住了她的去路，“丑丫头，跟我就别玩这一套了，我花了十三个亿买你回来可不是听你说一声谢谢的，你的谢谢还没有那么好听。”
<br/>
<br/>        夏夕绾不明白的望着他，“徐少，那你是什么意思？”
<br/>
<br/>        徐少南看着她装傻充愣的模样，突然上前了两步，欺近她，“上床，不懂啊，难道你跟你老公没上过床？”
<br/>
<br/>        夏夕绾往后退，膝盖磕到了床沿，一下子跌坐在了柔软的大床上，在海城的时候，她跟厉老夫人说过她已经结婚了，不过他们并不知道她老公是谁，后来陆司爵又抹去了一切痕迹。
<br/>
<br/>        夏夕绾澄亮的翦瞳看着徐少南欺近过来的俊脸，迅速抬手推他，“徐少，既然你知道我有老公，那你对有夫之妇应该没兴趣吧？”
<br/>
<br/>        “我有点好奇，你那个老公是谁，他没钱用了，把你送到这个私人酒会来拍卖？”
<br/>
<br/>        不是的！
<br/>
<br/>        她的陆先生以前有钱，现在更是超级有钱好吧！
<br/>
<br/>        不过夏夕绾是不会将她跟陆寒霆的短暂婚姻告诉他的，“徐少，今天我是来救我朋友的，所以落难了，我真的很感谢你能出手相救，但是我很爱我老公，没有婚内出轨的打算。”
<br/>
<br/>        徐少南单膝跪在床上，将纤柔的她给困在自己的胸膛里，“那你先离婚，离婚了再陪我睡。”
<br/>
<br/>        “…”这位徐少是个神经病吧。
<br/>
<br/>        夏夕绾很想脱身，不想沾染上徐少南这样难缠的大人物，她索性往床上一躺，闭上了双眼，“徐少，实话告诉你吧，我已经离婚了，那是我前夫了，所以我才一个人来帝都城的，我不想欠你的，你现在就来吧！”
<br/>
<br/>        “丑丫头，别跟我玩激将法，没有用。”徐少南伸手，手指抚上了她巴掌大的绝色小脸，“实话告诉你，从海城回来，我就没有忘记过你，既然你离婚了，以后就跟着我吧。”
<br/>
<br/>        他伸手去撕夏夕绾身上的薄纱长裙…
<br/>
<br/>        ……
