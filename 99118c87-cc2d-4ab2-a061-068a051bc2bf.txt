　　只见上面写道：
<br/>　　“想要令姐活命，三日之内，来我药王宗，过期不候！”
<br/>　　看到叶尘满脸的杀意，周围神龙卫众人，也是个个吓得不行，同时又羞愧不已，
<br/>　　“噗通！”“噗通！”“噗通！”
<br/>　　众人对视了一眼，同时跪倒在地上，
<br/>　　“叶帅！我等办事不力，没有保护好苏蔓小姐，请您惩罚！”
<br/>　　叶尘默然不语，深吸了一口气，方才缓缓道：
<br/>　　“此事错不在你们！掳走我蔓姐之人，应该精通幻术，以你们的实力，没有发现也很正常！”
<br/>　　说完这话，叶尘随手一挥，众人便齐刷刷的站了起来。
<br/>　　“此事我自会处理，你们都去歇着吧！”
<br/>　　声音落下的同时，叶尘身体一晃，已经消失不见。
<br/>　　……
<br/>　　几分钟后，
<br/>　　叶尘来到天海大学外的别墅，
<br/>　　萧若曦别人袭击，晕倒过去之后，叶尘命人将她送到了这里。
<br/>　　经过一番检查，让叶尘感到欣慰的是，曦月只是强行动用神魂秘法，连同萧若曦的神魂也受到了影响，所以沉睡了过去。
<br/>　　等叶尘将自己的神魂之力，灌入对方的体内，曦月的魂魄便慢慢苏醒了过来。
<br/>　　“曦月，你现在感觉如何？”
<br/>　　‘萧若曦’笑着摇了摇头，
<br/>　　“不过就是动用了一次神明魂术，没想到竟然耗费我将近一个月的努力，看来重塑元神的时间，又得往后拖一拖了！”
<br/>　　叶尘也不由得轻轻一叹，
<br/>　　“你没事就好！重塑元神的事情，我来想办法！”
<br/>　　说着，叶尘直接抱起‘萧若曦’，破空而起，一转眼便已经来到数百米的高空，趁着夜色，直奔云州而去。
<br/>　　曦月何等聪明，瞬间意识到应该是出事了，
<br/>　　“叶尘，怎么了？是家里出事了吗？”
<br/>　　叶尘一边飞速朝云州的方向疾驰，一边将事情缓缓讲了出来。
<br/>　　曦月听完之后，顿时大为担忧，
<br/>　　“那个什么药王宗，很厉害吗？你一个人去，有几分把握？”
<br/>　　叶尘淡淡一笑，
<br/>　　“放心！根据我的推测，地球上应该不存在化神境的修真者，我不敢说纵横世间无敌，保命肯定是绰绰有余！”
<br/>　　曦月这才松了一口长气，
<br/>　　他对叶尘的战斗力还是很有信心的，如果真如叶尘所说，地球上没有化神境修士，的确对他构不成太大威胁。
<br/>　　……
<br/>　　二十来分钟后，
<br/>　　叶尘已经带着‘萧若曦’，回到了云州黑森林。
<br/>　　楚飞烟等人早已经得到消息，见到叶尘带着萧若曦回来，虽然心中有些好奇，不过也没有多问，
<br/>　　“叶尘，岚姨那边，我暂时还没有将情况告诉他，你接下来打算怎么办？”
<br/>　　叶尘深吸了一口气，眼眸中闪过一抹寒色，
<br/>　　“药王宗既然敢招惹到我的头上，那就要做好付出代价的准备！”
<br/>　　“我要即刻动身，前往蓬莱仙界！”
<br/>　　楚飞烟顿时微微一惊，
<br/>　　“就你一个人吗？要不要带几名门下弟子，随你一同前往？”
<br/>　　叶尘果断摇了摇头，
<br/>　　“没有这个必要！我自己一个人，反而会少一些顾忌！”
<br/>　　“你们要做的，就是守好家门，说不定还会有其他势力，来找我们的麻烦！”
<br/>　　如果公孙林没有撒谎，那叶尘和他狂仙门，如今只怕已经成为整个仙界的眼中钉，甚至神、魔、妖三界，也会对他下手！
<br/>　　不过，如今的狂仙门，在叶尘和楚飞烟的经营之下，已经固若金汤，便是元婴境修士前来，也绝对讨不了好！
<br/>　　再加上，狂仙门背后还有国家，附近驻扎着军队，他们应该也不敢大张旗鼓的动手。
<br/>　　又对楚飞烟嘱咐了一番之后，叶尘先将萧若曦安顿了下来，然后便又直接朝东面疾驰而去。
<br/>　　半个多小时后，此时天色已经微微发白，而叶尘也已经跨越东江省，来到了东海海面之上。
<br/>　　只见他右手一翻，手中已经出现了三张羊皮残卷，赫然正是当初，他分别从岛国阴阳师南野尘八、岛国第一强者羽鹤大仙，以及狼王科洛夫三人手中，得到了的那三份地图残卷。
<br/>　　“原本以为，这辈子都用不上了，没想到最终还是派上了用场！”
<br/>　　叶尘微微感慨了一句，飞快的将三份残卷凑到了一处，顿时便组成了一份完整的地图。
<br/>　　“原来在这儿！怎么会？”
<br/>　　叶尘通过完整的地图，很快便找到了蓬莱仙界的入口所在，可是却发现，竟然离华夏东海岸只有数百里的距离！
<br/>　　这一发现，着实让叶尘有些吃惊，怎么也没有想到，这蓬莱仙岛居然就在华夏国的眼皮子底下。
<br/>　　“不管了！先按照地图所指，找到这个地方再说！”
<br/>　　叶尘身体一晃，顿时在海面上留下一道残影，整个人的速度，瞬间飙升到十倍音速，只用短短几分钟的时间，便已经抵达目的地。
<br/>　　可奇怪的是，周围只有一座孤零零的小岛，并没有其他任何入口。
<br/>　　叶尘站在小岛上，将神念完全释放开来，也没有发现任何奇特之处。
<br/>　　“难道这地图是假的？又或者是我找错到了地方？”
<br/>　　叶尘百思不得其解，只好又将地图翻来覆去，仔细查看了一遍。
<br/>　　忽然，
<br/>　　叶尘看到地图一角，那四行篆体的小字，
<br/>　　“阴阳交合，南北交替，日出之时，仙门大开！”
<br/>　　将这四句话，反复读了几遍之后，叶尘隐隐感觉自己，似乎领悟到了一丝其中包含的真谛，可一时半会却又有些模糊。
<br/>　　就在这个时候，恰好东方那一轮红日，缓缓从海平面上升起。
<br/>　　叶尘猛然望着那轮红日，脑袋嗡的一下，瞬间便一下子顿悟了。
<br/>　　“我明白了！原来如此！”
<br/>　　踏踏踏！
<br/>　　叶尘身体一晃，直奔日出的方向而去，奔出百余里之后，果然感应到周围，有一道古老而雄厚的封印气息。
<br/>　　“果然在这儿！看来当年绘制这份地图之人，还真是用心良苦啊！”
<br/>　　原来那四句话，才是找到蓬莱仙岛的关键，
<br/>　　在特定的地点，特定的时间，以日出为参照物，方能找到蓬莱仙岛的入口。
<br/>　　叶尘也是运气好，刚巧赶对了时间，所以才能把这个如此隐蔽的入口，给找了出来！
<br/>　　叶尘将神念完全释放开来之后，很快便找到了封印的破绽，然后双手虚抓，猛然一撕，前方登时便出了一道裂缝！
<br/>　　“药王宗！我来了！”
<br/>　　叶尘低声喃喃了一句之后，直接迈步走了进去……
