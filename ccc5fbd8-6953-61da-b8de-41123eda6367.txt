　　第166章
<br/>　　“军功，军功可没有我的脑袋重要。”方伯山道：“程老弟，咱俩关系不错，你就说没有看见我，好吧，就算我谢谢你了。”
<br/>　　程大雷顿了顿：“方大人，青牛山守不住，落叶城必破。一旦城坡，必然生灵涂炭，就算你这个城主的脑袋也保不住。现在我蛤蟆寨有地利有粮，你落叶城有人，为何你不能与我联手一战？”
<br/>　　“联手一战，那可是戎族！”方伯山咧咧嘴：“程老弟，我要逃命去了，我在青州有些关系，到了地方就安全了。你想死，别拉上我。这样我的城主印也在这里，送给你了，城中兵马都随你调动，你就对幽州王说没看见我好不好？”
<br/>　　方伯山将城主印强塞到程大雷手中。
<br/>　　握着掌心沉重的帅印，程大雷沉默良久。
<br/>　　他忽然注意到城主府有无数兵丁，忽然道：“可有人愿与我合力一战，驱除蛮敌，守卫河山？”
<br/>　　无人回应，程大雷孤零零举着城主印，显得既孤独又尴尬。
<br/>　　“大当家，咱们走吧。”徐神机扯扯程大雷的衣角，觉得有些心酸。
<br/>　　蛤蟆寨，程大雷的屋子。
<br/>　　天色昏暗下来，程大雷没有点灯，他一个坐在桌子前，桌上放着三枚大印。
<br/>　　幽州王的王印、方伯山的城主印，自己的将军印。
<br/>　　王印上蹲着麒麟，城主印上是一只黄虎，将军印上是一只蛤蟆。此刻三家印都在程大雷面前，都属于程大雷一个人的。
<br/>　　他坐在昏暗中，很久都一动不动。
<br/>　　有一个问题程大雷怎么想也想不明白：蛤蟆寨易守难攻，只要杨龙停或方伯山给自己一些兵，不是守不住。戎族虽强，可也并不是三头六臂，骑兵再横，到了山地他们的本事至少被减去一半。
<br/>　　为何就不能守守试试呢？为何一听到戎族的名头，就望风而逃，连握刀的勇气都没有了。
<br/>　　包括在蛤蟆寨，除了一个林少羽，没人支持自己。但林少羽是要做大侠的人，死对他来说不算什么，他心里其实没有任何胜算。
<br/>　　可程大雷心里是觉得能打赢的。
<br/>　　真孤独啊！
<br/>　　程大雷想起曾经与李行哉的谈话，那时双方就戎族为何百战百胜讨论过，程大雷作为一个资深键盘侠，把李行哉唬得一愣一愣的。
<br/>　　今日程大雷想起二人的谈话，大约明白他们为何要逃了。
<br/>　　三十年来，戎族的名声铺天盖地。只要帝国碰上戎族，不管占什么优势，结果都是失败。慢慢，败成了习惯，逃跑也成了习惯。所有人心底都觉得戎族是不可战胜的。
<br/>　　而他们之所以要逃，是因为他们有退路。像方伯山、杨龙停跑到一个远远戎族碰不到的地方，并不是什么难事。所以能活着，为何要去拼命，而他们都跑了，手下兵丁又凭什么去送死。
<br/>　　如此，大家都跑吧，跑个兵败如山倒，跑得几十年来，帝国在戎族面前直不起腰。
<br/>　　苏樱此时走进屋子，默默看了程大雷一眼。
<br/>　　“外面的人都在等你呢？”
<br/>　　“嗯，让他们再等一会儿。”
<br/>　　苏樱走到程大雷身边，将手搁在她肩膀上，慢慢道：“我知道你置下这份家业不容易，但留得青山在，不怕没柴烧。冀州青州都有苏家的生意，我们先到那里躲一阵子，戎族不长久的，等过了这件事，什么都可以回来的。”
<br/>　　程大雷起身，拿开苏樱的手，笑笑道：“走吧，我准备好了。”
<br/>　　夜幕，蛤蟆寨灯火通明，山寨弟兄在校场聚齐，点着火把驱散黑夜。
<br/>　　见程大雷出来，所有人的目光都追随着程大雷，直到他登上点将台。
<br/>　　“诸位兄弟……”校场鸦雀无声，只有程大雷的声音回荡：“你们的妻儿老小都在此地，这山寨也是大家每人出一份力建起来的，我只问大家，愿不愿舍了咱们的山寨，去做那丧家犬。今日我们有地利之便，山寨兵器粮草无数，是否有兄弟愿意随我一战，守住咱们山寨？”
<br/>　　无人回应，火把照亮一双双眼睛，所有人的眼睛又都看着程大雷。
<br/>　　“愿意走的，我不留，愿意战的，上前走一步。”
<br/>　　程大雷声音出口后，没人回答，只有林少羽干脆利落的向前一步。
<br/>　　“还有人么？”程大雷问。
<br/>　　略沉默一会儿，徐神机咬咬牙站了出来：“罢了，罢了，我一把老骨头也活得够久了，今日陪大当家一起去送死。”
<br/>　　秦蛮也站出来，苦笑一声道：“不过一死而已。”
<br/>　　关张二人看着刘悲，他苦着一张脸，往前跨了一步，却没有说话。
<br/>　　关张立刻往前跨了一步，赵子龙也随之站出来。
<br/>　　小白狼也往前走了一步。
<br/>　　“好啦，我也随你送死。”高飞豹突然站出来，没有理睬大哥制止的眼神。
<br/>　　苏樱看了看左右，也往前站了一步。
<br/>　　这些人肯走出来，并不是觉得能守住，不过是看程大雷一人站在那里，有些心酸。
<br/>　　程大雷看着这些人，人心不齐，凭这几个人，守不住蛤蟆寨。他忽然苦笑一声：“好啦，好啦，我没有要弟兄随我一起送死的权力。既然如此，那大家就商量商量怎么……逃吧。”
<br/>　　……
<br/>　　黑石城，城主府。
<br/>　　绿樱桃看着面前握刀的福德勒，吓得花容失色。
<br/>　　“你不要杀我，你让我做什么都可以，求求你别杀我。”
<br/>　　福德勒看看手中的刀，再看看跪在自己面前的绿樱桃，忽然一咬牙，把刀丢在地上。
<br/>　　“我放你走。”
<br/>　　“你肯放我……”绿樱桃难以置信。
<br/>　　福德勒点点头：“你走吧，从后门走，你向南走，那是你们的国家，去了那里就没有人能伤害你了，在这里不安全。”
<br/>　　“走……”绿樱桃摇摇头：“这里到处是你们的人，我一个弱女子，又能逃到那里去。”
<br/>　　“那怎么办？”福德勒挠挠头。
<br/>　　绿樱桃也是心慌意乱，她看了看面前的福德勒，觉得对方也不过是个少年而已。
<br/>　　“你会保护我的，对么？”绿樱桃忽然道出这么一句。
<br/>　　福德勒突然紧张起来，有些手慌脚乱：“我尽力……尽力不让谁伤害你啊。”
