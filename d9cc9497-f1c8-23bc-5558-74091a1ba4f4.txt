　　虚空中，画面不断呈现在蜀山五老的眼前，青年持剑满面狰狞，语气森林，暴露出自己丑陋不堪的面孔。
<br/>　　随着画面转变，蜀山五老脸色难看到了极点，特别是和阳长老，目光阴沉，眼底杀意四射。
<br/>　　当青年口中一句‘老不死的东西’脱口而出，和阳长老双眼一闭，隔空一掌，青年尸体瞬间化作一团血雾。
<br/>　　“孽畜……”
<br/>　　话语刚落，和阳长老口中喷出一口鲜血，脸色苍白，显然是气血攻心所致。
<br/>　　青年是他百年来座下唯一的弟子，在他眼中品德兼备，资质卓越，却不曾想到，知人知面不知心。
<br/>　　清微拂尘一挥，地上一枚纳戒就出现在他的手中，查看了下纳戒中的东西，叹息一声。
<br/>　　“都是一些普通的仙法，但此事必须严查，没想到蜀山百年来韬光养晦，却依旧有跳梁小丑暗中算计，倒是我们疏忽了。”
<br/>　　和阳长老压下心中复杂的情绪，苍白的脸上露出一抹牵强的笑意，旋即笑意就化作无尽的冷冽。
<br/>　　“师兄，请将此事交给老夫处理，蜀山不容叛徒存在！”
<br/>　　摇了摇头，清微平淡道：“此时就交由幽玄处理，你虽只是气血攻心，但需要修养，重则影响道心呐。”
<br/>　　幽玄眼底也闪过一抹冷意，虽他们已经不再出山，也没有当年的锐利，但他们依旧是地仙，可不是某些小丑能算计的。
<br/>　　视线瞥了眼真武殿内，清微有些无奈，道：“好好休息，今日之事隔日再谈。”
<br/>　　待清微与其他长老离去，和阳长老站在殿前，望着天权峰周围的景象，发丝更白了一分，最终悠悠一叹。
<br/>　　……
<br/>　　……
<br/>　　对于殿外的事情，慕清霄根本就没放在心上。
<br/>　　此事对于偌大的蜀山来说，不过只是个小插曲，就算青年携着仙法判处蜀山，也不可能令蜀山根基颤一下。
<br/>　　蜀山是庞然大物，某些势力只是小丑，否则也不止是在暗中策划。
<br/>　　门派争斗，慕清霄不感兴趣，但此事却让他对地仙境的强大，再度有了一个深刻的了解。
<br/>　　将某些情绪抛之脑后，慕清霄视线就落在真武殿内。
<br/>　　真武殿乃是蜀山禁地，先前的叛徒青年虽是和阳长老的座下弟子，但也没有出入真武殿的资格。
<br/>　　哪怕想要更高等的修仙法门，也只能求教长辈。
<br/>　　整个内殿有足球场大小，其中紧致有序的排满了书架，约有五十座书架左右，书架上则摆满了竹简。
<br/>　　除了书架，其中就是空荡荡的一片。
<br/>　　这时，和阳长老从门口走了进来，看向慕清霄的目光有些复杂。
<br/>　　半响过后，和阳长老顿时觉得有些尴尬，于是出声道：“这里放的仙法虽有高级，但不是顶级。”
<br/>　　闻言，慕清霄有些好笑，这老头子真是死要面子。
<br/>　　“晚辈虽胜任长老，但年龄与阅历都不如前辈，先前失礼，请长老勿放在心上。”
<br/>　　“无妨，你也知道，这里是蜀山禁地，我们又怎么可能将最重要的东西毫无设防的摆在这里，跟老夫来吧。”
<br/>　　和阳长老似乎也没在意刚才的事情，慕清霄虽杀了他的弟子，但也在是帮蜀山清理门户。
<br/>　　他不是老糊涂，当然是非分明。
<br/>　　若有所思的点了点头，慕清霄跟在和阳长老身后，慢悠悠的走在书架间。
<br/>　　不多时，来到真武殿的中央，和阳长老停在了一座书架前，伸出手掌轻轻在书架的竹简上一按。
<br/>　　“轰隆……”
<br/>　　陡然，整个真武殿在一瞬间颤抖起来，旋即殿内徐徐浮现出一抹光点，五十座书架开始移动。
<br/>　　慕清霄看了一眼四周，目光看向脚下，随后眼中就闪过一抹震惊之色。
<br/>　　“不是书架，是地面在移动，居然将阵法运用在真武殿上。”
<br/>　　“并非运用在真武殿上，而是整个真武殿就是阵法本身，当初祖师建造真武殿的时候，便在其中融入了阵法。”
<br/>　　和阳长老语气中满是尊敬与自豪，想当初他们祖师太清真人镇压了整整一个时代，蜀山更是白盟之首。
<br/>　　虽然那个时代依旧过去很久很久，但依旧值得他们吹嘘，蜀山虽不如以前，但也不算弱。
<br/>　　待轰鸣声消散，书架的位置有了明显的变换，但依旧排序整齐。
<br/>　　此时，真武殿中央，也就是两人所站位置的前方，居然出现了一口通道，通道中光线敞亮，将阶梯清晰的照映了出来。
<br/>　　通道的尽头有什么不言明了，慕清霄道：“和阳长老有伤在身，就不劳烦长老了。”
<br/>　　“也罢，尽快出来，开关在入口的左侧。”
<br/>　　和阳长老也不怕慕清霄对蜀山不轨，说罢就不再多留，转身径直离去。
<br/>　　慕清霄苦笑一声，还真是放心自己，也不能说是放心，而是不怕自己带着功法跑了。
<br/>　　若是先前，他还有些自信，但现在慕清霄可以确信，自己绝对不可能在地仙面前逃走，仅仅威压都能将空间禁锢。
<br/>　　况且，除了和阳长老，蜀山还有幽玄等长老，更有一个深不可测的清微老头子。
<br/>　　待和阳长老离去，慕清霄直接走进通道中，下一秒通道口就合并起来，墙壁上不留一丝的缝隙。
<br/>　　蜀山宫殿的墙壁不知是用何种材料打造，坚硬无比，星陨也不过如此吧？
<br/>　　当然，若是他用魂剑使出全力，未必就不能斩出一条道路。
<br/>　　通道中光线敞亮，两侧墙壁上都镶嵌着无比巨大的夜明珠，任何一颗拿去世俗中都是价值连城宝物。
<br/>　　通道并不长，仅仅走了数步就抵达尽头。
<br/>　　尽头是一间石屋，屋子空间不大，其中摆着四座书架，空气中有一股淡淡的檀木香。
<br/>　　来到书架旁，慕清霄拿起其中一卷竹简，将竹简摊开，脑海中就响起了系统的提示声。
<br/>　　“叮，《万剑诀》收录中，高阶剑诀，价值50000掠夺积分。”
<br/>　　微微一愣，慕清霄嘴角牵起一抹弧度，不愧是根基所在，随便一卷竹简，其中记载的都是高阶剑诀。
<br/>　　剑诀被系统收录，慕清霄就将竹简放回书架，继续拿起第二卷。
<br/>　　……
