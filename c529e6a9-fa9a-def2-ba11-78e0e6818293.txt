<br/>　　这样力量的轰击，任何一位九劫古帝承受住都要当场灭亡。
<br/>　　一道长虹轰击在了无上剑阵的中心，一千八百道无上剑气化为的无上剑阵，在这碰撞的瞬间便是出现了无数裂痕。
<br/>　　数百道无上剑气当即是被撞散，飞了出去。
<br/>　　无上剑阵，岌岌可危。
<br/>　　轰！
<br/>　　就在这时，又是一股雄厚的天地之力降临了。
<br/>　　在这关键时刻，从这光幕之中又是一道身影出现了，又是一尊九劫古帝降临了。

<br/>　　“燕皇？是燕皇！”

<br/>　　看清楚这来人，五行刀祖等人齐声说道。
<br/>　　他们三人联手虽然是在跟楚风眠交手之中占据下风，可若是在多一名九劫古帝，那结果可能就不一样了。
<br/>　　毕竟为了对付三人，楚风眠已经是底牌尽出了，在多上一名九劫古帝，他们就有五成把握，足以杀了楚风眠。

<br/>　　为了杀了楚风眠，他们宁愿是交出一些好处来，也要来拉拢燕皇。

<br/>　　“燕皇，你还不出手？杀了此人，我们可以将好处大头都给你！”
<br/>　　五行刀祖目光闪烁，看着燕皇没有出手的迹象，急忙的喊道。

<br/>　　在听到这话之后，燕皇的手心之中终于是在凝聚起来了灵力。

<br/>　　这样的伤势，纵然是五行刀祖在恢复过来，他的一身实力也会下降数成。

<br/>　　看到五行刀祖被打飞出去，碧螺老祖，葵海王死死的盯着楚风眠，恶狠狠的说道。
