　　“什么，那是什么凶妖，身上竟有此等暴戾凶气？！”
<br/>　　“竟然是一头血色的蚊妖，这身上的气息……是幽冥血海之中特有的气息，难道与冥河有关？”
<br/>　　“冥河教主上次证道之时不是被圣人打成重伤，利用血神子逃走了吗，怎么有出现了？难道是伤好了？”
<br/>　　“……？”
<br/>　　感受到那股凶悍的气息，诸神纷纷猜测这妖蚊的来历。阿弥陀佛与准提佛母脸色一沉，谁都看得出来他们的心情很差。
<br/>　　在众人疑惑的神情之中，那凶蚊直接飞入气运海中的金莲之上，唧的一声兴奋尖叫，它嘴角陡然长出又细又长的口器，带着一股无比锋利的锐气，瞬间便刺破十二品功德金莲的防御，大口吞吃起来。
<br/>　　“咕噜……咕噜……”
<br/>　　天地一片寂静，唯有那凶蚊吞噬金莲的声音不断响起，显然，众神俱皆傻眼了，谁也不曾想到会出现如此惊人一幕。
<br/>　　“嘶！”
<br/>　　“我不是眼花了吧，那妖蚊竟然……竟然真的将金莲吞噬了。”
<br/>　　“天啊，这个世界疯了吗？金莲防御无双，更是极品先天灵宝，就算是圣人也难以一击击碎，如今竟被一只凶蚊给吃了，这怎么可能？”
<br/>　　“这妖蚊到底是何来历，竟如此厉害，连金莲的防御都被它破了？”
<br/>　　“……”
<br/>　　天地间无数关注此地的大能都被惊动了，这么大跌眼镜的事情都出现了，这妖兽实在是太凶悍了，竟然连佛门的镇运至宝都敢碰，而且还有那份能力吞吃金莲。
<br/>　　阿弥陀佛与准提佛母脸色剧变，但却无可奈何，佛教初立，此刻正是汇聚起运之际，在此过程中，自己等人不可以外力干涉气运池，否则会影响气运。但此刻那妖蚊却正在大肆吞吃镇运金莲，若不阻止，便会更加倒霉。
<br/>　　一个两难的选择摆在他们面前，是灭杀妖蚊，还是等气运收集完毕再出手？两者对他们都十分不利，只能选择利益损失的少的。
<br/>　　“师兄，那厮是何处妖孽，竟敢吞噬我佛门气运金莲，如今该如何是好？”准提佛母焦急道。
<br/>　　“此乃洪荒第一只蚊子，为异种血翅黑蚊，出生于血海，先天带有部分血之法则与吞噬法则，横行无忌，极为凶悍。”阿弥陀佛深吸一口气，沉声道，
<br/>　　“如今只能等，等到西方气运汇聚完毕后才可动手擒拿它。”
<br/>　　“可是，师兄……”
<br/>　　“聚集气运只有一次机会，关乎到佛教的未来，金莲没了，以后还有其他，但佛教未来不可动。”阿弥陀佛摇摇头道。
<br/>　　“嗯，可那有需要多久？”
<br/>　　“很快即可。”阿弥陀佛眼睛盯着气运海中的血蚊，缓缓道。
<br/>　　东方诸神见到这一幕心中却是无比畅快，西方气运损失对于东方只有好处，没有坏处。
<br/>　　一些不知道情况修士心便却疑惑不解了。那血蚊吞吃金莲，佛教之人竟然不阻止，这是怎么回事？难道有什么我们不知道的秘辛？
<br/>　　“快看啊，降了，那金莲竟然降了一品！”随着血蚊的不断吞噬，十二品功德金莲竟然降到十一品，先天本源大量流失。
<br/>　　“又降了，十品了！”看着那金莲品级下降，为何心中会有些莫名的兴奋呢，难道看着别人倒霉，自己就会高兴，这就是把自己的快乐建立在别人的痛苦上的感觉？
<br/>　　“看，九品了！”
<br/>　　“……”
<br/>　　这时，西方气运终于不再往灵山汇聚了。第一时间，准提佛母出手，圣人出手，自然非同凡响，蚊道人就算再厉害也不过是大罗金仙修为，准提佛母不费多少力量便将那血翅黑蚊捉住，封住其修为，将之囚禁起来。
<br/>　　然而，此刻，佛教的镇运至宝仅仅剩下了九品，先天本源流逝的厉害，日后若无大量先天之物去弥补，这金莲便算是一生如此了。
<br/>　　昆仑山，元始收回目光，佛教之事算是暂告一段落，他将目光投向人间界，人间此刻正是诸侯纷乱，春秋五霸的时代，百家争鸣。
<br/>　　他看到慈航道人自封修为，历经人间疾苦，到处行慈运悲，普渡苍生，以此历练道心，磨砺心性。为了适应各种不同根性及类别的众生，慈航真人更是化出三十二种不同身份的应身，去经历各种类别人的故事。其中就有独觉身、缘觉身、梵王身、帝释身、自在天身、比丘尼身、女主身、童男身、童女身、非人身等。
<br/>　　他看到文殊广法天尊与普贤真人游戏人间，经历人情世故，人间冷暖，最后在五台山讲经论法。
<br/>　　他看到燃灯道人正在灵鹫山参悟大道，头顶琉璃灯光闪烁，道韵流转。
<br/>　　元始神秘一笑，原本阐教的四个叛徒，此刻竟然都在以各自的方式感悟着人间大道，也没有如原本轨迹那般投入佛教。
<br/>　　元始并不奇怪，若他们真的投入了佛教，元始那时才要怀疑自己的本事呢。
<br/>　　元始抬手间便在虚空刻下数道道痕，一股玄奥空间波动产生，随后，便看到远在人间的慈航、文殊、普贤与灵鹫山上的燃灯，瞬间到了元始面前。
<br/>　　“老师。”
<br/>　　几人先前还在一个熟悉的地方，但突然就来到一个不知距离多远的昆仑山，心中先是警惕看了对方一眼，待发现元始天尊的身形后，顿时放下心中警惕，对着元始一拜。
<br/>　　“嗯，本座观你四人颇具佛性，尔等是不是想修佛道？”元始点了点头，之后淡淡道，声音之中，听不出任何情绪。
<br/>　　刷！
<br/>　　四人闻言，脸色立刻大变，心中惶恐不安，连忙跪倒在地，大声道：“师尊，我等无意佛道，请师尊明鉴！”
<br/>　　“嗯。”元始淡淡应了一声，之后便不再说话，只是目光还在盯着他们。
<br/>　　五人之间的气氛顿时安静了下来，地上四人更加惶恐了，大气不敢出一口。
<br/>　　良久，元始才开口道：“起来吧，本座并没有说尔等不对。万法皆道，仙道是道，武道是道，魔道亦是道，佛道更是逃不过一个道字，本座并没有看不起佛道的意思。只是观尔等所为，佛道似乎更适合尔等。”
<br/>　　“是，师尊。”四人闻言，皆松了一口气，站起身来，只是据元始口中所言，他们亦猜不出元始心中所想。
<br/>　　“本座为你们讲道三天，三日之后，本座另有要事吩咐，尔等需尽心完成。”元始道。
<br/>　　四人相互对视一眼，道：“谨遵师尊法旨！”
<br/>　　元始不再多言，开始讲道，他此刻所讲之道并非阐述天地至理，而是讲刚刚成立的佛道。佛道他也不讲佛家修行禅功，不讲佛家神通法术，而是讲佛道经典。
<br/>　　后世之中的《金刚经》、《法华经》、《心经》、《般若经》、《楞伽经》、《无量寿经》、《阿弥陀佛经》等各种佛教经典，再加入他自己的理解通过讲道的方法讲了出来，就算是如今的大乘佛教，对佛道的理解，也没有如此厉害。
<br/>　　因为佛教初立，而元始所讲的却是未来无数年中后人所悟经典，这一对比，差距就出来了，其中有数个时代的差距。
<br/>　　慈航真人等先是大惊，没想到自己师尊对佛道的领悟如此厉害，甚至比佛教还要厉害，但随后，他们便将整个心思沉浸于无尽佛理之中，开始明悟自己的佛心，走出自己的道。
<br/>　　元始讲道，没有天花乱坠，没有紫气纵横，更没有天地妙音，有的只是平平淡淡的真实之道。元始早已达到了一言一语即是道，一举一动皆为法的境界，不需那些衬托，却也能做得更好。
<br/>　　就是这样，让四人深深沉浸其中，不可自拔。
<br/>　　昆仑山上，微风拂过，卷起几片落叶，在空中打了几个圈又再次落下，神山之中一片宁静祥和，唯有淡淡佛音在唱响。
