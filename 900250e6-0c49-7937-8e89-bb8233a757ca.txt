　　T国有一种叫做“租妻”的服务。
<br/>　　这种“租妻”服务主要面向西方国家的游客，提供服务的都是当地的年轻女子，她们其中一部分是单身，也有一部分是已婚的，她们向那些西方国家的游客提供“租妻”服务。在服务期间，她们向那些游客旅行作为妻子的义务，并且充当景点的导游，而游客只需要支付在游玩过程中的费用。
<br/>　　如果在“租妻”过程中，双方产生了感情，能够嫁到国外，进入发达国家生活，这是那些提供“租妻”服务女子所希望有的结果；可是真的能如愿跟着游客离开T国前往西方国家的却是少之又少，很多来“租妻”的游客在自己国家都是有家有室的。
<br/>　　在那些游客离开后，一些提供“租妻”服务的女子发现自己有了身孕，这个时候她们有些人会选择去把孩子流掉，然后去寻找新的“老公”；有的则是会选择把孩子生下来，这也是T国混血儿较其他东南亚国家要多的原因所在。
<br/>　　阿卡盼在成为普提查手下，成为毒贩之前也曾结过婚，有过自己的家庭，但他的妻子并不满足他们所拥有的生活，去向来自西方国家的游客提供“租妻”服务，试图以这种方式离开T国。在“临时老公”离开之后，阿卡盼妻子发现自己怀孕了，在去小诊所做流产手术的时候，因为大出血，阿卡盼的妻子死在了手术台上。
<br/>　　也正是因为这个原因，阿卡盼对那些来自西方国家的白种人有种源自内心的仇视。
<br/>　　阿卡盼用消音手枪顶在那个白人游客的后背，使得对方不敢轻举妄动。电梯一路往下，停在地下二层停车场。
<br/>　　“走！”
<br/>　　阿卡盼拿着消音手枪顶了顶白人游客的后背，示意他走出电梯，让另外两个人去把车开过来，在上车前，对着白人游客后颈来了一下，将那个白人游客击倒在地后，拉开后车厢门把佟千雪塞进汽车，自己也跟着钻进汽车，汽车一个加速直接朝着停车场的出口冲去。
<br/>　　被击倒在地的白人游客摸了摸自己的后脑勺，看着自己手掌上的血迹，摸出电话拨打了报警电话。
<br/>　　……
<br/>　　佟千莹回到酒店，还没走到电梯口，就被两个T国警察给拦住了，旁边的酒店经理负责进行翻译。
<br/>　　当得知佟千雪被人绑架带走后，佟千莹脸立刻变得苍白，身子一个踉跄，如果不是旁边的酒店经理上前扶住她的话，佟千莹说不定就会摔倒在地上。
<br/>　　佟千莹很快就从慌乱中恢复过来，第一时间跟大使馆方面取得了联系。大使馆方面在得知遭到绑架的，是一名现役军人的妻子后，问清楚佟千雪的位置后，立刻派人赶到了现场。
<br/>　　一名穿着黑色西装，衣领上别着一个国徽，戴着一副眼镜的中年男子走到佟千雪面前，说道：“我是驻T国大使，我姓杨。”
<br/>　　“你好，杨大使。”佟千莹伸手跟对方握了握手，说道：“刚才是我打的电话，遭到绑架失踪的是我的妹妹，我是杭山市来Q市进行学术交流的，我妹妹是跟随我来T国旅游的，出事的时候，我正在外面工作，回到酒店后，才得知的消息。”
<br/>　　“你说你妹妹的丈夫是现役军人，请问你方便告知他的服役部队吗？”杨大使说道：“我可能需要跟你妹夫所在部队进行联系。”
<br/>　　杨大使在得知佟千雪是现役军人家属的时候，心中就有了大胆的猜测，绑架的人绝对是有预谋的，或是为了报复，或是为了获取什么东西。
<br/>　　“我妹夫的部队似乎还挺神秘的，我也不知道具体的部队番号，我只是一次偶然的机会听我妹妹提起过，好像是叫什么15049科学研究所。”佟千莹回答道。
<br/>　　“你确定是15049科学研究所？”杨大使皱了皱眉，脸色变得有些严肃起来。
<br/>　　龙隐部队在T国有特勤小组在活动，而大使馆方面有时也会因为需要跟龙隐部队特勤小组进行配合。所以杨大使自然是知道这个“15049科学研究所”到底是什么地方！
<br/>　　被绑架的人居然是龙隐部队队员的家属，杨大使立刻意识了情况不妙，对佟千莹说道：“你现在跟我回大使馆，我需要立刻跟国内进行联系。”
<br/>　　“我妹妹是不是很危险？”佟千莹着急的说道：“她现在还有着身孕，绝对不能出事啊。”
<br/>　　佟千莹被杨大使带回了中方驻T国大使馆，并且被妥善的进行了安排。杨大使返回自己办公室后，首先跟龙隐部队驻T国的特勤小队负责人取得了联系。
<br/>　　龙隐部队驻T国的特勤小队队长姓葛，单名一个“铭”字。葛铭抵达大使馆后，没有任何的犹豫，直奔杨大使的办公室而去。
<br/>　　……
<br/>　　龙隐部队特种作战基地，赵天宇办公室，红色专线电话突然响了起来。
<br/>　　赵天宇停下手中的工作，看了眼办公桌上响个不停的红色专线电话，眉头皱起，这部电话一旦响起，就必然是有大事情发生。
<br/>　　在红色专线电话又响了一声后，赵天宇接起了电话，“喂？”
<br/>　　“龙头，我是T国特勤小队葛铭。”
<br/>　　电话那头传来葛铭的声音，“我现在在大使馆，几个小时前，我们这里发生了一件事情。战术A组慕容飞的妻子在四个小时前，在Q市王权酒店遭到绑架，失踪了。”
<br/>　　“你说佟千雪在T国遭到了绑架？！”
<br/>　　葛铭带来的消息不得不说让赵天宇感到无比意外，就连说话的声音都是拔高了几个度。
<br/>　　“是的，目前还不确定是谁动的手，至于动机……”
<br/>　　“复仇！”
<br/>　　葛铭还没有把话说完，就被赵天宇把话接了过去。
<br/>　　龙隐部队在境外有过很多行动，尽管他们尽可能隐瞒了自己的身份，可是这个世界上没有不透风的墙，再加上还有CiA、克格勃、摩萨德、军情六局这些老牌的情报组织，对于龙隐部队，他们肯定建立了相关的档案，所以要得到龙隐部队资料对于某些人和组织来说，并非那么难。
