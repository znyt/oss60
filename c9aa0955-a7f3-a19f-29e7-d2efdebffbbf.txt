　　国王的尖叫声停下，紧张的看着眼前的大洞，她真的跟不上楚涵这种说干就干的节奏。

<br/>　　国王又叫了出来：“啊！你！你——”
<br/>　　深洞中露出一个脑袋，楚涵看着她：“你到底要不要来？”
<br/>　　国王闭嘴，跟着跳下。

<br/>　　楚涵看着差不多，话也不说，直接时空月影斩出！
<br/>　　刷刷刷！

<br/>　　国王只是发了个愣，下一刻隧道中的那百来人，就全体倒下！
<br/>　　血液从前方的道路流淌过来，来到了她的脚边……
<br/>　　血流成河！
<br/>　　楚涵面不改色，继续朝前走：“跟上。”
<br/>　　国王一个颤栗，紧紧跟着楚涵的步伐。
<br/>　　当她路过那些尸体时，还能感受到地面传来的温热，甚至还有一两具抽·搐了几下。
<br/>　　国王惊悚的看着眼前楚涵的背影，她一直以为白允儿已经算是杀戮中的极致代表，杀起人来从未有过情绪波动，但没想到楚涵，更特么淡定！
<br/>　　而且这种战力，真的太恐怖了！
<br/>　　楚涵一路朝前，时不时的就有一队人马冲过来，全部都被他一次性屠杀干净，甚至有时候双方话都还没说出口，就已经全体倒地。

<br/>　　国王已经在这一过程中麻木了，有时候人还没杀完，就已经走了过去。
<br/>　　两人就这么一路走了不知道多久，前方的道路忽然开阔起来，紧接着，前方就是一处山谷平地，他们所在的地方，则是一个山洞口。
<br/>　　这时候的国王才恍然回神，有些惊诧眼前的开阔场景。
<br/>　　楚涵则是笑了笑，还真的很像猎者联盟！
<br/>　　之前挡住他们的那座山，内部果然被掏空了，其内是一座大型的建造厂！
<br/>　　建造的东西，也很让人诧异！
<br/>　　国王的双手都在颤抖，声音带着冷意：“有人盗窃了天运国的技术！”
<br/>　　眼前是一架才建造了一半的飞船，材料也是星玉，甚至外型也与楚涵见过的那艘飞上天空的一模一样。
<br/>　　根本就是仿造版！
<br/>　　楚涵意味不明的看了眼国王：“你的大臣里可不止国师一个内鬼啊！”
<br/>　　两人的出现，又杀了那么多看守队伍，自然引起了这里人的注意，前方正有好几队全副武装的人站在地面，虎视眈眈的看着楚涵二人。

<br/>　　两人的谈话引起了这群人的哈哈大笑。

<br/>　　“国王啊国王，天女已经不在了，你已经没有任何依仗难道还不够清楚吗？”

<br/>　　一句句嘲笑的声音响起，楚涵看了下人数，觉得用时空月影斩不太划算，而且这飞船也不能留。

<br/>　　楚涵一边让月影斩屠杀在这山谷中的叛变者，一边用修罗战斧砍飞船上的星玉，不一会儿就将半艘飞船上最重要的一部分砍坏了。
