　　第58章
<br/>　　“秦蛮，丢出去。”
<br/>　　“是。”秦蛮爽快答应一声。
<br/>　　程大雷没有选择怀柔，也没有施展自己在厚黑学上的造诣，他的选择是以暴制暴。
<br/>　　秦蛮站出来后，柳家也站出来五个人，都是二十左右的精壮汉子。走出来后脸上挂着戏谑的笑意，那表情仿佛在说，就能着你站出来呢。
<br/>　　“本来你是没有资格和我动手的，但今天我不介意陪你耍耍。”一脸上刺着字的汉子道。为防止奴隶逃跑，有时会在他们脸上刺字。不过这一项执行得不严，有时刺字有时不刺。像今天的柳家人，大部分成年男人脸上都有字，而女人脸上则全部没有，破坏掉相貌，是会影响她们售卖价格的。
<br/>　　“柳家军纵横海上几十年，令海盗望风而逃，你们在山上可能没听说过，所以你才敢和我们动手。”
<br/>　　“当初咱们要负责青牛山剿匪，这里早已经没山贼了。”
<br/>　　秦蛮有些无语，目光看向程大雷。程大雷已经流露出不满：还特么不动手在等什么！
<br/>　　秦蛮领会其意，跨步向前。他不动时像个老实木讷的老农，可一旦动起来便如猛虎出栏，大手抓住那刺字之人的胳膊，挥手砸出，七八米远的距离，狠狠砸在地上。
<br/>　　孟子云今天没有滋事，他看到这一幕捂住眼睛，心中想：我就知道是这样！
<br/>　　“直娘贼，他偷袭我，竟然敢偷袭！”刺字汉子在地上大汉。
<br/>　　他只是第一个被丢开的人，然后秦蛮就像猛虎入羊群，一个个精壮汉子像是面口袋一样被砸在地上。
<br/>　　柳家人看错了蛤蟆寨的山贼，至少看错了秦蛮。山贼总是不入流的，就像很多江湖游戏中，新手任务都是平灭一伙山贼开始。NPC对玩家说：你已经了解熟悉这个世界，现在某某地有一伙山贼作乱，你去杀10个山贼再来见我吧。
<br/>　　然而，秦蛮不一样，他是顶级山贼，能百人敌。蛤蟆寨连续发生的几次战斗，如果没有秦蛮是决计无法完成的。
<br/>　　柳家众人一片震惊，互相对视一眼，又有几个精壮汉子闯上去，这时候什么江湖道义，或者军人荣誉都算不得什么了。
<br/>　　可这并没有用。
<br/>　　柳家一共九十三人，有老人，有小孩，有妇人，真正还有战斗力的不过三十几个。仅仅这些人，并没法用人数优势压倒秦蛮。
<br/>　　无非是给秦蛮练手而已，秦蛮越来越兴奋，打到最后，他甚至扯开上衣，露出精壮的背部肌肉，汗水从脖子滑落。
<br/>　　而他身下，歪七扭八躺倒一大片人，不停呼痛的惨叫声是此刻秦蛮最后的BGM。
<br/>　　那白发老者名叫柳慕云，是现在柳家活着的人中辈分最大的，也是今天向程大雷发难的主持者。
<br/>　　程大雷一直坐着石头上，双手撑着下巴，并没有对眼前的场面表现任何惊讶。
<br/>　　柳幕云将目光看向他，眼神狐疑不定。
<br/>　　“还不走吗？”程大雷一只手放下，用另外一只手撑着脸颊：“难道你们以为我吃人肉的传说是假的？”
<br/>　　柳幕云不想做山贼，可他也不想离开蛤蟆寨，这是一件很矛盾的事情，但却是他的真实想法。
<br/>　　柳芷为难的看着这一幕，慢慢挪到程大雷面前。
<br/>　　“程当家，奴家……”
<br/>　　程大雷抬起脸颊，吐出一个字：“滚。”
<br/>　　最后山寨大门打开，柳家人全部离开蛤蟆寨，然后大门对他们关上。
<br/>　　等人都走尽后，程大雷嘴里吐出一句谁都不明白的话：“你们想当林冲，我可不是王伦喔。”
<br/>　　柳慕云带着其他人离开蛤蟆寨，但却并没有走远，他们就停留在蛤蟆岭下。几个老者聚在一起，嘀嘀咕咕商量着什么。
<br/>　　他们不想落草为寇，但却又的确无处可去。柳家原先的家主因为一些事被落叶城城主抄家，究竟有没有被冤枉，或者其中牵扯着什么利益，这些都已不太重要。
<br/>　　重要的是，现在这些人都是奴隶。
<br/>　　简单来说，他们是黑户，没有身份，不能进城，没有田地。这些事对程大雷来说肯定不算什么，山贼天生就是和官家对着干的，可对这些习惯了在帝国统治下生活的人来说，帝国偌大的山河，已没有他们的容身之地。
<br/>　　九十几个人，有些在做奴隶时受了伤，有些还生着病，相比之下，没有吃饭这样的小事，就算不得什么了。
<br/>　　天大地大，却无人立锥之地，除了……蛤蟆寨。已有人悄悄将目光望向蛤蟆寨。
<br/>　　蛤蟆寨内，徐神机可以说大发雷霆。
<br/>　　“这帮人真是狼心狗肺，我们救了他们，竟然忘恩负义想抢我们的山寨！”
<br/>　　“他们不想抢我们的山寨。”程大雷倒是不如何生气。
<br/>　　“可他们和我们动手，这件事总是有的吧！难道对救命恩人，就能这样做！”
<br/>　　程大雷没有理他，长长打了声哈欠。
<br/>　　黄三元过来，道：“大当家，人没有走，都在下面守着呢。”
<br/>　　程大雷点点头，示意自己明白。
<br/>　　其实程大雷可以理解他们，山贼终究是被人视作妖魔的存在，何况是和海盗打了半辈子交道的柳家。军人的荣耀，是死在战场上，最后马革裹尸，让史官在竹简上留下自己的名字。
<br/>　　程大雷并不鄙视这份荣耀，相反还十分尊重。
<br/>　　程大雷的意思是：你既然为这份荣耀活着，就也得准备为这份荣耀去死。
<br/>　　一手高举着荣耀，一手让别人为这荣耀去死，一边想要拥有荣耀，一边还想吃得好，住得好。
<br/>　　这……怎么能和程大雷前生的人一样无耻呢。
<br/>　　“等天黑传我的话出去……”程大雷对黄三元交待一番，回房补觉去了。
<br/>　　在天黑夜幕降临的时候，山寨大门打开，黄三元走出山寨，居高临下将程大雷的吩咐转达一番。
<br/>　　“蛤蟆寨开始招募人手，想要加入山寨者必须达到一定考核标准，并愿意遵守山寨规矩。”
<br/>　　这帮人饿的饥肠辘辘，蛤蟆寨里的炊烟飘到这里，黑夜里众人抬起一双双明亮的眼睛，望着着蛤蟆寨。
<br/>　　程大雷砸出了自己的态度：屈辱，或者去死。
<br/>　　
<br/>(本章完)
