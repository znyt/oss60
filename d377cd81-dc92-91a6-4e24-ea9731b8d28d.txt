　　莽通的脸上，带有着一丝绝望，大声的吼道。
<br/>　　“跟他拼了！宁死也不要被他镇压！”
<br/>　　“笑话，几个废物，就算是你们全盛时期，我都不会将你们放在眼中，更何况现在你们就连力量都没有多少！”
<br/>　　雷绝的眼神中，不屑到极致，脸上猖狂的笑道。
<br/>　　“负隅顽抗罢了，统统给我镇！”
<br/>　　雷绝的拳风，再次连连落下，似乎是要将楚风眠，都给一齐镇压。
<br/>　　“小子，你的实力的确不错，有着跟我抗衡的资本，只是很可惜，你还是太傻了，居然是白白将灵力耗费掉，现在你就算是后悔也没用了，乖乖去做一个魔奴吧！”
<br/>　　雷绝看着楚风眠，猖狂的大笑起来，他故意让楚风眠去抵挡魔意，正是要消耗楚风眠的灵力。
<br/>　　楚风眠的实力，他的确有着几分忌惮，但是现在，一个没有灵力的武者，又有什么用处。
<br/>　　“你真以为你的算计，我看不破？”
<br/>　　楚风眠微微一笑，心神一动，他身上本来缺少的灵力，近乎是在一瞬之间，全然的补充了起来。
<br/>　　一瞬之间，楚风眠身上的灵力，便是全部瞬间恢复过来。
<br/>　　“你！”
<br/>　　雷绝的眼神中，都是大吃一惊，看着眼前的一幕，简直是目瞪口呆。
<br/>　　刚刚楚风眠明明已经是消耗了那么多的灵力，他也无数次的探察过，楚风眠的灵力，的确已经是消耗一空了。
<br/>　　可是现在楚风眠的灵力，居然是如此之快的就全然恢复过来了，就算是吞服丹药，也不可能如此之快。
<br/>　　“小子，就算是你灵力恢复了又如何？也没有任何的意义！今日你还是必死无疑！”
<br/>　　突然之间，雷绝动了，他高高跳起，收回大部分威压之力，凝聚到拳风之上，便是再次凝聚一拳，向着楚风眠，直接轰击了过去。
<br/>　　“小子，我知道你在击败了武风云，有些实力，不过武风云，在我们内门看来，也不过都是废物罢了！”
<br/>　　“今日我便是让你知道，这内门弟子跟外门弟子的差距！”
<br/>　　雷绝出手，他这一拳的力量，凶狠到了极点，简直是天崩地裂，拳风仿佛是化为了一把巨大无比的铁锤，对着楚风眠的胸口，猛然砸了过去。
<br/>　　这一拳，便是要将楚风眠给彻底重伤，将楚风眠打到没有任何还手之力。
<br/>　　但是楚风眠的表情，却是平静到了极点，面对着这打过来的一拳，他也同样握拳，一拳打出。
<br/>　　“吼！”
<br/>　　随着楚风眠的一拳打出，一声龙吟，顿时出现。
<br/>　　这龙吟响起的一刻，周围封锁的空间，似乎都已经开始松动起来，在楚风眠的身边，突然便是无数龙威，直接守护这楚风眠的身躯。
<br/>　　楚风眠打出的那一拳，便是直接拳风化龙，咆哮一声，将雷绝打出的一拳，直接轰破。
<br/>　　这拳风化龙，更是发威，直接冲着雷绝的身躯，轰击过去。
<br/>　　“砰！”
<br/>　　一声巨响。
<br/>　　雷绝的身躯，都被一下子镇飞了出去，他的胸口上，衣衫便是打碎，从这衣衫之中，露出了一件灵器铠甲。
<br/>　　这灵器铠甲上，都是打出了一个拳印，被硬生生的打碎。
<br/>　　若是没有这灵器铠甲护身，这一拳之下，雷绝就会被打穿胸口，直接陨落。
<br/>　　这等力量，是何等的恐怖，先是化解了雷绝那一拳的力量，然后更是直接将灵器铠甲，都给打碎。
<br/>　　雷绝的表情，简直是惊骇到了极点，眼珠子都要掉了出来。
<br/>　　这灵器铠甲，可是一件地级灵器，跟随了他无数岁月，不知道是保护了他多少次的性命。
<br/>　　曾经雷绝受到一尊刚刚踏入生死境的强者追杀，他便是就依靠这灵器铠甲，活了下去。
<br/>　　当时那生死境强者，全力之下，都只是能够在这灵器铠甲上，打破一个缺口。
<br/>　　却是没有像楚风眠这样，一拳之下，直接全然破碎。
<br/>　　这灵器铠甲上的灵力，如今都已经是散发了出去，全部消失了，这也就意味着，这灵器铠甲，就此已经是彻底废了。
<br/>　　这等恐怖至极的灵力，居然是由一尊御风境武者打出来的？
<br/>　　楚风眠之前在外门中，击败武风云的时候，表现出的实力，都已经极为恐怖了。
<br/>　　但是现在看来，当时楚风眠表现出来的实力，最多却只能算是冰山一角，楚风眠的真正实力，还要更加恐怖。
<br/>　　“我的天！”
<br/>　　“楚风眠的实力，怎么会……”
<br/>　　“太可怕了，这等实力。”
<br/>　　另外三人，都是目光呆滞的看着眼前的变化，他们也想不到，楚风眠居然会突然爆发出如此可怕的力量来。
<br/>　　随后他们三人的表情，便是大喜，楚风眠隐藏着如此实力，那现在，便就不是真正的必死局面了。
<br/>　　“你！你居然拥有这样的实力！”
<br/>　　雷绝满脸惊骇，指着楚风眠，不可置信的喊道。
<br/>　　“以你的实力，为什么会听从我的话，你早知道我的计划，为什么会等到现在！”
<br/>　　“不留你一条命，怎么将你背后的人抓出来，没想到，这居然还是一条大鱼。”
<br/>　　楚风眠冷哼一声道。
<br/>　　“雷绝，你残害同门，勾结地魔门，已经不配作为北邙学院的弟子了，今日我便杀了你！除去了你这个祸害！”
<br/>　　开口同时，楚风眠便是大手一挥，又是一拳，向着雷绝轰击了过去。
<br/>　　本来在他的预料之中，刚刚的那一拳，已经是足以轰杀雷绝了，只是没想到他的身上，居然还有着这样一件铠甲灵器护体。
<br/>　　不过这一次，可没有铠甲灵器可以在帮他挡下来了。
<br/>　　“不好！暗古魔尊救我！”
<br/>　　看着楚风眠打出的一拳，雷绝脸上惊慌到极点，大声喊道。
<br/>　　突然之间，一道血红魔爪，顿时抓了过来，抵挡着楚风眠的灵力，一爪之下，将楚风眠打出的一拳，其中的灵力，直接抓碎。
<br/>　　“年轻人，做事不要太绝。”
<br/>　　一道沙哑声音响起，正是那暗古魔尊开口道。
<br/>　　“此人毕竟是为我做事，给老夫一个面子，留他一命吧。”
<br/>　　暗古魔尊开口同时，他的大手一挥，便是直接向着雷绝抓了过去，要将雷绝给带走。
