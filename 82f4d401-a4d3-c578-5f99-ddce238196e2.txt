　　这一枚通红色的果子从空戒之中飞出来的一刻，一股热浪便是向着楚风眠席卷而来。
<br/>　　一眼望去，在这一枚果子之中，仿佛是有着火焰在熊熊燃烧。
<br/>　　“这是炎朱果？”
<br/>　　楚风眠大手一挥，一股灵力镇压而下，才将这一枚通红色果子之中蕴含的狂暴灵力给镇压了下去。
<br/>　　这这样一枚小小的果子之中，都蕴含着足以让九劫古帝都要避退三舍的力量。
<br/>　　那炙热的火浪，如果是一尊九劫古帝稍有不慎被波及到，都要受伤。
<br/>　　楚风眠记得，从古籍之中，有记载过眼前这通红色果子，名为炎朱果。
<br/>　　这炎朱果，乃是火系圣药，是任何修炼火系功法武者梦寐以求之物。
<br/>　　相传这一枚炎朱果，都足以增加一尊修炼火系功法的九劫古帝突破半步天人境的一成几率。
<br/>　　一成几率，听起来少，可对大部分的九劫古帝来说，哪怕是有着半成突破半步天人境的几率，他们都会选择冲击。
<br/>　　毕竟大部分的九劫古帝，可是穷其一生，都摸不到半步天人境的门槛。
<br/>　　这炎朱果，不止是在九域，整个大罗天星域之中都彻底的灭绝了，再也找不到了，现在这火天的空戒之中居然有。
<br/>　　而且数量，更是不在少数，楚风眠一眼看过去，足足是发现了十四枚。
<br/>　　“好东西，我乃是冰火神体，这炎朱果，正好对我有用。”
<br/>　　楚风眠看到这炎朱果的一刻，大笑一声，便是毫不犹豫，一把抓起其中的一枚炎朱果，吞噬到了口中去。
<br/>　　与此同时，楚风眠也是运转起吞天秘术来，开始炼化这一枚炎朱果之中蕴含的灵力。
<br/>　　这样一枚炎朱果，换做是寻常同境界的武者，至少是需要炼化数日的时间。
<br/>　　不过如今在吞天秘术下，这炎朱果刚刚被楚风眠吞入到了身躯之中去，便是破碎开来，化为了无比精粹的灵力，涌入到了楚风眠的身躯之中去。
<br/>　　一时间楚风眠便是感觉到，他的身体之中有着一股热流涌动。
<br/>　　这炎朱果之中蕴含着纯粹的火系灵力，所以对修炼火系功法的武者妖神，有着极大的好处。
<br/>　　楚风眠虽然并非是修炼的火系功法，可楚风眠凝炼了冰火神体，说起来楚风眠的身躯之中对火灵的掌控，比起许许多多修炼控火之道的武者还要厉害的多。
<br/>　　这炎朱果的灵力，一瞬间便是就被冰火神体所吞噬，来增添这楚风眠的力量。
<br/>　　“一枚蕴含的力量太少了。”
<br/>　　楚风眠感觉到，他的力量提升，甚至就连百分之一都不到，至于距离突破境界，冲击半步天人境的瓶颈，更是远远触碰不到。
<br/>　　对寻常的九劫古帝来说，这样一枚炎朱果，都足以是让他们有所机会去冲击半步天人境了。
<br/>　　纵然是无法冲击半步天人境，也至少是需要提升五成以上的力量。
<br/>　　可现在这样一枚炎朱果被楚风眠所炼化，却是就连百分之一的力量都无法增长。
<br/>　　这就是楚风眠力量强横的另一面，楚风眠的力量强横无边，虽然是九劫古帝的境界，但是却是要比起许许多多的半步天人境还要强大。
<br/>　　正是如此，楚风眠想要在提升境界，也是无比困难。
<br/>　　以楚风眠一身的力量，这样一枚炎朱果，不过是杯水车薪罢了。
<br/>　　楚风眠又接连炼化吞噬了两枚炎朱果，他的力量才终于是增加的百分之一左右。
<br/>　　不过楚风眠也随之停下的炼化炎朱果，因为楚风眠已经是发现了，在吞噬炼化了三枚炎朱果之后，这炎朱果便是对楚风眠，再也没有效果了。
<br/>　　像是诸天奇物，一些圣丹，圣药，一尊武者如果服用大量的话，就会失去效果。
<br/>　　这炎朱果的极限，便是三枚，一旦是炼化了三枚炎朱果之后，在多的炎朱果，也没有任何的用处了。
<br/>　　楚风眠也不想白白浪费剩下的炎朱果，便是将这些剩下的炎朱果全部收了起来。
<br/>　　这些炎朱果，对楚风眠来说用处不大，可是要是给一名寻常的九劫古帝，三枚炎朱果，至少可以增加两成冲击半步天人境的机会。
<br/>　　这剩下的十一枚炎朱果，至少足以缔造出一位半步天人境武者了。
<br/>　　“可惜我想要突破，遥遥无期，就算是炎朱果，对我用处也不是很大。”
<br/>　　三枚炎朱果，最多也就是给楚风眠增添了百分之一突破半步天人境的机会。
<br/>　　可这种机会，现在楚风眠强行突破，无异于是在自己找死。
<br/>　　现在楚风眠炼化了太初气息，成为了天地异种之后，已经是成为了天道的仇敌，一旦是楚风眠突破境界，只怕是降下的劫雷，不会亚于逆天大劫的威能。
<br/>　　所以在没有准备好一切的情况下，楚风眠决不能贸然的选择突破。
<br/>　　想要冲击半步天人境，楚风眠必须要做好万全的准备才行。
<br/>　　“如果能够找到一些仙物就好了。”
<br/>　　楚风眠不由的想到，想要让楚风眠有把握冲击半步天人境，最好的办法便是寻找仙物，只有仙物，才有增添楚风眠突破的几率。
<br/>　　不过想要寻找仙物，谈何容易，仙物二字，便是代表这，至少是足以让天人境，羽化成仙的人物，都去争夺的宝物。
<br/>　　像是仙药，仙草，仙丹等等，远不是楚风眠这个境界的武者，可以触碰到的。
<br/>　　楚风眠一边想着，一边也是在扫视着其他几人的空戒，在他们的空戒之中，楚风眠也发现了一些不亚于炎朱果级别的宝物。
<br/>　　这个级别的宝物，对楚风眠力量的提升不是太大，可依然有些用处。
<br/>　　楚风眠也毫不客气，将所有对他有用的宝物，全部吞噬，至于剩下的那些，则是都被楚风眠收取了起来。
<br/>　　除此之外，还有着一些灵器，炼器材料等等，都被楚风眠分了下去，赐予了剑道门的众人。
<br/>　　这些灵器对楚风眠都没有多少的作用，至于那五人手中的半仙器，则全部被楚风眠交给了神树，让神树吞噬。
<br/>　　神树可以吞噬灵器，在增添力量，这五件半仙器，虽然已经是被楚风眠硬生生打裂，威能消散了不少，可毕竟是天人炼制，上面沾染这仙力，对神树来说，可是大补之物。
