　　“试验准备的怎么样了。”比杨德身材魁梧，即便穿着宽松衣袍，依旧难以掩盖雄壮身躯。
<br/>　　面貌如同刀削斧凿，棱角分明。一双眼睛漆黑幽深，充满神采。
<br/>　　“卡金帝国那边，已经重新开始投入资金。幸好之前准备了备份资料，不然就只能重新开始了。”帕里斯通坐在对面沙发上，右腿搭在左腿上翘着，一脸的随意。
<br/>　　“即便是现在，想要取得成果，也要一两年的时间。”帕里斯通满脸无奈：“揍敌客家的那小子，动手太彻底了。资料、人员一个不留。少了经常接触试验的人员，再次启动需要更多时间。”
<br/>　　“只要能够进军黑暗大陆，等多久都没问题。”比杨德沉声道，脸上表情丝毫不变：“只是可惜了赛右，少了一个好手。”
<br/>　　“揍敌客啊，真是不简单。”帕里斯通收起脸上随意，一脸感慨道。
<br/>　　“仅仅只是一个十三四岁的少年，就能够打败赛右。揍敌客家的老家伙，该有多厉害？”帕里斯通有些拿不定注意，苏重的恐怖战斗力，让他的认知有些混乱。
<br/>　　“多厉害？很厉害！”比杨德沉默半晌道。因为父亲尼特罗的缘故，他比别人更了解揍敌客。知道揍敌客，有着一个不知道活了多久的老怪物。
<br/>　　“那卡金帝国那边怎么办。揍敌客摧毁了实验室，那些大人们可是很生气呢。”帕里斯通脸上带着玩味的笑容。揍敌客和卡金帝国，甚至还有巴托奇亚共和国和卡金帝国，这之间的矛盾争斗，让他很感兴趣。
<br/>　　“我会去见见那位天才揍敌客。”比杨德沉声道。
<br/>　　“你敢杀他？那代价是不是太大了，你就不怕揍敌客报复？”帕里斯通饶有兴味的看着比杨德。
<br/>　　“和进军黑暗大陆相比，付出再大的代价都值得。”比杨德平静道。
<br/>　　帕里斯通挑了挑眉，没说话。对于比杨德的执着，他早就见识过。
<br/>　　“既然你出手，那小子的好日子也到头了。相比于取悦卡金帝国而杀他，那颗重力水晶反而更重要。”帕里斯通想了想道。
<br/>　　一切纠葛的起因，不过是一个小小的重力装置。黑暗大陆充满危险，有的地方重力分布非常复杂。
<br/>　　前脚可能还是正常重力，下一步可能就会踏入超重领域内。猝不及防之下，很容易受伤折损。
<br/>　　当初搜集重力装置，为的就是抵挡这种情况。就只是一种可能的情况，却引起了一系列后续事件。
<br/>　　不过，有比杨德出手，到如今也该结束了。
<br/>　　……
<br/>　　世界树很高且很陡，近乎直角，攀爬起来非常困难。而且，随着高度攀升，还要承受高度所带来的恐惧。
<br/>　　为了攀爬，很多人都带着工具，一边攀爬，一边在树干上安装踩踏装置。
<br/>　　苏重身体素质极其强大，只是随手扣住树皮的皲裂，便能借力攀升。遇到光滑树干，他就直接发动龙牙，五指如钩，直接插进坚硬树干内部成为借力点。
<br/>　　依仗强大体魄，苏重飞快的向上攀爬，超过一个又一个攀爬者。就好似行走在平坦道路上一样，那轻松写意，让其他挑战者看得目瞪口呆。
<br/>　　他很快就爬到了顶端。
<br/>　　一个巨大的鸟巢安在顶端，好似一个帽子。苏重爬上鸟巢，就看到数个人高的巨大鸟蛋，堆积在巢**。
<br/>　　“也不知道是什么鸟，只是蛋都那么大。孵化后完全成长起来，恐怕比神雕还要大。”
<br/>　　猎人世界很奇特，怪兽、珍兽、魔兽什么稀奇古怪的东西都有。像这种体型庞大的鸟类并不少见。
<br/>　　只是打量了片刻，苏重便不再理会。静静的坐在鸟巢边缘，静静的看着远处天空。下方的城市，远处的森林，一切都在脚下。心胸不由开阔。
<br/>　　“这里倒是一处放松心情的好地方，等找到地下岩浆。可以把这里作为修炼后的放松地点。”
<br/>　　四处看了看，大大小小的山峰全被苏重光看在眼中。并未发现有火山存在。到处都是一片绿色。
<br/>　　“大概是世界树吸收了熔岩热量，地下岩浆丝毫无法突破地表，四周根本就没火山。”苏重无奈。想要像在枯枯戮山一样，直接钻进火山底修炼，已经没那种好事了。
<br/>　　世界树能生长，证明地下有着不小的熔岩区域。只要找到正确方位，进入地下，一定能找到自己需要的地下岩浆河流。
<br/>　　欣赏了一会儿远方景色，苏重一步踏出鸟巢，快速下坠。他准备开始调查。
<br/>　　之后几天，苏重以世界树为中心，不断探查。进入城市，调查相关信息。希望能够找到地下岩浆的信息。
<br/>　　数天的调查，让苏重彻底绝了寻找火山口的主意。世界树周围不仅没有火山，甚至连深入地下的洞穴都不存在。
<br/>　　数日来毫无结果，苏重有些失望。慢悠悠走上世界树木质阶梯，苏重准备到树顶上散散心。
<br/>　　木质阶梯末端，有着一个简单的木屋。里面有一个壮硕中年人，他向每一个攀爬大树的人推销求救手环。依靠搜救工作，赚取财富。
<br/>　　几乎每一个攀爬的人，都会买一个手环。苏重没买。
<br/>　　看到苏重走过来，中年人走出木屋，手里拿着一个手环走向苏重。
<br/>　　苏重挥了挥手，示意自己不需要。可中年人却脸上带着笑，径直走到苏重面前。
<br/>　　“你在找地下岩浆的消息？”中年人一开口，就说出了苏重来此地目的。
<br/>　　苏重扫了对方一眼，并不意外。他从未隐藏自己的目的，大张旗鼓的寻找地下岩浆，被别人知道并不奇怪。
<br/>　　而且这中年人，能够长期霸占搜救工作，这一颇具利润的行业。肯定是地头蛇，消息灵通。
<br/>　　“手环给我吧。”苏重把钱递给中年人。
<br/>　　壮硕中年人脸上笑眯眯，一副你很上道的表情。点验清楚钞票，中年人也不隐瞒：“正南方向，二十多公里有一处小山谷。山谷最深处，有一个天然温泉，被树木遮掩着。那里硫磺味很浓，说不定有你要的东西。”
<br/>　　“还有吗？”苏重并不满足。既然找到自己，就绝不可能只这么点儿消息。
<br/>　　中年人嘴角一翘起，知道自己赌对了：“五十万戒尼。这可是独家消息，要不要你说了算。”
<br/>　　苏重眼睛一眯，这要价可有点儿高。
<br/>　　“可以。”苏重平静道。
<br/>　　中年人闻言大喜。
<br/>　　“你最好保证消息准确，否则。”苏重右手猛然一握。
<br/>　　砰！
<br/>　　一声闷响，空气直接被苏重捏的爆鸣！一阵强风在两人之间飞卷。
<br/>　　中年人浑身一僵，瞳孔骤然收缩。本以为是只肥羊，没想到是个硬茬！
<br/>　　他顿时收起所有小心思，老老实实的把消息告诉苏重。
