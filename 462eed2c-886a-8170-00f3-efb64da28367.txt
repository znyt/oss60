　　“走吧。”
<br/>　　这木邪主动退去，楚风眠也懒得在计较什么。
<br/>　　他看得出来，这木邪也是隐藏这不少的实力，他的真正实力，只怕是比起夜天君，冷面魔君等人，都是在伯仲之间。
<br/>　　楚风眠出手想要对付他，也并非是一件容易的事情。
<br/>　　但是若是真正的生死搏杀，楚风眠也不会怕这木邪，如今在这归墟城中，没有人想要提前暴露出自己的底牌，楚风眠也一样。
<br/>　　等到进入到了归墟天之后，楚风眠便是有着一万种办法，杀了这木邪跟这些玄天宗弟子。
<br/>　　在一旁，那王辉听到楚风眠的话，稍稍愣住，随后急忙的反应过来。
<br/>　　“两位这边请。”
<br/>　　王辉一边说着，一边便是将楚风眠跟寒月漓带走了，一直是来到了归墟城中最大的一座洞府之一，进入了其中。
<br/>　　这洞府，便是归墟城的上九天洞府之一，是归墟城中资源最好的洞府。
<br/>　　看着楚风眠进入其中，那几名玄天宗弟子的脸色，都是露出几分恨意。
<br/>　　他们这些玄天宗弟子，在南域几乎都足以称得上是一手遮天的存在，别说是有人敢跟他们为敌，根本都从来没有人敢跟他们争抢什么。
<br/>　　现在居然是从楚风眠的手中，吃了这样一个亏，他们一个个的眼神中，还充满这怒火，身上的灵力都被凝聚出来，随时可能发怒的样子。
<br/>　　不过在这木邪的面前，他们却都不敢出手。
<br/>　　“现在不是跟此人起冲突的时候，他敢招惹到了我们玄天宗，已经是一个死人了，不过我们这次来的目的，还是为了归墟天中的宝藏，现在不好暴露太多，等到了归墟天，便是此人的死期！”
<br/>　　木邪冰冷的开口道，他的话语中都是蕴含这十分的怒意。
<br/>　　他又看了一眼那鹰鼻男子，开口道。
<br/>　　“你们照顾好虎师弟，在归墟天开启前，让他的伤势恢复。”
<br/>　　“是。”
<br/>　　另外两名玄天宗弟子，便是急忙点头，便是将那鹰鼻男子，带到了一处稍小一些的洞府之中去，开始休养生息。
<br/>　　楚风眠的这一战，令平静的归墟城都是嫌弃了几分波澜，不少人都是议论着。
<br/>　　尤其是楚风眠一招便是击败那鹰鼻男子，这也让不少人有些诧异。
<br/>　　一直以来没有出现过什么样强者的北域，终于是出现了一名像样的天才了。
<br/>　　这种议论，很快便是都逐渐平息。
<br/>　　大部分的武者，都是选择好了自己的洞府，倒是没有在起什么冲突。
<br/>　　在这个时候，起冲突是最为愚蠢的行为，倒是一直都是相安无事。
<br/>　　“这次归墟天开启，看来是将九域中的大部分天才，都是吸引过来了。”
<br/>　　楚风眠坐在上九天的洞府之中，扫视这外面无数武者的气息，自言自语的开口道。
<br/>　　这上九天的洞府，是一座巨大无比的庭院，在这庭院中，浓郁的灵气，甚至都是化为了一口小小的灵泉，其中的灵液，都是最为精粹的灵气凝聚而成的。
<br/>　　这庭院绝对不亚于外面的一些修炼圣地。
<br/>　　楚风眠就坐在这灵泉旁边，汲取这其中的力量，虽然他的力量达到瓶颈，不能再进一步，不过楚风眠却是足以将其中的灵力，凝聚成为灵石，收集在他的空戒之中，积蓄力量。
<br/>　　“这些天才，正好，每个人的身上都有着无数奇遇，其中不少人都懂得许多剑术，正是对我有着大用，是我将九域剑术再度凝聚最好的机会。”
<br/>　　楚风眠打量着周围武者，心中冷笑这想到。
<br/>　　“我的黑玄剑阵，还可以继续祭炼，他们手中的灵器，都可以融入到黑玄剑阵中，他们身上的武技，全部也都足以融入到我的九域剑术之中，等到了归墟天，便是我疯狂杀戮的时候。”
<br/>　　楚风眠的眼神中，都露出几分杀意。
<br/>　　如今在外的那些武者，不少人都想要杀了楚风眠，但是楚风眠同样有着一样的想法，谁敢来招惹楚风眠，楚风眠便是就要杀了谁。
<br/>　　在归墟天中，没有规则，碰到谁楚风眠都可以直接斩杀。
<br/>　　“月漓，等到了归墟天中，小心一点，这进入归墟天开启的法阵，极为混乱，进入其中到底会出现在哪里，都是未知的，很有可能会被直接传送到一些危险凶地之中，遇到危险，尽可能的躲藏起来，只要是捏碎玉符，我就可以寻找到你，来帮助你。”
<br/>　　楚风眠看着旁边的寒月漓，手中灵力便是凝聚化为一枚玉符，直接递了过去。
<br/>　　这归墟天是一处独立的空间，所谓的开启，也是许多强者以无上手段，趁着归墟天力量最为薄弱的一刻，从中撕裂开一个口子。
<br/>　　但是一旦武者进入其中，到底会被传送到哪里，都是一个未知数，所以楚风眠也不能保证可以跟寒月漓在一起。
<br/>　　而且也有些运气不好的武者，进入其中，直接传送到了归墟天中的一些凶地之中，当场陨落的。
<br/>　　真正足以从归墟天中活着离开的武者，十不足三，至少七成的武者，都会死在其中。
<br/>　　不过一旦是可以从归墟天中活着离开，都足以得到极大的好处，像是毒手魔君，在进入归墟天的武者中实力只能排位末流，并没有机会夺取大的宝藏，都有着那么多的收获。
<br/>　　如此之大的收获，也足以是让无数天才武者，进入其中，甚至都要限制名额。
<br/>　　寒月漓接过玉符，凝重的点了点头。
<br/>　　关于归墟天的危险，她如今也已经知晓，十不足三的生存率，任何人都不能轻视这归墟天的危险。
<br/>　　到了归墟天中，任何武者都是敌人，能够依靠的，只有自己。
<br/>　　现在寒月漓的实力，虽然在这一次进入归墟天中的武者中，算不上顶尖，但是也一样算不上弱。
<br/>　　应该不会太过危险。
<br/>　　七日时间，很快便是过去。
<br/>　　在这七日时间之中，又有不少武者，来到了归墟城中，起了几次冲突。
<br/>　　只是这起的几次冲突，都是点到而止，每个人都占据了一处洞府，开始休息起来。
<br/>　　一直到了这七日之后，整个归墟城中，才突然出现了一声声钟声。
