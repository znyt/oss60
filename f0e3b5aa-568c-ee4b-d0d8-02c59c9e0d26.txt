　　嗖，萧逸的身影在原地一闪。
<br/>　　下一秒，瞬间出现在被击飞的慕容杀身前。
<br/>　　“不好。”慕容杀一惊，刚才那一拳，让他清楚知道，面前的少年，实力极强。
<br/>　　他再不敢大意，立马调动真气护身。
<br/>　　然而，萧逸再次一拳，顷刻间破了他的护身真气，手上早已凝聚出烈焰手套。
<br/>　　在灵器的振幅下，漫天紫炎澎湃而出，将慕容杀包裹住。
<br/>　　“怎么可能这么强？”慕容杀惊骇至极。
<br/>　　武者的肉体，并不强大。若无真气护身或调动，实力跌了九成不止。
<br/>　　之前，萧逸之所以能将他轰飞。
<br/>　　就是靠着自身媲美洞玄四重的肉体，强行破了他的护身真气，而后紫炎爆炸，将其震飞。
<br/>　　如今也一样。
<br/>　　慕容杀想调动真气护身，但真气刚刚从体内溢出，便已经被周围的紫炎所烧毁，根本凝聚不起来。
<br/>　　或者更简单来说，紫炎与他的真气在互相消耗着。
<br/>　　这样诡异的情况，让他不敢再有任何动作。
<br/>　　只能被动地与面前的萧逸对视着，警惕着。
<br/>　　“慕容杀，我再问你一次，有没有证据。”萧逸冷冷地问道。
<br/>　　“我...”慕容杀一时语塞，不敢乱说话。
<br/>　　那漫天的紫炎，将萧逸与慕容杀身旁十米内的地方完全覆盖。
<br/>　　如同一道火墙，让任何人都不敢靠近。
<br/>　　包括盛宝商行的护卫，以及慕容胤。
<br/>　　慕容胤不过是洞玄四重，连慕容杀都不敌易霄的紫炎，他哪里敢有任何动作。
<br/>　　周围的宾客，一个个都惊骇地看着。
<br/>　　“紫炎易霄，竟然强到这个地步吗？”
<br/>　　“慕容杀可是洞玄六重武者，竟被逼得不敢说话。”
<br/>　　宾客们惊讶地说着。
<br/>　　这时，漫天紫炎内，慕容杀脸色一阵变幻，道，“紫炎，谁都知道那次的商物，是你抢的，你狡辩有用吗？”
<br/>　　萧逸冷笑一声，反问道，“你说呢？”
<br/>　　话音刚落，他将周围的紫炎悉数散尽，冷声道，“盛宝商行，既然无凭无据，若再敢拦我，易某真的会杀人的。”
<br/>　　说罢，他淡淡地转过身离开。
<br/>　　全场武者，无人敢拦。
<br/>　　慕容杀以及慕容胤，只能不甘地看着他潇洒离去，却不敢有任何动作。
<br/>　　事实上，慕容杀就算无法凝聚真气护身，但实力仍旧是洞玄六重。
<br/>　　若他敢放手一搏，硬抗紫炎，说不定能拼着重伤的代价，擒下萧逸。
<br/>　　只不过，他不敢。
<br/>　　萧逸那一手雷霆动作，彻底震住了他。
<br/>　　......
<br/>　　百武城外。
<br/>　　白家主惊讶地看着萧逸，道，“易霄小兄弟，你到底是什么修为？”
<br/>　　“年纪轻轻，竟然有如此实力。”
<br/>　　萧逸疑惑道，“白家主看不出我的修为吗？”
<br/>　　白家主摇摇头。
<br/>　　“嗯？”萧逸心头一阵疑惑，“白家主可是成名已久的破玄境武者了，竟看不出我的修为境界？”
<br/>　　按照幽魂面具的特效，理应是高出自己一个大境界的武者，就能看穿自己的气息和修为的。
<br/>　　这时，白家主道，“易霄小兄弟你身上应该有可以隐藏气息的宝物吧，而且还不是一般的宝物，乃是重宝。”
<br/>　　“重宝？”萧逸面露疑惑。
<br/>　　白家主点头道，“如果我没有猜错的话，就是你脸上这块面具。面具上明显有一层禁制，让我无法感知你的气息。”
<br/>　　萧逸本以为，幽魂面具，顶多算亚灵器。可现在看来，连白家主这个层次的人都无法看穿这层效果，那么，它的功效和品阶，得重新审视了。
<br/>　　这时，白家主笑笑，道，“易霄小兄弟不要误会，我只是觉得，你身怀重宝，我白家也不知这次该拿什么东西报答谢你才好，更不知什么东西你才看得上眼。”
<br/>　　萧逸笑笑，道，“白家主不必介怀，真要答谢易某的话，便帮我找些妖兽精血吧。”
<br/>　　“妖兽精血？”白家主一愣，道，“这些寻常东西，如何能当答谢之物。”
<br/>　　萧逸道，“我要的数量很大。”
<br/>　　“数量很大？”白家主疑惑道，“妖兽精血，用以药浴，可增强修炼速度。可是这东西，一瓶就够用很久的了，多了也是无用。”
<br/>　　“哦，对了。”白家主道，“你是炼药师，怕是有其它炼药用途吧。”
<br/>　　“嗯。”萧逸点点头。
<br/>　　“那好。”白家主认真道，“随我回白家一趟吧，我白家有不少库存，皆是四级妖兽精血。”
<br/>　　“另外，四季城内也有不少商行。白家全力替你收购，一切费用由我白家承担。”
<br/>　　萧逸拱了拱手，道，“那便谢过白家主了，不过，易某暂时不去白家了。”
<br/>　　“嗯？”白家主面露疑惑。
<br/>　　萧逸苦笑一声，道，“如今，百武城盛宝商行的人视我为肉中刺，眼中钉。我与他们恩怨不浅，还是不与白家走得太近为好，免得牵连了你们。”
<br/>　　白家主眉头一皱，道，“易霄小兄弟这是什么话？”
<br/>　　“初次识你，你便救小女一命。而我白家除了区区五十万两外，再无其它答谢之物，已是欠你人情。”
<br/>　　“今日，得你相助，我才能得到星芒石，又欠一人情。”
<br/>　　“我白家，认你是朋友，岂会在乎那些。”
<br/>　　“再者。”白家主傲然道，“我白家，也不是好惹的。”
<br/>　　“我知道你抢过盛宝商行的货物，虽然不知你是出于什么原因。但，北山慕容家还不至于为了这些小事而和我们白家过不去。”
<br/>　　萧逸笑笑，有破玄境武者坐镇的家族，说话就是有底气。
<br/>　　但他还是摇摇头，道，“易某还有要事在身，确实没时间去白家了。白家主先替我收购妖兽精血吧，过些天我会再登门叨扰。”
<br/>　　“既是有要事在身，那我也不强求。”白家主道，“总之，你记着，白家认你是朋友。若有需要，尽管来白家，白家无任欢迎，也会全力助你。”
<br/>　　“谢白家主了。”萧逸道谢一声，拱拱手，道，“如此，易某告辞。”
<br/>　　话音落下，萧逸身影一闪，消失在原地，飞奔远去。
<br/>　　......
<br/>　　萧逸离去，白家主脸色一正，对身后的白管家道，“回白家后，我要闭关一番。”
<br/>　　“你们，发动白家的所有力量，无限量收购妖兽精血。”
<br/>　　“一月后，我要看到让我满意的数量。”
<br/>　　“是，家主。”白家众武者应了一声。
