　　少倾之后，‘巫天御’动了，身形缓缓的站了起来。
<br/>　　“此肉身还不错！”巫天御向苏莫笑着说道，但此刻的巫天御已经变成了阎魔鬼圣。
<br/>　　“鬼圣前辈，你现在重新拥有了肉身，不知多久能恢复到巅峰境界？”苏莫问道。
<br/>　　“有了肉身的滋养，魔魂和修为想要恢复，少则十年，多则百年！”阎魔鬼圣说道。
<br/>　　苏莫闻言点了点头，对方毕竟是武圣强者，即便现在落魄，但恢复巅峰也是迟早的事情。
<br/>　　“对了前辈，您老见多识广，给我说说太阴族的情况！”
<br/>　　随即，苏莫向阎魔鬼圣问道，对方活了这么久，又是武圣境的强者，想必应该了解一些太阴族。
<br/>　　“太阴族？”
<br/>　　阎魔鬼圣闻言一怔，随即思忖了片刻，道：“太阴族是天荒星域有名的强族，不在古灵星河，而在附近的明虚星河；不过，太阴族的势力非常庞大，辐射周围数十个星河！”
<br/>　　“这么厉害！”苏莫闻言倒吸了一口凉气，势力辐射数十个星河，这也太强了吧！
<br/>　　一个星河之内，星辰数量都是数以万计，甚至数以十万计，由此可见太阴族的强大。
<br/>　　随后，苏莫向阎魔鬼圣详细的了解了一番太阴族的情况，对方也将所知全部告诉了他。
<br/>　　太阴族，与普通人族无异，但只修炼至阴至寒的功法。
<br/>　　这让苏莫心中好奇了起来，青璇是纯阴之体，是修炼至阴至寒功法的绝世天才，难道太阴族是要将青璇纳入太阴族？
<br/>　　和阎魔鬼圣聊了片刻，苏莫对于太阴族有了大概的了解，他暂时实力太低，不准备去太阴族，还是先回神武学府提升实力为好。
<br/>　　“苏莫，后会有期了！”阎魔鬼圣向苏莫抱拳，准备离开了。
<br/>　　“后会有期！”苏莫同样抱拳。
<br/>　　“苏莫，汝的天赋震古烁今，万古罕见，十年内可成尊、百年内可成圣、千年内有希望问鼎虚神，希望将来能看到汝威临宇宙的那一日！”阎魔鬼圣郑重的说道。
<br/>　　“承前辈吉言，我若不死，必然问鼎！”苏莫自信的说道，现如今，他的自信心已然强大到了爆棚的地步。
<br/>　　吞噬战魂的恐怖，也已经彻底的展露了出来。
<br/>　　现在，他身具无数战魂，三种特殊体质，试问天下间，谁人的天赋都超越他！
<br/>　　“嗯！”阎魔鬼圣微微颌首，随即深深的看了苏莫一眼，身形一闪，破空而去。
<br/>　　少倾之后，苏莫也飞身离开，向途山镇而去。
<br/>　　几日之后，苏莫到达了途山镇。
<br/>　　时隔近两年，途山镇又有了很大的变化，已经建起了高高的城墙，井然成为了一座小城。
<br/>　　城中虽然人口不是太多，但也算是热闹繁华。
<br/>　　唰！
<br/>　　苏莫直接飞进了苍穹神宫的府邸之中，他的到来，不可避免的有引起了府中一片轰动。
<br/>　　随后，苏莫立刻召集了苍穹神宫的一众长老议事，详细的询问了一番目前苍穹神宫的发展。
<br/>　　目前的苍穹神宫，已经有了十几位武帝境强者，比如帝释、海明江、钱副阁主等人，纷纷突破到了武帝境界。
<br/>　　不过，众人都只是武帝境一重境界。
<br/>　　宫中除了我们十几位武帝之外，武皇境武者，已经有四万多人，武王境武者不下二十万。
<br/>　　可以说现在的苍穹神宫，比几年之前，强大了不知道多少倍。
<br/>　　而到目前为止，苍穹神宫都没有再招收弟子，人员依旧全部是苍穹世界带出来的人。
<br/>　　这让苏莫非常的欣慰，最后赐予苍穹神宫大量的储物戒。
<br/>　　这些储物戒内，都是他整理的修炼资源，下品灵晶十万，极品灵石上万亿，各种兵器宝物、功法武技，更是多不胜数。
<br/>　　不过，让苏莫稍稍有些意外的是，姜枫然和司空炎两人，目前居然在宫中闭关。
<br/>　　两人出去历练了一年多，半月之前回到了苍穹神宫，均是受了很重的伤势，在闭关疗伤。
<br/>　　这让苏莫有些无语，不知这两人在外招惹了什么人！
<br/>　　结束议事之后，苏莫也没有急着离开，而是准备在途山镇小住几日。
<br/>　　……
<br/>　　一座宽敞的房间之中，苏莫和父亲苏洪相对而坐。
<br/>　　“莫儿，以后有时间，就常回来看看！”苏洪笑着向苏莫说道，他很欣慰儿子如今的成就，但是长年在外漂泊，也是让他这个做父亲的时常挂心。
<br/>　　只不过，他的修为和天赋太低，也帮不上苏莫什么忙！
<br/>　　“父亲，我会的！”
<br/>　　苏莫点了点头，随即沉声说道：“父亲，这次找你来，是想看看能不能为你提升天赋！”
<br/>　　“提升天赋？”苏洪闻言一怔，天赋还能提升？如何提升？
<br/>　　“嗯，若是能给你提升天赋，将来晋升武帝乃至武尊，也不在话下！”苏莫笑道。
<br/>　　目前的苏洪，修为只有真罡境八重境界，连真玄境都不到。
<br/>　　这是因为天赋太低，就算有再多的资源宝物，都没有多少用处。
<br/>　　正常情况下，别说晋升武帝或武尊，就算达到武王之境，都是千难万难。
<br/>　　“这么厉害？”苏洪闻言震惊了，若真是如此的话，那真的是天大的好事。
<br/>　　天赋，一直是他的心病。
<br/>　　虽然因为是苏莫的父亲，他在苍穹神宫地位很高，所有人都很尊重他。
<br/>　　但是他知道，这一切都是沾了儿子的光，不然他屁都不是。
<br/>　　“目前，也只是有可能，具体是否可行，还需要一试才知道！”苏莫叹了口气说道。
<br/>　　他要将自己吞噬的战魂，挑一个与父亲融合，但是能不能成功，他自己也没有把握。
<br/>　　若是不成功，他也没有办法，而若是成功了，他就能让苍穹神宫拥有无数超级天才。
<br/>　　尤其是司空炎等九宫之主，让他们全部变成双生战魂、甚至是三生战魂的天才。
<br/>　　这是苏莫超级夸张的设想，或者说是逆天的计划。
<br/>　　苏洪闻言脸色一暗，原来儿子只是要试验一番，具体还不知道能不能成功。
<br/>　　“莫儿，现在还等什么？马上开始吧！”
<br/>　　少倾，苏洪深吸了口气，不管能不能成功，他都要一试才行。
<br/>　　“嗯！”苏莫点了点头，面色凝重了下来，随即轻声道：“父亲，你闭上眼睛，不要睁开！”
<br/>　　“好！”苏洪立刻便闭上了双眼。
