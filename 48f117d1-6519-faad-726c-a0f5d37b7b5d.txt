　　“轰——”
<br/>　　两拳对撞，万劫魔尊与之前一样，站在原地没动丝毫，共工也与之前的祝融一般，被轰出数丈之远。
<br/>　　“共工，你平时不是很能吹吗？你也一样不行啊。”祝融毫不放过此次机会，打击道。
<br/>　　“你——”共工不服气，他想说自己还没有尽全力，但话到了口边，他就说不出了，因为他知道，眼前此人一样没有出全力。
<br/>　　若是对方出全力，他能否挡得住，他也不知道。
<br/>　　“阁下很强，我来！”十二祖巫之首的帝江对着万劫魔尊道了一声，神色间闪过一丝兴奋，眼前这人竟然能够以肉身之力与他们祖巫对拼，真是让人意外又惊喜啊。
<br/>　　“爆空拳！”
<br/>　　帝江乃空间祖巫，身上天上具有空间法则，他这一拳，便使周围数十丈的空间震动起来，空间的震动频率叠加，汇聚成一拳，一拳空出，不是一人在战斗，而是周围的空间全部在战斗。
<br/>　　万劫魔尊周围的空间变得支离破碎，仿佛下一秒，便会被击碎。他只觉得自己周山三丈范围内，空间被凝固，一个足以爆破空间的拳头在接近。
<br/>　　他不再犹豫，抬起手来，一指点出，这一指，汇聚了他全部的肉身力量，但他依旧没有用上法力。
<br/>　　以点破面！力之极致，万法可破！
<br/>　　指尖与拳头的相撞，并没有想象之中的被击飞，帝江的这一拳，透过层层空间，击中的不是万劫魔尊的手指，而是他的身体，同样，魔尊的这一指一样不是击中他的拳头，而是绕过空间，击在帝江的胸膛之上。
<br/>　　空间法则，万劫魔尊一样会。
<br/>　　万劫魔尊的身体只是动了动，他运用周围的空间将帝江轰来的巨大力量尽量卸去，他实际承受的力量绝不是帝江的全部力量。而帝江却要承受他的一指之力。他的胸膛之上，留下来一个深深的白印。
<br/>　　显然，帝江不是没有受伤，魔尊的强大指力已经透过他的体表，进入了他的体内进行破坏。
<br/>　　“好了，还要继续吗？”万劫魔尊收回双手，站在原地，淡淡道。
<br/>　　“大哥，我来。”烛九阴站在一旁道。
<br/>　　“不必了。”帝江摆了摆手，道，“你已经证明了你的实力，我们可以谈谈如何对付妖族之事。”
<br/>　　话虽这么说，但帝江依旧没有将盘古殿的守护大阵打开，引魔尊进去，显然，他依旧是不完全相信万劫魔尊。
<br/>　　万劫魔尊微微一笑，毫不在意，他相信，换作是他，他一样不会将一个来历不明而且刚刚打过一架的人引进自己家中。
<br/>　　“若是本尊没有猜错的话，尔等之前是在为妖族立天庭之事烦恼。”万劫魔尊道。
<br/>　　“那又如何？”帝江脸色不变，淡淡道。只要是个脑子不笨的人，谁都能猜得出来。
<br/>　　“本尊此次前来，与诸位还有一个交易要做。但，在做交易之前，本尊不介意先给尔等一个意见。”万劫魔尊道。
<br/>　　“是何交易？”烛九阴问道。
<br/>　　“交易慢说不迟，本尊有一言，诸位可以采纳一番。那妖族立天庭，天降功德，可见天命所归，不可阻挡，你们就算想阻拦，也阻挡不了。巫族乃盘古血脉后裔，这洪荒世界都是盘古开辟的，你们巫族自然也有资格执掌洪荒大地。正好此时妖族大部分势力都搬到了三十三天去了，巫族可趁此机会，祭祀盘古，宣告洪荒，立族掌洪荒大地。”
<br/>　　万劫魔尊如此说，也不是要害巫族，就算他不说，巫族也迟早会这么做的，这是大势，不可阻挡。但如今说出来，也只是为巫族多争取一些利益而已。若是晚了，妖族事情忙完了，一样可以回来侵占洪荒大地，到时候，争斗自然是少不了一番，有争斗自然就会有伤害，那就意味着有了损失。
<br/>　　“嗯？道友此话言之有理。哈哈哈，好，就如此办。”帝江、祝融等十二祖巫闻言，纷纷一愣，觉得万劫魔尊所言有些道理，他们虽然不修元神，不是特别懂功德、气数什么的，但也看到了天庭的好处。
<br/>　　本来正在发愁，而今万劫魔尊的提议，顿时让十二祖巫茅塞顿开！于是乎，众人一时都心动不已。
<br/>　　“好了，接下来，我们可以来谈谈交易了。”万劫魔尊打断了众人的心思，淡淡道。
<br/>　　“你说吧，只要不过分，我们都可以答应。”帝江道。
<br/>　　“本尊这里有可以让祖巫突破巫神的功法……”
<br/>　　“什么？巫神是什么境界？”万劫魔尊还没有说完，便被祝融打断了。
<br/>　　“巫神者，乃祖巫之后境界，与混元大罗金仙同等级。”他这么一说，众人便全部明白了，祖巫突破混元，这可是大事啊，想他们祖巫要停在一个境界不知多久。只因他们一直没有元神，不修大道，想要突破混元，此生怕是无望了，如今听到希望，怎能不激动？
<br/>　　“道友此话当真？要知道，对此，就连道祖都没有办法。”帝江等人激动道。
<br/>　　“道祖有没有办法本尊不知，但本尊却是有办法让尔等突破。”
<br/>　　万劫魔尊淡淡道。他知道，道祖不是没有办法，而是不想让祖巫突破，徒增变数，因为这对于他要掌控天道来说，会变得更加麻烦。
<br/>　　如今魔尊这样说自然不假，之前元始的肉体便已经突破到了混元，现在让一个祖巫的肉身突破混元，也不是没有办法。当然，就算祖巫的肉体突破到了混元，依旧不算真正的巫神，真正的巫神，还需要将道行提升到混元，道行不够，依旧不算巫神。
<br/>　　那又要如何突破巫神？这就需要靠他们不断开发自身血脉了。
<br/>　　血脉，何为血脉？不是什么人都拥有血脉的，只有一个种族的祖先的道行达到一定的境界，才能在自身血液中留下大道，那时他的后裔才可能拥有血脉。
<br/>　　他知道，一个种族所蕴含的大道，往往存在一个个体内的血脉之中，区别的，只是大道的多与少，也就是血脉的纯与杂。
<br/>　　一个种族，有传说中的神级血脉，王级血脉，这些血脉，蕴含的大道最为丰富而纯粹，拥有这种血脉，往往是家族中的天之骄子，受无数人重视。
<br/>　　也有一品血脉，不入流血脉，拥有这样的血脉，往往都是普通大众，寻常到不能再寻常。
<br/>　　十二祖巫，作为盘古的第一代后裔，血脉自然不用多说。肯定是最为顶尖的血脉，洪荒之中，没有什么比他们更高贵的血脉了。他们血脉内蕴含的大道，不知有多纯粹、多精妙，只是他们平时都注重修炼肉体，而盘古留下最为宝贵的东西，却被他们忽略了。
<br/>　　虽然说体内拥有强大的血脉，超脱最是困难。因为只有打破他们的祖先血脉的锢制，才能超脱。但他们是谁？他们可是盘古的血脉后裔啊，盘古可是半步大道境的大神。他们不需要超脱盘古，只需要多挖掘一些血脉之中的大道，就足够他们用的了。
<br/>　　可惜，他们空守宝山而不知。正是愚蠢！
<br/>　　十二祖巫激动了好一阵子，才停下来，问道：“道友想要交易什么？”
<br/>　　他们自是知道，这样的功法，需要的代价也肯定不少，他们已经做好被宰的准备了。只要此人的要求不过分，他们都可以答应。
<br/>　　“本尊的要价很便宜，三滴祖巫精血，交易突破巫神境界的法门。”
<br/>　　……
