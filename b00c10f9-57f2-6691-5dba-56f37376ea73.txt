        一时间，童筱颖无言以对。
<br/>
<br/>        落以柔收回目光，甩手离去。
<br/>
<br/>        待在这个大得身处宫殿般的房间里，童筱颖有些无所事事，索性打量起房间里的布局。
<br/>
<br/>        浴室、洗漱间、卫生间、换衣间、书房、阳台、小客厅，全都配齐了，除了没有厨房以外，这哪像卧室，简直就是一个两百多平方的户型。
<br/>
<br/>        忽然窗外传来汽车的鸣笛声，童筱颖循声往阳台上走去，顿时被楼下的一幕怔住。
<br/>
<br/>        十两加长型豪车队伍缓缓从大门口驶入，绕着中央的喷泉，在铺着红地毯的台阶前停下。
<br/>
<br/>        所以说，这才是大门！
<br/>
<br/>        刚刚落以柔带她走进来的时候，是经过了花园和人工湖的，那里岂不是后门。
<br/>
<br/>        而此时，傅夜沉一袭西装革履，带着身着晚礼服的落以柔和郑忠威，还有一路黑发黄皮肤的团队，从台阶上走下来，毕恭毕敬地等候。
<br/>
<br/>        为首的那辆加长型豪车停下来后，一个身着白色鱼尾裙的女人，在侍从的拥护下，从车内钻了出来。
<br/>
<br/>        女人头挽栗色的发髻，别着银色的小皇冠，像公主一样美丽。
<br/>
<br/>        一阵微风拂过，吹动了那女人肩上的雪纱，飘逸着清纯，而晚礼裙勾勒出她那令人喷血的好身材，又有极致的妩媚。
<br/>
<br/>        她来至傅夜沉的跟前，主动向傅夜沉伸出了手。
<br/>
<br/>        傅夜沉则很自然地微微欠身，抬起手来，托住她的那只手，带着她一起上了台阶。
<br/>
<br/>        到此刻，童筱颖终于明白落以柔那话中的另外一层含义了。
<br/>
<br/>        晚饭的时候，菲佣确实把晚餐给端到了房间里来，但是童筱颖很没胃口吃饭。
<br/>
<br/>        真正让她见识自己与傅夜沉的差距在哪，是她没有听取落以柔的警告，离开房间，站在旋梯上俯视楼下正在开上流舞会时所看到的一切。
<br/>
<br/>        他们所有的人都会跳交际舞，就连落以柔也会。
<br/>
<br/>        只是，当傅夜沉与起初那个宛若公主般高贵的女人一起跳完一曲华尔兹，赢得了所有人的掌声后，童筱颖再次感觉到，她和傅夜沉真的是两个不同世界的人。
<br/>
<br/>        他是游刃在奢华与豪门里的贵族，而她只不过是大山里走出来的平凡。
<br/>
<br/>        傅夜沉和那个女人跳舞的时候，就像童话里的王子与公主，如果还能再升级，他们必定是国王与王后。
<br/>
<br/>        她不会跳交际舞，也不懂时尚，但她会种田、会种菜、会下厨、会设计房子、也会自己造房子，所以她与这里的周遭显得格格不入。
<br/>
<br/>        那么，她来这里是为了什么？
<br/>
<br/>        见他。
<br/>
<br/>        嗯，对。
<br/>
<br/>        现在已经见到他了。
<br/>
<br/>        其他的任何事情，好像显得已经没有那么重要了。
<br/>
<br/>        童筱颖转过身去，黯然神伤地回到了房间，重新提起了自己的行李箱。
<br/>
<br/>        离开这里，是她现在最好的选择。
<br/>
<br/>        她是从哪里进来的，就是从哪里出去的。
<br/>
<br/>        一个人孤零零地走在蜿蜒的公路上，行李箱的滚轮碾过地面嘎吱作响。
<br/>
<br/>        她好不容易走到一个小镇上，结果两个骑着摩托车戴着头盔的男子，“嗖”地一声，将她的行李箱给抢走了。
<br/>
<br/>        因为她的手当时还紧紧地拉着行李箱的拉杆，随着行李箱被抢走的惯性，她打了个踉跄摔倒在地。
<br/>
<br/>        双膝和手掌心，瞬间被粗糙的水泥地面擦出了大片的血迹。
<br/>
<br/>        所有的证件以及手机，都在行李箱里。
<br/>
<br/>        童筱颖顿感无奈，只能去了镇上的警局报警，结果因为语言不通，还被警局里的警察怀疑是偷渡者。
<br/>
<br/>        在这种暗无天日，狼狈到十分落魄的情况下，她不得不把傅夜沉的手机号码写给了警察。
<br/>
<br/>        大概半个小时后，傅夜沉依旧是那身帅气的西装革履，出现在了警局里，他用一口流利的法文和警察交流，又是签字又是放押金的走完流程后，才转身看了一眼坐在墙角的童筱颖。
<br/>
<br/>        看到她双膝上血肉模糊，聋拉着脑袋，他剑眉不悦地微微蹙起。
<br/>
<br/>        “你可以走了！”他将目光从她身上收回，淡漠地说道。
<br/>
<br/>        童筱颖怔愣地抬起头来看向他的时候，他已经转身往警局门口走去了。
<br/>
<br/>        她连忙起身，忍着膝盖上的疼痛，跟在了他的身后。
<br/>
<br/>        很难得，她这次没有哭，其实是想哭的那种，只是哭不出来而已。
<br/>
<br/>        “既然来了，就好好地待在房间里不行吗？你非得一声不吭地跑出来跟我闹？一个多月的时间，还不够你冷静吗？童筱颖，你到底想我怎样？要告我的人是你，要我离你越远越好的人也是你，你现在又跑到我面前来惹这样的麻烦，你是想我怎么做，你才满意？”傅夜沉忽然停下脚步，转过身来，疾言厉色地质问。
