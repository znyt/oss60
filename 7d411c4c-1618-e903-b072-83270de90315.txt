过了一会儿，里面传来他低沉磁性的嗓音，“进。”
<br/>
<br/>        夏夕绾推门而入，走过去将手里的咖啡放在了他的办公桌上。
<br/>
<br/>        陆寒霆拿着钢笔的大手顿住了，他那双狭眸扫了一眼咖啡，然后抬头看着她，“这是什么？”
<br/>
<br/>        “咖啡，如果你工作实在繁忙需要熬夜的话，还是喝杯咖啡吧，不要抽烟了，抽烟有害健康，你已经抽了很多了。”夏夕绾道。
<br/>
<br/>        陆寒霆丢下了手里的钢笔，将挺括的后背慵懒的抵靠进了椅座里，“呵”，他勾起薄唇，从喉头里溢出一声低笑，“我是问你现在又想干什么，你每天这样周旋在不同的男人身边，取悦不同的男人，一点不累么，这样是不是很能满足你的虚荣感？”
<br/>
<br/>        他说什么？
<br/>
<br/>        夏夕绾纤长的羽捷颤了颤，她好心给他煮了一杯咖啡，他怎么像吃了火药一样，又是一顿羞辱她？
<br/>
<br/>        “你现在心情不好，我不跟你计较，你工作吧，我出去了。”夏夕绾转身就走。
<br/>
<br/>        陆寒霆伸手，直接将办公桌上那杯咖啡给拂落掉了。
<br/>
<br/>        嘶。
<br/>
<br/>        滚烫的咖啡溅到了夏夕绾纤白的小手上，她白皙的手背当即被烫红了一块。
<br/>
<br/>        陆寒霆当然也看到了，他不是故意的，只是受不了她又来撩他，刚才他就站在阳台那里看着，她刚下了陆子羡的车就进了他的书房。
<br/>
<br/>        现在见她的手被烫红了，他心头一紧，心脏像是被一只大手给狠狠拽住了，很疼很疼，每一下呼吸都是疼的。
<br/>
<br/>        男人的尊严让他没有办法现在站起身去查看她的烫伤，陆寒霆伸出大手，一把拽住了她纤细的皓腕，将她用力的扯了过来。
<br/>
<br/>        夏夕绾直接跌坐在了他结实的大腿上。
<br/>
<br/>        陆寒霆捏着她小巧的下颌，大概因为疼，她白皙的小鼻翼都变得红红的了，看着楚楚可怜，他抿了一下薄唇，沉声训斥道，“把眼泪收回去，我不吃这一套，疼就给我忍着！”
<br/>
<br/>        夏夕绾澄亮的翦瞳里覆着一层水光，将烫红的小手藏在身后，她仰着巴掌大的绝色小脸望着他，“你，就这么讨厌我吗？”
<br/>
<br/>        陆寒霆一僵，她是这么认为的吗，认为他讨厌她？
<br/>
<br/>        他也不知道，每一次见到她，他都会生气，像是以前她对他做过什么不好的事情，大概类似…抛弃过他，他对她总是很矛盾，明明被她吸引，心里又有点恨她，觉得她不是什么好女孩。
<br/>
<br/>        所以他总是能轻易的点燃，只要她跟其他男孩走得近了，他就想刺痛她。
<br/>
<br/>        但是现在看着她眼里的水光，他又心疼到不行，话出口了就后悔了。
<br/>
<br/>        这种感觉几乎让他抓狂。
<br/>
<br/>        他此刻的沉默在夏夕绾眼里就像是默认，贝齿咬了一下红唇，她小声道，“虽然我不知道你为什么这么讨厌我，但…对不起，我以后会离你远远的，你也没有必要每次看到我都羞辱我，我虽然嘴上没说，但是心里也疼了。”
<br/>
<br/>        说着夏夕绾动了动，想从他怀里起身，“你放开我，我想要回房间休息了。”
<br/>
<br/>        陆寒霆突然将一份文件拿出来，丢在了办公桌上，“签了它。”
