“我陆寒霆在你眼里是什么人恩？我自恃眼光甚高，不是什么女人都会看上，除非那个女孩跟你一样生了一张清丽绝色小脸，还有你的聪慧坚韧和勇敢…只可惜你是这个世界上独一无二的，除了你，再也不会有人能让我这般的心动了。”
<br/>
<br/>        夏夕绾觉得他还挺会说甜言蜜语的，她伸手去推他，“走开，不要压着我，你好重。”
<br/>
<br/>        陆寒霆看了她一眼，将她给松开。
<br/>
<br/>        夏夕绾爬回了自己的被子里，抬着小巧的下颌拿目光看他，“这么说来你跟你那个小新娘只见过一面了，她喜欢你么？”
<br/>
<br/>        陆寒霆想了一下，“当时她拽住了我的手指。”
<br/>
<br/>        “怎么拽的啊…”夏夕绾伸出小手缓缓的将他修长的食指给拽在了手心里，“这样拽的啊…”
<br/>
<br/>        陆寒霆深邃的瞳仁微微收缩了一下，他的大脑里迅速浮现起了多年前的画面，那时他才七岁，站在摇篮边看了一眼那个小女孩。
<br/>
<br/>        一丁点大的小婴孩刚出生没多久，一点都不像其他孩子那样皱巴巴的，她已经粉雕玉琢了。
<br/>
<br/>        当时她躺在鹅黄色的毛毯里，睁着一双黑漉漉的大眼睛望着他，小手在空中乱舞，一把拽住了他修长的食指，还咧开了小嘴对着他笑。
<br/>
<br/>        奶奶的，香香的，甜甜的，能将人的心都给看化。
<br/>
<br/>        当时母亲打趣他，“看，我家阿霆的小新娘很喜欢阿霆哦。”
<br/>
<br/>        陆寒霆看着夏夕绾这双顾盼流转的水眸，她的模样突然就和当年的那个小女孩给重合在了一起。
<br/>
<br/>        陆寒霆低眸就往她的红唇上亲去。
<br/>
<br/>        但是夏夕绾避开了，她快速的松开了他的食指，将蚕丝被裹在自己的身上，还骂了他一句，“陆先生，你色火攻心的样子真可怕！”
<br/>
<br/>        “…”
<br/>
<br/>        陆寒霆上下滚动着喉结，平稳着自己的呼吸，然后他伸手将夏夕绾搂抱在了自己的怀里，“睡觉！”
<br/>
<br/>        夏夕绾在他精硕的胸膛里仰起小脸，闹归闹，但是今晚的收获还是不错的，他开口跟她提了自己的家庭。
<br/>
<br/>        “陆先生，你母亲是…怎么去世的啊？”
<br/>
<br/>        夏夕绾可以感觉到，他很爱自己的母亲。
<br/>
<br/>        陆寒霆闭上眼，一张俊脸波澜不惊，只是淡淡的掀了掀薄唇，“我母亲后来又怀孕了，我父亲不让她生，给她灌了堕胎药，那时她怀孕七月，自己拿刀亲手剖开了自己的肚子，把孩子取了出来，以为七月的孩子可以养活的，可是孩子浑身乌青，已经死了，那是一个妹妹，没过多久，我母亲就自杀了，她从阿娇房的高台上一袭红衣跳了江，尸身一直没有找到。”
<br/>
<br/>        夏夕绾一僵，她被这个故事给震撼到了，久久的都没能回神。
<br/>
<br/>        她不知道一个女人要有多大的勇气，被逼到怎样的绝境里才会拿起刀亲手剖开自己的肚子。
<br/>
<br/>        这时陆寒霆翻了一个身，将她纤柔而温暖的身体揉进自己的怀里，喃喃道，“绾绾，这一次是你自己送上门的，以后我绝不会再放开你的手，绝不会再让你走。”
