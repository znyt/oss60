        她是从这里走出去的，在聒噪的大城里走了一遭，最后还是回到了这里。
<br/>
<br/>        外婆，你的筱颍，回来了，你开心吗？
<br/>
<br/>        此时此刻的童筱颖，已经心如止水。
<br/>
<br/>        翌日，滨江城，傅氏集团大厦。
<br/>
<br/>        落以柔把一杯刚泡好的红茶放到了傅夜沉的桌上。
<br/>
<br/>        傅夜沉一向准时在七点五十五分进办公室，西装外套随手一挂之后，就会卷起白衬衫的袖子，在老板椅内坐定，喝上一口红茶。
<br/>
<br/>        八点整，他会开始工作。
<br/>
<br/>        自从得知傅夜沉的身份后，落以柔不再向以前那样对傅夜沉随心所欲了。
<br/>
<br/>        该收敛的时候，她还是会收敛的。
<br/>
<br/>        经过连华生的一番调教，从品酒到交际舞，从随机应变到临时发挥，才两个月还不到的时间，落以柔成了傅夜沉的“万能”秘书。
<br/>
<br/>        有些事情，郑忠威做不来的，这位“万能”女秘书，可以做得游刃有余。
<br/>
<br/>        “中午和‘兴实’的张总约在老地方吃饭。”落以柔拿着傅夜沉给她配的小号平板，一丝不苟地向傅夜沉报告着他今天这一天的行程。
<br/>
<br/>        傅夜沉淡定地问：“张总那个项目的合约情况？”
<br/>
<br/>        “张总说，他会令你满意。”落以柔咧嘴一笑，这可是她把嘴皮子说破了，才让张总答应那些傅夜沉提出的条件的功劳。
<br/>
<br/>        “‘瑞高’的尾款催得如何？”
<br/>
<br/>        “会计部说他们最后的百分之十会在年底前付清，不开票，现金入帐。”
<br/>
<br/>        “年底前他们若是失信，就停了他们的整栋大厦的所有电梯。”一讲起正事，傅夜沉是绝对的一板一眼。
<br/>
<br/>        落以柔干笑着应了声：“了解。”
<br/>
<br/>        瑞高的高管都住在四十八楼，这电梯一停，他们若不是支付尾款，那就得天天爬楼梯了。
<br/>
<br/>        而她家这位傅大总裁，其实一点都不缺他们那点尾款，他只不过是不喜欢做生意时那些不讲诚信的人罢了！
<br/>
<br/>        其实了，是该给那些不讲诚信的买家一些教训。
<br/>
<br/>        “我晚上是不是还有个饭局？”
<br/>
<br/>        “傅老太爷安排了市长的千金与你相亲。”
<br/>
<br/>        在师父郑忠威请假的这段日子里，落以柔才当了他两个月不到的秘书，就成了他的左右手。
<br/>
<br/>        傅夜沉露出了一个不耐、冷肃的眼神。
<br/>
<br/>        “帮我找个理由。”
<br/>
<br/>        “大哥，你不想去吗？”闻言，落以柔的心中有了一丝窃喜，但她平静内敛的脸上什么都没有表现出来。
<br/>
<br/>        私底下的时候，她仍旧会唤他大哥。
<br/>
<br/>        这个称呼，她叫习惯了，他似乎也听习惯了。
<br/>
<br/>        “……”傅夜沉只是冷冷地扫了落以柔一眼。
<br/>
<br/>        落以柔立即识趣地回答道：“那我告诉傅老太爷……你突然得去云城视察分公司的工作？”
<br/>
<br/>        “可以。”
<br/>
<br/>        “就说你得在云城待一晚，明天才会回滨江城？”落以柔驾轻就熟地说着细节。
<br/>
<br/>        “嗯。”傅夜沉也同意了。
<br/>
<br/>        “然后，说你这个月都很忙，必要时……传真一份‘加油添醋’的行程表给傅老太爷？”这种法子，是师父郑忠威教给她落以柔的。
<br/>
<br/>        看样子，在这之前，傅夜沉让郑忠威做过不少类似的事情。
<br/>
<br/>        不知道为什么，傅夜沉越是拒绝那些五花八门的红线，她落以柔的心里就越开心。
<br/>
<br/>        “好。”傅夜沉淡淡地应了声。
<br/>
<br/>        “那如果，傅老太爷派人带那位千金大小姐来办公室里找你，该怎么办？”落以柔连这事都想到了，就是为了以防万一。
<br/>
<br/>        “随便你用什么方式拦下这件事情。”傅夜沉云淡风轻地说。
<br/>
<br/>        落以柔扬了扬柳眉，忍不住俏皮地打趣道：“要不，当那位千金大小姐找上门来的时候，大哥你就‘桌咚’我，或者‘沙发咚’，反正只要能让那位千金大小姐误会，知难而退，怎么‘咚’我都行！总裁与秘书嘛！不整点桃色花边新闻出来，还真就对不住外界那些流言蜚语。大哥，你说对不对？”
<br/>
<br/>        “你很闲？”傅夜沉那一双原本能迷倒任何女性的黑眸，此时闪着冷光。
<br/>
<br/>        “好吧！我会想其他办法，把这事给拦下的！”自知玩笑稍微过火，落以柔连忙一本正经地说。
<br/>
<br/>        总而言之，她确实替他想了办法，找了各种借口，不去见那位市长的千金，也不让那位市长千金来见他。
<br/>
<br/>        这日，知道傅夜沉正在接一通私人电话，所以落以柔完成了分内工作后准备走出他的办公室，但他却用手势要她留下，所以她只好站在他的办公桌前等着。
<br/>
<br/>        “小沉，爷爷不希望你一直吊死在童筱颖那一棵大树上！”是爷爷傅炎廷的私人电话。
