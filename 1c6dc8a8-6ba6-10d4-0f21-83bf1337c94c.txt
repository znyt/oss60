        “沫、筱颍……”傅昼景眸光晃动，低沉的声音有些哽咽，“真的是你吗？你真的还活着吗？”
<br/>
<br/>        童筱颖眨巴着眼睛看着这个男人，正思忖着这个男人是不是认识自己的时候，这男人便疾步上前，抓住她的手腕，将她拉到他跟前，抱了个满怀。
<br/>
<br/>        傅昼景将童筱颖抱得很紧，紧到让她有些喘不过气来。
<br/>
<br/>        “先生……”童筱颖使出浑身解数，将傅昼景推开后，一脸厌恶地呵斥，“请自重！”
<br/>
<br/>        “筱颍，我是小卫，你……”傅昼景欲言又止，对上童筱颖的眸子时，他从她的眼中看到的全是陌生的眼神。
<br/>
<br/>        “对不起啊！我真不认识你！”童筱颖敛了敛脾气，但仍旧没好气地说。
<br/>
<br/>        “筱颍，你……失忆了？”傅昼景迎上前，双手紧紧地抓住童筱颖的肩头，难以置信地问，“你把我忘了？”
<br/>
<br/>        童筱颖挪开傅昼景的手，后退了一步，与傅昼景保持着一段距离：“先生，我真不认识你。我还有事，先走了。”
<br/>
<br/>        “筱颍，你再仔细看看我，我是小卫。我们俩自幼相识，我们曾经还是夫妻。我们……”傅昼景再次捉住童筱颖的手腕，将她拉到自己跟前时，欲要说的那些话，却再也无法说下去了。
<br/>
<br/>        童筱颖这才仔细打量起面前的这个男人，他跟傅夜沉长得有几分神似，但是又没有傅夜沉身上那种独特的气质。
<br/>
<br/>        不过，话说回来……
<br/>
<br/>        她跟他曾经是夫妻？！
<br/>
<br/>        是怎么一回事？
<br/>
<br/>        童筱颖的大脑里顿时有些混乱。
<br/>
<br/>        “傅昼景啊傅昼景，我就知道你一直对童筱颖余情未了。你这样纠缠着她，就不怕你家那位正宫娘娘跟你闹吗？”
<br/>
<br/>        就在两人僵持着的时候，苏漫雪的声音从两人身后传来。
<br/>
<br/>        童筱颖和傅昼景同时循声望去，只见苏漫雪正将手中的手机塞入手提包中，踏着高跟鞋，步态妖娆地向他们走了过来。
<br/>
<br/>        苏漫雪走到童筱颖的身旁，手腕上挂着包包，双手抱臂地咧嘴道：“既然你失忆了，我就帮你回忆一下吧！面前这个男人是个已婚男人，而你了，是这个男人包养的小三。”
<br/>
<br/>        “苏漫雪，你胡说八道些什么？”傅昼景呵斥道。
<br/>
<br/>        童筱颖一脸懵。
<br/>
<br/>        “你跟这个男人在一起的时候，和这个男人上过无数次床，各种姿势，全都试过。这男人的床技相当好，可以把你在床上制服得舒舒服服。但是了，这个男人很渣，搞大了你的肚子，又不想负责，索性派人逼着你去医院里堕胎，还说是为了你好。”苏漫雪愤愤不满地看着傅昼景，滔滔不绝起来。
<br/>
<br/>        傅昼景疾言厉色地呵斥道：“苏漫雪，你够了！不要在筱颍面前乱说话！”
<br/>
<br/>        童筱颖看着他俩一人一句，忍不住插上话，问道：“你们所说的这些，跟我又有什么关系？我都不认识你们俩。”
<br/>
<br/>        “怎么会没关系呢？他搞大了你的肚子，导致你无法怀孕，你难道不恨他吗？”苏漫雪凑到童筱颖的面前，勾唇一笑。
<br/>
<br/>        傅昼景放开了童筱颖的手，一阵恼火地抓住苏漫雪的手腕，拉着她向另一边的走廊拐角走去。
<br/>
<br/>        安全通道里，傅昼景将苏漫雪甩到了墙壁上，怒吼：“苏漫雪，你到底想怎样？”
<br/>
<br/>        苏漫雪揉了揉被撞疼的肩膀，呵呵地笑了笑，反问：“你觉得我想怎么样？当然是小三上位喽！”
<br/>
<br/>        刚刚她所说的那些话，其实是她苏漫雪和他傅昼景之间的私事。
<br/>
<br/>        他害得她差点就无法生育，她能不恨这个男人吗？
<br/>
<br/>        但是，恨归恨，她还想要嫁给这个男人。
<br/>
<br/>        苏漫雪也不知道自己这是怎么了。
<br/>
<br/>        或许，是因为经常跟他上床，上出感情来了。
<br/>
<br/>        她知道傅昼景跟季思妍没有领结婚证，只是他们是公开的夫妻身份，所以，苏漫雪一直希望傅昼景再对外公开他和季思妍已经没感情了，婚姻破裂，离婚了。
<br/>
<br/>        这样，她才能名正言顺地跟他在一起。

<br/>        “我们彼此只是床伴关系而已，当初，这句话可是你说的。”傅昼景嗤之以鼻。
<br/>
<br/>        苏漫雪不以为然地笑了笑：“怎么？发现童筱颖还活着，就这么急着想要跟我撇清关系吗？你是不是觉得，她把你忘了，就一定也把你那位大哥也忘了？你知不知，童筱颖今天是跟谁一起来参加晚宴的？”
<br/>
<br/>        “她跟谁来的？”傅昼景剑眉微蹙，下意识地反问。
<br/>
<br/>        苏漫雪扬了扬嘴角，回答道：“你大妈，傅夜沉的母亲。而且，童筱颖可是当众唤她‘妈妈’，你说，你和童筱颖还有可能吗？”
