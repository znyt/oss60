        明护国这才默认了明司寒入赘的事情，只是叹息道：“既然如此，我们明家就退让一步便是。你入赘，以后爷爷会补偿你的。”
<br/>
<br/>        “爷爷，你就别生气了。气坏了身子可不好。”明司寒苦笑。
<br/>
<br/>        明护国叹了口气，便将电话给挂断了。
<br/>
<br/>        明司寒看着手机上的通话结束，无奈地摇了摇头。
<br/>
<br/>        就在他正准备收起手机时，背后一阵凉风袭来，肩上有重物一压，两眼便被一双手给蒙住了。
<br/>
<br/>        “猜猜我是谁？”一个女生的声音在脑后响起。
<br/>
<br/>        明司寒不禁皱起眉头，伸手抓住女生的手腕，将她从自己的后背上扯了下来。
<br/>
<br/>        “贺思思，你有完没完？”
<br/>
<br/>        明司寒的粗鲁，惹得贺思思手腕上传来硬生生的痛楚。
<br/>
<br/>        贺思思瘪着嘴，两眼泪汪汪地看着明司寒，吸了吸鼻子后，委屈道：“人家从北城千里迢迢赶来临海城看你，你还对人家这么凶！”
<br/>
<br/>        “现在看完了吗？”明司寒却仍旧一脸不耐烦地问道。
<br/>
<br/>        贺思思微微点了点头。
<br/>
<br/>        明司寒见她身边立着行李箱，于是伸手抓住了她的行李箱拉杆，直接拖着她的行李箱往校门口走去。
<br/>
<br/>        “司寒哥哥，你要带我去哪里呀？”贺思思见状，破涕为笑地小跑了起来，跟上去问道。
<br/>
<br/>        明司寒看都懒得看贺思思一眼，冷冷地回答道：“送你去车站，你给我滚回北城去！”
<br/>
<br/>        “不要！”贺思思立马停下脚步，伸手抓住明司寒的手臂，嘴角又往下瘪出一个委屈的弧度。
<br/>
<br/>        明司寒索性停下了步伐，拧着眉头看着贺思思，挪开她的手后，质问道：“你到底要怎么才不会缠着我？”
<br/>
<br/>        “司寒哥哥，你是知道的。我缠着你，是因为我喜欢你啊！”贺思思鼻子一酸，眼泪瞬间在眼眶里打转。
<br/>
<br/>        明司寒一如既往地冷冷回绝：“可是我不喜欢你！”
<br/>
<br/>        “司寒哥哥，你不要这样，好不好？”贺思思可怜兮兮地努起嘴来。
<br/>
<br/>        明司寒却将目光挪向一旁，语气依旧冷漠：“你再缠着我，别怪我翻脸。”
<br/>
<br/>        “我哪里比不上那个连星儿呢？她有的，我都有！她凭什么当你的未婚妻？更何况，她压根就不喜欢你。”贺思思再次抓住明司寒的臂弯，撒泼似的摇晃着明司寒的胳膊。
<br/>
<br/>        “你跟她私底下见过面？”明司寒有些敏感地皱起了眉头。
<br/>
<br/>        贺思思这才意识到自己说漏了嘴，连忙口是心非地敷衍道：“我才没有私底下跟她见过面咧！只是，我们都是女孩子。我第一次见她，就看得出，她压根就不喜欢你。”
<br/>
<br/>        “反正是豪门之间的联姻，喜不喜欢，压根就无所谓。”明司寒淡淡地说道。
<br/>
<br/>        贺思思上前一步，凑到明司寒的跟前，仰着头，十分憧憬地看着明司寒接着说道：“那司寒哥哥让我当你的情人，好不好？”
<br/>
<br/>        “我觉得你疯了。”明司寒再次甩开贺思思的手，拖着她的行李箱一边继续往前走，一边碎碎念叨了一句，“跟你哥哥一样，都是个疯子。”
<br/>
<br/>        “司寒哥哥，我不要你送了。我自己会回去！”贺思思看着明司寒大步走远的背影，立马又追了上去，从他手中夺回了自己的行李箱。
<br/>
<br/>        明司寒瞥了贺思思一眼，冷酷无情地说道：“那你好走，不送。”
<br/>
<br/>        他说完，转身便向校园内走去。
<br/>
<br/>        留下贺思思独自一个人站在校道上，讷讷地目送着他的背影。
<br/>
<br/>        贺思思深吸了口气后，掏出手机给自己的哥哥贺泽川打了通电话：“喂，哥，你派司机到临海大学的正校门口来接我吧！我今晚去你那儿住。”
<br/>
<br/>        不出十分钟，贺思思在临海大学的校门口外坐上了一辆豪华私家车。
<br/>
<br/>        这次开车的人不是贺泽川的司机，而是他贺泽川本人。
<br/>
<br/>        贺思思坐进副驾驶座后，就开始跟哥哥贺泽川抱怨：“司寒哥哥他又凶我了。”
<br/>
<br/>        “你没事跑来临海城做什么？”贺泽川皱着眉头问。
<br/>
<br/>        贺思思没好气地瞥了贺泽川一眼，努嘴道：“我想见见司寒哥哥不行吗？”
<br/>
<br/>        “你就在这么喜欢明司寒？”贺泽川接着说道。
<br/>
<br/>        贺思思连连点头，应道：“已经不是喜欢，是爱到离不开他的地步了。”
<br/>
<br/>        “他已经有未婚妻了，所以，这事情，你就别想了。”贺泽川一边开车，一边好言相劝。
<br/>
<br/>        贺思思却不屑地说道：“那个未婚妻，我见过了。她跟我说，她压根就不喜欢司寒哥哥。”
<br/>
<br/>        话题到这，贺泽川忽然间意识到了一个问题，便没再接贺思思的话。
