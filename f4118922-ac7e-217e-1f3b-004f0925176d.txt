        “这位夏家少小姐，倒是被你驯服得很乖巧啊！”连华生忍不住打趣。
<br/>
<br/>        夏玄灵出门后，裴金澈没心情和连华生开玩笑，而是开门见山地直接坦白：“你爸爸的实验成功了！”
<br/>
<br/>        “我老爸的实验？我老爸的实验成果不就是你吗？一个痛觉神经被麻痹，却拥有细胞快速更新能力的你。”连华生不以为然地咧了咧嘴。
<br/>
<br/>        裴金澈抿了抿唇，深沉道：“在她的身上，我看到了我未来的命运。华生，那个叫童筱颖的女人，她并没有死。”
<br/>
<br/>        “嘶——”连华生听后顿时毛骨悚然，“诈尸呢？”
<br/>
<br/>        “她是所有绝症的解药，我的血和她的血融合后，可以让体内死去的每个细胞重新复活，并且还可以抑制病变细胞的繁殖、扩散和转移。我给她供了大半年的血，这半年里，她体内进行了自我修复。上个月，她已经醒过来了，因为细胞更新，她失去了所有的记忆。刚开始，她还很乖，愿意让他们采血，但是最近她出现了躁动，情绪不大稳定。我无意间偷听到，你爸爸他们在决定把她变成植物人，彻彻底底成为解药的母体。”裴金澈说到这里，不禁握紧了拳头，“而我的下场，也已将跟她一样。”
<br/>
<br/>        “我终于明白，阿沉为什么把童筱颖藏得这么深了。原来，他早就知道，童筱颖存在的意义是什么了。所以，才不停地否决童筱颖是他们要找的那个女孩。”连华生恍若大悟后，又纳闷了起来，“那童筱颖岂不是真正的‘夏玄灵’，如果是这样的话，夏局知道这件事情吗？她可是夏局的亲生女儿啊！”
<br/>
<br/>        “夏叔叔知不知这件事情，我就不清楚了。”裴金澈深沉道。
<br/>
<br/>        连华生瘪着嘴，凝眉沉思。
<br/>
<br/>        这大半年的时间里，他老爸的实验室对他下了禁足令，原来是在背着他搞活物研究。
<br/>
<br/>        “我以前逆来顺受，是因为接受了自己是实验品的命运。但现在不同了，我有玄灵，我想陪她一起白头偕老。童筱颖那个女人，也是阿沉的心头肉吧！你告诉阿沉，童筱颖还活着。我想，他一定会去救童筱颖。”裴金澈接着说道。
<br/>
<br/>        连华生倒是看穿了裴金澈的心思：“与其说是去救童筱颖，你其实是希望阿沉救你吧！”
<br/>
<br/>        “也可以这么说。”裴金澈毫不遮掩地点了点头。
<br/>
<br/>        他虽然替傅夜沉掌管傅氏集团，但是还有郑忠威在监督他，集团里所有的交接事宜，都是郑忠威跟他碰面，他没有机会接触傅夜沉。
<br/>
<br/>        “我帮了你，岂不是搬起石头，砸了我老爸的脚。反正，阿沉以为童筱颖已经死了。而童筱颖那个女人压根就不适合阿沉。更何况，现在的她，可以说完完全全地已经不是以前的那个童筱颖了。我这样的理解，没错吧？阿沉去救一个只是外貌和童筱颖长得一模一样的女人，完全没必要。”连华生振振有词道。
<br/>
<br/>        裴金澈顿时恼火地揪住连华生的衣领：“我告诉你，等你爸爸他们把我和她身上的血都抽干了，接着，你爸爸他们就会卖掉她身上任何一个器官。他们这么做，就是在杀人！童筱颖少了一个肾，她的另一个肾在傅昼景的体内与傅昼景完全融合了。这件事情，你可以去调查，便知道我所说的是不是真的了。连华生，你是医生，你的职责是什么，你应该比我更清楚！”
<br/>
<br/>        “医生的职责是救人，但也有个时候会牺牲一个人去拯救更多人的生命。”连华生一脸淡定。
<br/>
<br/>        裴金澈反驳道：“就算你们要牺牲掉一个人，那也得那个人自愿对不对？我和童筱颖，都不是自愿的！一个星期后，你爸爸就会给童筱颖动手术，让她永远沉睡。我未来的下场也会跟她一样。你是我朋友，我把希望寄托在你身上，所以，你还有一个星期的时间考虑。”
<br/>
<br/>        “……”连华生听完裴金澈的话后，在此刻沉默了。
<br/>
<br/>        夏玄灵买完红糖回来，见他们愣坐在沙发上不吭声，两人之间的气氛也格外僵硬，不禁纳闷道：“你两——这是吵架了吗？”
<br/>
<br/>        连华生瞥向夏玄灵，从沙发上起身，若无其事地回答：“你给他泡红糖水喝，我还给他开了药，一日三次，每次两粒。”
<br/>
<br/>        “阿澈他是怎么了？”夏玄灵关问道。
<br/>
<br/>        “他——”连华生下意识地瞄了裴金澈一眼，只见裴金澈会意地向他使了使眼色，他才改口道，“只是贫血而已，多吃点樱桃、猪肝等补血的食物。”
