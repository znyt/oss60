　　　　他是肉身投影过来的，跟真实的身体没有差别，如果受伤，回到人间，还是一样的伤情，如果死了，那就是真的死了。
<br/>　　叶少阳也不知道自己内脏受伤没有，想来应该没大事，他也血肉之躯，如果真是受伤太重，一样会死的。
<br/>　　叶少阳忍着痛，左手在伤口附近点了几下，先止住血，抬头看去。芮冷玉已经站了起来，直勾勾地看着他。
<br/>　　叶少阳注意到，她的两只眼睛——确切说是两个瞳孔，变成了一种诡异的红色，比鲜血淡一点，有点像是玫瑰色，闪烁着一种幽光。
<br/>　　她的脸庞没有任何表情，完全是一副木然的样子，像一个梦游的人，浑身上下，不断释放出浓烈的煞气。
<br/>　　叶少阳心中一阵绞痛。
<br/>　　人还是那个人，但是这种感觉，却是让他感到无比的陌生，完全不是以前的芮冷玉。
<br/>　　转世鬼童，完全苏醒了。
<br/>　　“冷玉……”叶少阳忍着疼痛呼唤了一声，虽然眼前的芮冷玉完全是个陌生人，但叶少阳相信，她的神智还是存在的！
<br/>　　一定是被煞气填充，一时间迷失了自我，或者就像被通玄道人占据躯体的木子一样，原本的元神和意识，被挤压到了某个角落，暂时没法支配身体。
<br/>　　总之，她还是她！
<br/>　　就算她真的成了转世鬼童，她仍然是自己的未婚妻！
<br/>　　叶少阳深吸一口气，开始呼唤芮冷玉的名字，说各种话，希望能唤回她的记忆。
<br/>　　听到他说话之后，芮冷玉呆板茫然的脸上，第一次出现了表情，皱了皱眉头。
<br/>　　这个发现，让叶少阳心头大喜，更加不断地大声跟她说话。
<br/>　　“小玉，我是少阳啊，你忘了吗，我们本来说好了，处理完这件事，回人间就去结婚的，连婚纱照都照好了，只等你师父来，给我们择个日子，就可以完婚了。对了，你师父，一谷大师，你还记得他吗？”
<br/>　　说这些话的时候，叶少阳自己也等于回忆了一遍，说到自己视线模糊，泪流满面。
<br/>　　芮冷玉脸上迷惑的表情越来越严重，总算是慢慢朝着他走了过来。
<br/>　　叶少阳心里那个激动，举起自己的左手，继续说道：“你还记得吗，你曾经在我手中用神念留下了三个字，让我去猜，猜到了你就会执行。这三个字我后来猜到了啊，是‘嫁给你’……”
<br/>　　芮冷玉已经走到他面前来。
<br/>　　“小玉，你想起我了吗？我是少阳啊！”
<br/>　　叶少阳激动的用手去摸她的脸。
<br/>　　“少阳，叶少阳……”芮冷玉口中喃喃说道。
<br/>　　“是啊，是我啊。”
<br/>　　叶少阳摸着她的脸，咧开嘴，想给她一个笑容，结果芮冷玉也抬起手，却不是给他任何回应，而是拔出了松纹古定剑，对着叶少阳胸口刺了过来。
<br/>　　叶少阳急忙躲闪。
<br/>　　剑锋险而又险地擦着皮肤过去，刮破衣服，留下一道血痕。
<br/>　　芮冷玉又提剑刺来。
<br/>　　叶少阳抽身急退，只好把七星龙泉剑拔出来，且战且退。
<br/>　　“你怎么了，小玉，你不认识我了？”叶少阳急的叫起来。但是芮冷玉脸上的迷惑消失了，只剩下仇恨。
<br/>　　印堂处出现了一颗漩涡般的紫色印记，不断闪着光。
<br/>　　叶少阳挥剑抵挡，一边后退，起初只是想要唤醒她，后来发现她身上散溢而出的煞气越来越浓郁，修为也越来越强。
<br/>　　每一次将她手中的剑荡开，叶少阳感到越来越吃力，虽然自己还能应付，但是想要擒住她，却是越来越难了。
<br/>　　叶少阳内心深处，涌现出了一种深深的无力感，甚至有一点绝望。
<br/>　　他没有想过放弃芮冷玉，但是目前这个局面，他真的有点不知道怎么办好了。
<br/>　　怎么会……变成这个样子？
<br/>　　“鬼童归位！”
<br/>　　就在叶少阳不知所措的时候，突然一声高呼，从树林里传来。
<br/>　　叶少阳循声望去。
<br/>　　一辆看上去像是马车的通体漆黑的车辆，从树林里移动出来，前面自然没有马，车厢也是空的，上面有一道黑色的华盖。下面坐着一个女子，身穿一袭紫色上面有水波纹饰的长袍，嘴唇涂成紫色，连眼睑也涂成紫色，整个看上去有点像是一些古装片里化妆比较夸张的那种妖女。
<br/>　　她半身倚靠在车厢上，看上去很舒服的姿势，手里还端着一个古铜色的酒器，脸上带着一丝邪魅的微笑，朝叶少阳望过来。
<br/>　　叶少**本就不认识她。
<br/>　　鬼车边上，还走着一个男子，穿长衫，络腮胡子，头发凌乱，整个一大叔模样，一只手提剑，一只手提着个酒葫芦，目光落在芮冷玉脸上，表情有些凝重。
<br/>　　两人身后，跟着很多身穿甲胄的士兵，身上都散发着魂气，一看便知都是鬼魂，还有一些是骷髅。
<br/>　　鬼兵和尸兵！
<br/>　　阴司的人？
<br/>　　叶少阳怔了一下，随即想到，肯定不是阴司的人，这两个人身上邪气太重，不像是阴司的鬼将，有着一种执法的威严之气，而且这两个人，自己都不认识。
<br/>　　联想到之前他们喊出的那句话，叶少阳猛然就明白了：这两个人，是太阴山来的！
<br/>　　“鬼童，觉醒了。”那女子望着芮冷玉，喃喃说道，沙哑的声音中透着一丝忧郁。
<br/>　　自从两人现身之后，芮冷玉就停止了叶少阳的进攻，转头看着他们，神色跟之前对待叶少阳一样，透着一丝疑惑。
<br/>　　“殿下，我是天池罗刹，这位是雪魔。我们都是您的仆人，您觉醒得太早，好在右君预先料到，特派我们来接您回太阴山，免遭荼毒，速跟我们走。”
<br/>　　“太阴山……”芮冷玉脸上，露出一丝迷茫。
<br/>　　叶少阳却是听得心惊，雪魔他是不知道，但天池罗刹，说起来也算是听说过，太阴山有四大罗刹，都是右君手下最得力的干将，这个天池罗刹，八成就是其中一个了。
<br/>　　雪魔说道：“她刚觉醒，往昔的记忆并未恢复，这又不是在鬼域，不可能跟鬼王有神识联动，先带她回去再说。”
<br/>　　说完长剑一点，凌空飞了过来。
<br/>　　芮冷玉没有任何反应，没有抗拒，也没有接近。
<br/>　　叶少阳这才回过神来，纵身一跃，挡在了雪魔前面，龙泉剑在空中挽了一个剑花，布置成剑阵，将雪魔挡住。
