　　第105章
<br/>　　程大雷刚回屋子，柳芷便端着一盆热水进来。
<br/>　　“爷，您先烫烫脚，水正烧着，等下就能洗澡。”
<br/>　　程大雷的衣食住行都由柳芷照顾，端茶倒水，铺床叠被，连程大雷的衣服都是由她洗，无微不至到令程大雷都觉得不好意思。程大雷并不是一个衣来伸手饭来张口之人，由这样一个水灵的小姑娘贴身伺候着，老实说……感觉还蛮不错。
<br/>　　“这个世界也不是一点意思都没有呐。”程大雷将脚放入水盆中，连日奔波的疲惫此刻才消减些。他想起一事，问道：“柳芷，你觉得他们为何要联合起来对付我们，我一直觉得我的人缘还是蛮不错的。”
<br/>　　“大概是他们吃不上饭吧。”
<br/>　　“吃不上饭就能抢我，去哪儿说理去。”
<br/>　　程大雷没好气的说一声，伸手去拿毛巾，柳芷早已拿着毛巾蹲在他脚下。
<br/>　　咦……好堕落啊，忙让柳芷站起来，他可受不起这个。
<br/>　　程大雷始终无法理解这个时代，嫁鸡随鸡嫁狗随狗，在这个小妾可以随意送人的时代，程大雷所作所为简直可以算作男人标杆。
<br/>　　漫长的冬日闲适且无聊，伸手能将手指头冻掉的天气里，什么都做不了。程大雷想做些什么，也只能等着来年开春。他也只好和所有人一样，窝在山寨里熬冬。
<br/>　　终日和柳芷窝在屋里逗闷子也没什么趣味，偶尔程大雷也会随秦蛮一起出去打猎，以作消遣。
<br/>　　本是兴趣所至，但亲身经历过才知道，这实在不是什么好差事。
<br/>　　大雪封山，积雪厚处有一人多高，山洞、树坑、悬崖都被隐在大雪之下，一脚踏错，结果可能就是一命呜呼。
<br/>　　幸好有秦蛮引路，才算没有遇到什么危险。
<br/>　　“秦蛮，我能做什么？”程大雷裹着大袍子，手中握一杆长矛。
<br/>　　“别添乱。”
<br/>　　“……”程大雷。
<br/>　　“大当家你小心些，别惊了树洞中的山熊，咱们今天能打些狍子，野兔就行。”
<br/>　　“好打么？”
<br/>　　“不好打。”
<br/>　　二人继续往山中走，大雪覆盖，一片白茫茫，可雪地上时不时就会出现一个个深坑，连里面的黑土也被翻了上来。
<br/>　　“秦蛮，你仔细看看这里，这是什么动物挖的？”程大雷激动道。在这个混乱的世界，如果真出现什么古怪的动物也不算奇怪：三头犬，半人马之类的。
<br/>　　“人。”
<br/>　　“呃……好吧。”程大雷有些尴尬：“谁挖这些干嘛，不怕把手冻僵？”
<br/>　　秦蛮停下脚步，指着眼前的雪林道：“大当家，你莫看这里白雪封山，其实厚厚的积雪下面还长着不少植物，可以用来填肚子，我从前就挖过，当然现在用不着了。”
<br/>　　“人参？”
<br/>　　“草根。”
<br/>　　程大雷撇撇嘴，正行着，前方雪堆中突然冒出两个女人。像是一对母女，母亲大概三十岁，可模样显得更老，女孩不过七八岁，都裹着黑色麻布袍子，脸上挂着黑泥。
<br/>　　“你们是干嘛的……”程大雷眼前一亮，莫非是什么新职业：“女巫？”
<br/>　　“挖草根的。”秦蛮从怀里掏出一块菜饼，塞到女孩手中：“你们是哪家山寨的？”
<br/>　　两人没有回答，争着吃那块冻僵的菜饼。女人连一粒冰屑都不放过，就在程大雷面前撩开袍子，翻找掉进脖子中的饼屑。
<br/>　　“这……”
<br/>　　程大雷准备说什么，秦蛮拍拍他的胳膊，带着他离开了。
<br/>　　二人没有继续打猎，而是回到山寨。程大雷心情郁郁，那个女孩的眼神总是在他脑海中出现。
<br/>　　程大雷来到地牢。
<br/>　　“大，大！”
<br/>　　“小，小！”
<br/>　　“哈，又是大，老娘赢了，老娘一直大，从来都大！”
<br/>　　还没走进去，就听到大声的喧哗声。程大雷走进看才发现，小白狼和高飞豹两拨人正隔着牢房中间的栅栏赌色子。
<br/>　　其中最兴奋的就是小白狼，她似乎完全忘了高飞豹曾经想对他们下黑手。
<br/>　　“呵，你们玩得挺热闹，要不算我一个。”
<br/>　　见程大雷进来，两拨人才离开栅栏，回到各自牢房中，睁大眼睛看着程大雷。
<br/>　　“告诉大家一个好消息，我决定先释放一部分人，剩下的人继续关着，看你们表现。你们几个里面，谁想第一个出来？”
<br/>　　一伙人聚在一起嘀嘀咕咕，忽然一个人站起来，道：“姓程的，你不用假惺惺装好人，老子是不会向你服软的，谁走我也不走，老子和你死磕。”
<br/>　　其他人扼腕叹息，那表情分明再说：这话本该我说的，怎么让他抢了先。
<br/>　　程大雷感觉莫名其妙：山贼什么时候开始变得这么硬骨头了。
<br/>　　杏花岭和飞虎寨两拨人竟没有一个要走的，人人表情亢奋，说什么宁肯死在牢里，也不会离开牢房之类的话。
<br/>　　程大雷一开始觉得这是因为山贼的骨气，后来明白不是这么会事，这帮人是真不想走。
<br/>　　“不行，今天必须走八个，你们两拨人商量一下，看哪几个走。”程大雷道。
<br/>　　“老李，你走吧。”
<br/>　　“哥哥啊，我上有八十老母，下有八岁小儿，就让我在这里多住两天吧。”
<br/>　　“放心，你放胆去吧，以后你的老母我孝敬，你的孩子我养，你的老婆……”
<br/>　　这样的对话在牢房内乱成一片，搞得像被关在这里是天大的幸运，放出去反而是死路一条。
<br/>　　这时候有人给牢房送饭，程大雷也就出来了，他脑海里还是一头雾水，琢磨不明白是怎么会事。
<br/>　　刚出地牢，就被刘悲喊住。刘悲带着程大雷到蛤蟆岭的蛤蟆嘴处。这里是蛤蟆岭的制高点，从这里望去，整个青牛山大半都收入目中。
<br/>　　“刘兄，干嘛呢，要开饭了？”
<br/>　　“大当家，你来看……”刘悲信手一指，如沙场点兵，挥斥方遒。
<br/>　　程大雷近前一步，站在刘悲身旁，远眺望去。
<br/>　　“山若冰砌，林若雪雕，天地魁伟，不过如此。”程大雷用一种天下英雄唯使君与吾耳的眼神看着他：“此等雄浑景色，凡夫俗子不懂欣赏。刘兄怀里，有大山河呀。”
<br/>　　“你说什么呢？”刘悲看了程大雷一眼：“我是让你看，偌大一个青牛山，只有蛤蟆寨有炊烟……”
<br/>　　
<br/>(本章完)
