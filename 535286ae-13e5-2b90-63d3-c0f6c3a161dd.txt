　　第二日，
<br/>　　叶尘从修炼状态中醒来，打开房门，却看到张凌云，早已经在外面等候。
<br/>　　看到叶尘出现，张凌云当即迎了上来，
<br/>　　“叶兄弟，你这次可一定要帮我啊！”
<br/>　　紧跟着，
<br/>　　张凌云便将自己要与张齐云比试的事情，全部一五一十的讲了出来。
<br/>　　叶尘听完之后，眉头微微一皱，
<br/>　　“你的意思，是想让我帮你炼丹，来维持你拍卖会的生意，从而来夺得少阁主的位置？”
<br/>　　张凌云见叶尘面露不悦之色，顿时心里咯嘣一声，连忙道：
<br/>　　“叶兄弟，我知道，你让我办的事情，我还没有做到，现在就让你替我办事，的确有些不厚道，可是我这次若是败给了我二哥，将会一无所有，到时候可就没有办法替你办事了！”
<br/>　　叶尘依旧默然不语。
<br/>　　张凌云顿时急了，只好一咬牙，接着又道：
<br/>　　“我保证，只要您帮我坐上少阁主的位置，我必定调动整个万宝阁的力量，为您寻找天宝级灵物，不不！但凡您有任何需要，我张凌云万死不辞！！”
<br/>　　张凌云显然也是被逼急了，连口气也变得无比恭敬。
<br/>　　叶尘又沉吟了片刻，方才缓缓道：
<br/>　　“承诺在没有兑现之前，一文不值！比起毫无价值的空头支票，我更看重实实在在的利益！”
<br/>　　张凌云听到这话，脸色顿时微微一变，有些尴尬，
<br/>　　“那叶兄弟你的意思是？”
<br/>　　叶尘淡淡道：
<br/>　　“很简单，我可以帮你炼丹，只不过，我有两个条件！”
<br/>　　张凌云顿时精神一震，
<br/>　　“叶兄弟说笑了，别说两个条件，就算是二十个！二百个！我也全都答应你！”
<br/>　　说完这话之后，张凌云就后悔了，他还生怕叶尘提出什么非分的条件，不由得尴尬一笑，连忙又加了一句，
<br/>　　“当然了，叶兄弟应该也不会提出什么非分的条件吧？”
<br/>　　叶尘淡淡一笑，
<br/>　　“放心，我的条件很简单！”
<br/>　　“第一，我需要神魂类的天材地宝，不限等级，越多越好！”
<br/>　　张凌云微微一愣，虽然他一直很好奇，叶尘为何对神魂类的灵物如此痴迷，不过还是当即道：
<br/>　　“这个当然没问题！天宝级的灵物，可遇而不可求！地宝级的灵物，我还是能找一些的！”
<br/>　　“很好！”
<br/>　　叶尘点了点头，
<br/>　　“至于第二个条件，我可以帮你炼制丹药，不过我炼制的丹药，一半归我！”
<br/>　　“这……”
<br/>　　张凌云顿时眉头一皱，显然有些为难。
<br/>　　叶尘嘴角微微一扬，
<br/>　　“怎么？你不舍得？”
<br/>　　张凌云苦笑道：
<br/>　　“叶兄弟不要误会！若是在平日，区区一些丹药，我自然不会舍不得，可是如今我要和老二打对阵，若是再给你一半丹药，我怕我会输啊！”
<br/>　　叶尘微微一笑，
<br/>　　“这个你尽管放心！区区一个四品炼丹师而已，我一只手就足够吊打他！”
<br/>　　张凌云听到叶尘如此狂言，顿时两眼一瞪，露出满脸的不可思议之色。
<br/>　　连四品炼丹师都不放在眼里，难道他是五品炼丹师，甚至更高？
<br/>　　要知道，五品炼丹师，即便是在那些高级星域，也是受万人追捧的存在。
<br/>　　他一个不到二十岁的少年，真的具备这样的实力吗？
<br/>　　张凌云疑窦丛生，张了张嘴，却终究没有问出口。
<br/>　　他如今被逼到绝路，除了选择相信叶尘，已经别无任何办法。
<br/>　　然而，
<br/>　　接下来几天，叶尘的表现，却让张凌云气得差点骂娘。
<br/>　　原来，
<br/>　　自从第一天，叶尘洋洋洒洒在纸上，写下了几百味药材，交给张凌云后，便没了下文。
<br/>　　之后一连三天，叶尘连丹房的门都没有进，一直躲在房间里，不知道在忙些什么。
<br/>　　眼看着，张齐云那边的拍卖场，因为那位柳大师的丹药，生意已经异常火爆，而叶尘这边却一颗丹药也没有炼出来。
<br/>　　“这小子是不是故意在耍我？不管了！我今天一定要把他给揪出来，好好问个明白！”
<br/>　　忍到第五天，眼看着自己的成绩，已经远远落后于张齐云，张凌云实在忍无可忍，怒气冲冲的来到叶尘门前，便要破门而入。
<br/>　　然而，就在这时，
<br/>　　轰隆！
<br/>　　叶尘房间中，突然发出一道巨震！
<br/>　　继而，
<br/>　　以叶尘的房间为中心，周围的空气，陡然变得无比炙热起来！
<br/>　　下一刻，
<br/>　　在张凌云等人惊愕的目光中，一团透着恐怖气息的火焰，自叶尘的房间之中冲出！
<br/>　　整个房间，瞬间便被焚烧一空！
<br/>　　蹬蹬蹬！
<br/>　　所有人顿时吓得纷纷倒退。
<br/>　　好在那团恐怖至极的火焰，很快便消退了下去，最终缩小成了一道五彩色小火苗，落在叶尘的手掌之上，缓缓跳动着，
<br/>　　“这……这难道就是传说中……比三昧真火还要高一等的，五昧真火？”
<br/>　　人群中，有人忽然失声高呼。
<br/>　　“什么！这就是五昧真火么？难怪我感觉到如此恐怖！”
<br/>　　众人一片骇然。
<br/>　　原来，
<br/>　　叶尘这几天之所以一直在闭关，就是为了提升自己火焰的等级。
<br/>　　想要炼出高品质的丹药，一大半的功劳都在火焰上面。
<br/>　　叶尘的火焰，从三昧真火提升到五昧真火，炼丹水平自然也跟着水涨船高。
<br/>　　“去！帮我之前让你准备的药材取来，顺便再去找十个丹炉，我要开始炼丹了！”
<br/>　　叶尘心念一动，将掌心中的五昧真火收回，直接冲着张凌云下达指令道。
<br/>　　“哦哦，好好！”
<br/>　　此时的张凌云，满腔的怒火，早已经消失的无影无踪，取而代之的，却是深深的震撼和惊骇，当即命令下人，按照叶尘的吩咐去办。
<br/>　　很快，
<br/>　　十个灵器级别的丹炉，以及上千斤的药材，全部被人取来，放在了院落之中。
<br/>　　叶尘缓步走到那十个丹炉的中央位置，
<br/>　　“起！”
<br/>　　随着叶尘一声低喝，那十个丹炉几乎同时悬浮了起来，然后在真元的催动之下，开始发出一道道璀璨夺目的光芒，
<br/>　　“进来！”
<br/>　　随着叶尘再次一声低喝，那些药材顿时便按照一定的顺序，开始鱼贯飞入炉鼎之中。
<br/>　　“五昧真火！开！”
<br/>　　随着叶尘再次低喝，
<br/>　　呼呼呼呼！
<br/>　　叶尘掌心中的火焰，顿时一分为十，朝那十尊炉鼎包裹而去。
<br/>　　而周围众人看到这一幕，早已经惊愕的说不出话来。
<br/>　　一心十用，同时操控十尊炉鼎，而去炼制的丹药，也各不相同！
<br/>　　此等炼丹手段，他们简直闻所未闻。
<br/>(本章完)
