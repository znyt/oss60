        &lt;h3&gt;第三卷第一百零  四章  辛苦的潘青松&lt;/h3&gt;
<br/>
<br/>        陈子澜和同事已经在西城呆了四天了，他们在调查一个案件。
<br/>
<br/>        泉城的一个小伙子因为和开裆接触上了，将家里折腾得一无所有，他的母亲一气之下跳楼了。
<br/>
<br/>        他母亲跳楼引起了他们的注意。
<br/>
<br/>        细挖深查，他们顺着线索摸到了西城。
<br/>
<br/>        但是当他们刚刚发现有点蛛丝马迹的时候，那个有嫌疑的人就在上午出了意外，酒驾，撞上了护栏，翻到高架桥下，砸了几辆车，意外死亡。
<br/>
<br/>        陈子澜他们赶到车祸现场的时候看着惨状，只是唉声叹息。
<br/>
<br/>        线索又断了。
<br/>
<br/>        本来想下午随同事们一起回去的，现在来西城方便很多，从西城到泉城的高速路通了，回泉城只要一个多小时。
<br/>
<br/>        “陈姐，你既然来了就去看看潘哥呗。”同事说。
<br/>
<br/>        “算了吧，我随你们一起回去。”陈子澜觉得还是回去上班。
<br/>
<br/>        “给你留一辆车，我们几个开一辆车，你这几天都在西城，潘哥都不知道，事情已经这个样子了，我回去写一份报告，你明天早上回来看看，反正明天早上你回来上班也是赶得上。”
<br/>
<br/>        “也好。”陈子澜并没有坚持，最近忙这件事情都很少和潘青松联系，而且他们手头只要有事情，基本上只能内线联系，包括自己的亲人都不能联系。
<br/>
<br/>        陈子澜给潘青松打电话，但是潘青松没有接，电话忙音。
<br/>
<br/>        陈子澜直接就开车去了潘青松单位分的房子，潘青松是领导，他们单位给他们分了一套三居室，因为手头上的资金不宽裕，迟迟都没有装修，搬进去才两年多，说句丑话，装修的钱还是子昂和子悦一人拿了一半，连带装修和买家具，一共花了子悦和子昂大概有四十多万。
<br/>
<br/>        当初搬新家的时候，晚上潘青松笑着说：“老婆，弟弟妹妹们都那么有钱，我都可以不上班了，求包养。”
<br/>
<br/>        “行，你不用上班了，我每天给你吃一碗凉皮就好了。”
<br/>
<br/>        “像我这样玉树临风，人见人爱，花见花开的油腻中年大叔，怎么只能吃一碗凉皮呢，最起码一碗油泼面吧？”
<br/>
<br/>        “都油腻了还花见花开？反正我这朵花不会为你开的。”
<br/>
<br/>        “当真？”潘青松看着陈子澜，眼里盛着笑。
<br/>
<br/>        “当真！”陈子澜坚定的神情就像是上战场。
<br/>
<br/>        “你说的哦，别求着喊哥哥。”潘青松哈哈大笑。
<br/>
<br/>        “今天是我们搬家的第一天，人生第一喜事，乔迁新居，第二，花烛夜。嗯，老婆，你说是也不是？”潘青松的声音像细细的春雨落在陈子澜的心尖。
<br/>
<br/>        陈子澜打开家门，她其实是很少来西城的，都是潘青松回去。
<br/>
<br/>        家里一尘不染，潘青松永远都是那个清爽明亮的人。
<br/>
<br/>        她曾经问潘青松：“我不在你身边你是怎么过的？”
<br/>
<br/>        “上班，下班，回家，有必要的应酬是会去的，没有应酬就回家收拾收拾卫生，跑跑步，健健身，当然，想你的时间最多，因为我无论在干什么都会想到你。”
<br/>
<br/>        其实，她和潘青松在一起的时候，他们的生活状况也是如此。
<br/>
<br/>        她打开冰箱，冰箱里的菜看来有好几天了，潘青松应该这几天都没有在家里吃饭，不知道晚上回不回家吃饭。
<br/>
<br/>        她下厨房做了点西红柿鸡蛋酱，给自己煮了点面条，给潘青松也留了一些酱，要是潘青松晚上没有吃饭的话，他回来她就给他也煮点面条。
<br/>
<br/>        一直到晚上十一点，潘青松还没有回来，她打电话，电话还是忙音。
<br/>
<br/>        陈子澜知道潘青松在忙着。
<br/>
<br/>        他们看着似乎没有什么事情可干，但是一忙起来便会没日没夜。
<br/>
<br/>        十二点钟，陈子澜听到了开门时，潘青松见穿着睡衣的陈子澜，欣喜的说：“我就感觉老婆回来了。”
<br/>
<br/>        “怎么这么晚才回来。”陈子澜迎上去帮潘青松脱下外套，她看见潘青松的眼睛都是红的。
<br/>
<br/>        她深深的叹了口气。
<br/>
<br/>        “吃饭了吗？”陈子澜不等潘青松回答便岔开话题。
<br/>
<br/>        “吃过了，吃的肯德基。”
<br/>
<br/>        “这是有三天多没休息？”
<br/>
<br/>        “什么都瞒不过老婆的火眼金睛。”
<br/>
<br/>        “那就快点休息。”
<br/>
<br/>        “嗯，老婆等着急了。”潘青松边洗手边回答。
<br/>
<br/>        陈子澜看着他那熬得发红的眼睛又心疼又好笑，这人啊，在外面看着温文儒雅，只要在她面前就没个正行。
<br/>
<br/>        “这几天忙得都没有时间给你打电话，老婆今天怎么就来了呢。”潘青松拉着陈子澜的手朝卧室走去。
<br/>
<br/>        “我到西城好多天了。你打电话也打不通。”
<br/>
<br/>        “嗯，我这几天也一样，你看，电话到现在还没有开机。”潘青松拿出手机充电。
<br/>
<br/>        “工作形式很紧张？”
<br/>
<br/>        “比前几年要严厉很多，这是非常好的事情，我们有些人啊，已经被腐蚀的很严重了，是该清理清理了。”
<br/>
<br/>        “是的，社会在发展，人们的生活水平都在提高，对于金钱的渴望也会随着地位的提升、生活水平的提高而提高，有些人能保持初心，有些人就陷入对金钱游戏中了。”陈子澜感慨的说。
<br/>
<br/>        “这几天我还会很忙，二哥在家里还好吧？”潘青松想到二哥，二哥还算是两袖清风。
<br/>
<br/>        “还好，在家里给爸妈帮帮忙，写写论文。”陈子澜扒拉着潘青松的头发说，好像有白头发了。
<br/>
<br/>        “二嫂做的事情对二哥的打击很大，他主动调离工作岗位，请了长假，其实二哥这样做是对的。”潘青松低着头，任妻子在头上寻找着。
<br/>
<br/>        “二哥知道很多，不知道的呢？不过好的是二哥都没有经手，现在对二哥影响不大。”陈子澜叹口气，二哥算是毁在二嫂手中。
<br/>
<br/>        “二嫂到现在都不知道二哥回了老家，她给我打了几次电话问我，我说不知道，你说她怎么就不回老家去看看。”潘青松理解不了冯睿的做法。
<br/>
<br/>        “二嫂从来都不把我们家的人当人看，她一个人怎么会主动回老家去找二哥呢。”
<br/>
<br/>        “生着一个公主的心却没有一个公主的命，到现在还改不了那种自恃清高的态度。”
<br/>
<br/>        俩个人聊了一会家事。
<br/>
<br/>        陈子澜看着潘青松实在有些疲惫的样子，便主动结束了谈话。
<br/>
<br/>        “时间不早了，早点休息。”
<br/>
<br/>        “老婆。”潘青松又准备卖萌。
<br/>
<br/>        “老实点。”陈子澜紧紧的拉着潘青松的手，潘青松动弹不了，便轻轻的躺，在陈子澜的身侧。
<br/>
<br/>        不过一分钟，便传出了潘青松轻微的鼾声。
<br/>
<br/>        午夜，是黑的聚集地，也是黎明到来的前端，像潘青松和陈子澜夫妇就是守护一方净土的卫士。
<br/>
<br/>        世界很大，哪能都是一片祥和，只是有他们这样的人儿，在清理着藏污纳垢的犄角旮旯，维护着一片宁静和安逸而已。
