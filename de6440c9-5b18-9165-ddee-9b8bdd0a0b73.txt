        “筱颖……”傅夜沉忽然轻唤着童筱颖的名字，他的大掌捧住她的脸，将她的小脸轻轻地抬起来，直直地迎向他自己的眼，一往情深地说，“我想吻你。”
<br/>
<br/>        “不——”可以……
<br/>
<br/>        不容她拒绝，他倏地倾下身，封住了她的唇，黑眸在极近的距离里迎向瞠大眼的她。
<br/>
<br/>        当炙热的温度从自己的唇上传来时，童筱颖感觉自己被下了定身咒。
<br/>
<br/>        傅夜沉的黑眸灼热，眸里好似烧了火，双手捧着童筱颖的脸，薄唇吸吮她的唇。
<br/>
<br/>        童筱颖完完全全的措手不及，无法反应，瞠着一双眸，瞪视着另一双眼里的自己，两人靠得如此的近，她整个人都软瘫在了他的怀中。
<br/>
<br/>        他炽热的薄唇碾过她轻颤的唇瓣，吻很温柔，透着禁欲系的清甜……
<br/>
<br/>        童筱颖被他吻得双颊泛红，身体的每一个细胞仿佛都敏感起来，原本一双瞠大的眼不由自主地慢慢低垂。
<br/>
<br/>        这种又酥又麻又像触电般的感觉，从唇瓣间蔓延到四肢百骸各个敏感细胞上。
<br/>
<br/>        不知不觉中，童筱颖缓缓地闭上了眼睛，慢慢地享受着这个吻。
<br/>
<br/>        她没有回吻他，但她不再抗拒他的吻了？！
<br/>
<br/>        傅夜沉受宠若惊的心里，情动难抑。
<br/>
<br/>        吻她，是他惦记了一辈子的事情。
<br/>
<br/>        他捧住她的脸，贴着她的唇，吻得难舍难分。
<br/>
<br/>        办公室内，这过于亲热的画面，让候在门外不敢吭声的落以柔感到尴尬与心酸。
<br/>
<br/>        她没看到他两是怎么吻起来的，但在她的潜意识里，一定是童筱颖勾引的傅夜沉。
<br/>
<br/>        这个女人，到底给傅夜沉灌了什么迷魂汤？
<br/>
<br/>        “咳——”落以柔深吸了口气，清了清喉咙，叩响了办公室的玻璃门，试图给办公室里这两个吻得热火朝天的人降降温。
<br/>
<br/>        “傅总，五分钟后，您得去一号会议室开会。”
<br/>
<br/>        落以柔的声音传入耳中时，傅夜沉艰难地让自己从这个太过甜蜜的香吻中回过神来。
<br/>
<br/>        他依依不舍地结束了这个深吻，饱满的额头抵着她的眉心，鼻梁流连忘返地在她鼻尖上摩挲。
<br/>
<br/>        傅夜沉什么话也没说，黑眸里的烈火未褪，隔着几公分的距离，耐心地等待着她睁开双眸。
<br/>
<br/>        灼热的气息，就在她的鼻尖前，童筱颖的心跳不知所措地乱了谱，浑身一阵颤抖。
<br/>
<br/>        她没有勇气睁眼，几乎不敢面对他。
<br/>
<br/>        这个吻，莫名其妙地让她脸红心跳，还有点回味无穷。
<br/>
<br/>        “还想我继续吻你，嗯？”傅夜沉意味深长地说，薄唇微扬，笑意不减地逗她，“要不，我今天不工作了，我们回家继续……”
<br/>
<br/>        童筱颖的眼睛倏然睁开，几乎忘记了刚刚那种难以言说的情愫，瞪着一双铜铃般的大眼睛看着傅夜沉。
<br/>
<br/>        她的双手一直抵在他的胸膛上，将他的衬衫揪成两团褶皱，此刻，她才彻底回过神来，松手、稍稍远离，用力地往他胸口上一捶：“你坏！”
<br/>
<br/>        天啦！
<br/>
<br/>        她刚刚居然……
<br/>
<br/>        沉浸在他的那个吻中了……
<br/>
<br/>        “傅总，需不需要我再重新安排您的行程？”落以柔站在总裁办门前，手里握着行程表，面无表情，故作一丝不苟地问。
<br/>
<br/>        童筱颖一脸尴尬，连忙阻止道：“我马上走！”
<br/>
<br/>        傅夜沉一只手拉住童筱颖的手，一直手的指托着她的下颚，指尖轻抚着刚刚被他吻得有些红肿的唇，然后宠溺地刮了刮她的鼻梁，叮嘱道：“回去的路上，开车注意安全。”

<br/>        “嗯。”童筱颖乖巧地点了点头，将手从傅夜沉的手中抽了出来，转身离去。

<br/>        当她与落以柔擦肩而过的时候，落以柔用余光瞥见到她整个耳根都红了个通透，心生不屑，回眸定睛看向傅夜沉。

<br/>        只见他刚刚还对一个女人柔情似水，转瞬间，回到工作岗位上，他又是一脸冷峻严肃。
<br/>
<br/>        认识他的这段时间里，她真的从未见过他对谁有过这般的耐心和温柔。
<br/>
<br/>        难道说……
<br/>
<br/>        他是爱上了那个童筱颖吗？

<br/>        傅夜沉似乎毫不在意，脱下西装外套，挂在落地衣架上后，淡定地拿起桌上的文件，说道：“现在去一号会议室。”

<br/>        他只身走到办公室门口时，忽然想起了什么，微微偏头，吩咐道：“下去跟他们吩咐一下，以后夫人来公司，不许再阻拦。”
