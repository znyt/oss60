　　“所以我问你，你怎么看。”大长老言语中若有所指。
<br/>　　“臭小子，大长老的意思是，你到底有没有做过那等让人发指之事，谣言是否属实。”二长老直接不耐烦地问道。
<br/>　　萧逸仍旧语气不变，反问道，“那你们认为呢？如果谣言是真的，你们要交我出去？”
<br/>　　“废话。”二长老冷喝道，“若你真的如此丧心病狂，我们还护着你不成？”
<br/>　　“呵呵。”萧逸忽然笑了出来。
<br/>　　“臭小子，还笑得出来？”二长老顿时一怒。
<br/>　　他是个武痴，也是个大老粗，那须发皆怒的样子，着实吓人。
<br/>　　萧逸双手一摊，道，“好吧，我承认，谣言是真的，交我出去吧。”
<br/>　　“小混蛋，你...”二长老双目一瞪。
<br/>　　“逸儿，别乱说话。”三长老连忙拉了拉萧逸。
<br/>　　“好了。”大长老摆摆手，让二、三长老不用出言。
<br/>　　二长老仍旧怒气不减，道，“大长老，你看萧逸这臭小子算什么话，还一副死猪不怕开水烫的样子，我真想扒了他的皮吊打一顿。”
<br/>　　大长老示意二长老稍安勿躁，看向萧逸，问道，“你真的不怕我们交你出去？”
<br/>　　萧逸没有回答，反问道，“那你们会交我出去吗？”
<br/>　　大长老认真地打量了萧逸一下，而后笑笑，道，“当然不会，你身为萧家的少家主，代表了萧家，还轮不到慕容家的人处置你。”
<br/>　　萧逸笑笑，“那不就得了。”
<br/>　　二长老再次怒道，“不交你出去归不交，但你那语气和态度算个什么事。”
<br/>　　“得了。”大长老出言道，“这小子早就知道我们不会把你交出去，知道你在唬他，故意气你罢了。”
<br/>　　二长老愣了愣，问道，“什么意思？”
<br/>　　萧逸笑道，“你们心中早就有了答案，根本无需问我。”
<br/>　　二长老仍旧满脸疑惑，“这又是什么意思？”
<br/>　　“笨蛋。”大长老骂了一声。
<br/>　　萧逸解释道，“二长老以为，其他的少家主会像我这样混得那么惨吗？别人家的少家主，哪个没有强者保护？慕容娇儿乃是慕容家当代家主的千金，平日里起码有三个以上的后天境武者保护。我这点儿微末修为，能对她用强的才怪。”
<br/>　　萧逸继续道，“至于刚才大长老问我怎么看，恐怕说的不是慕容娇儿这事，而是关于慕容千军吧。”
<br/>　　“聪明。”大长老满意地赞扬了一声，道，“那种谣言，稍微有点脑子的人都不会信。”
<br/>　　说着，大长老瞥了二长老一眼。
<br/>　　二长老脸色讪讪，装作看不到大长老的目光。
<br/>　　大长老继续道，“慕容家摆明了只是存心找事，不过，偏偏来闹事的是慕容千军，我们这些老一辈也不好出面对付他。”
<br/>　　萧逸接话道，“放心，慕容千军敢来萧家撒野，我萧逸也不怕他，自然会把场子找回来。”
<br/>　　“很好，有种。”大长老再次赞扬了一句。
<br/>　　“不过。”大长老随即脸色肃然，道，“慕容千军很强，他的天赋比萧若狂更高，成名也更早。三年前已是闻名一时的天才，后天一重武者，如今起码是后天五重以上的修为。”
<br/>　　“就算是家族内一般的执事都不是他的对手，你有把握把场子找回来？”
<br/>　　萧逸点点头，道，“有，挑个日子，我亲自去慕容家找他。”
<br/>　　恰在此时，门外一个族人脸色着急地跑了进来，显然是找三长老的。
<br/>　　“额。”这个族人看到大长老和二长老都在，愣了愣。
<br/>　　三长老见状，呵斥道，“何事慌慌张张的？”
<br/>　　族人这才反应过来，想说些什么，但看到萧逸在场，却又忽然支支吾吾的。
<br/>　　“到底怎么回事，不妨直说。”三长老道。
<br/>　　“禀三长老。”族人这才说道，“陨星山脉那边的族人传回来消息，慕容家的子弟由于之前少家主的谣言，正疯狂地针对我们萧家的子弟。”
<br/>　　“据消息称，在慕容千军的带领下，我萧家不少子弟虽没有伤亡，但有不少人都受了重伤。”
<br/>　　陨星山脉，在紫云城之外，存在着大量的紫云铁矿脉。
<br/>　　紫云铁，是一种珍贵的铁矿，被广泛用于兵器制作，价格不菲。
<br/>　　传言，每到日落之时，那一座座紫云铁矿，散发的耀目紫光，将整个上空的云朵都映衬为紫色，极为美丽。
<br/>　　紫云城，也是因此而得名，并成为附近最出名的城池。
<br/>　　萧家作为紫云城三大家族之一，在陨星山脉中拥有一大片领地和数座大型紫云铁矿脉。
<br/>　　而陨星山脉中，又有无数妖兽和险地，故极为危险，家族中派有强者常年驻扎。
<br/>　　家族中的子弟，也大多在那里历练。
<br/>　　族人继续汇报道，“我们萧家的子弟被打伤后，还被强行驱赶，甚至有数座小型紫云铁矿脉被慕容家的人强行夺了去。”
<br/>　　“混账。”脾气火爆的二长老当即大怒。
<br/>　　大长老也脸色阴沉，“慕容家这些年来一直野心勃勃，妄图称霸紫云城。这次的无中生有中伤萧逸，还有抢夺紫云铁矿脉，恐怕只是一个开始。”
<br/>　　三长老当即道，“大长老不用担心，我现在就带领家族武者去陨星山脉，压一压他们的威风。”
<br/>　　“不可。”大长老摇摇头，道，“毕竟是小辈之间的事，我们这些长辈过去，只会破坏规矩，等同和慕容家撕破脸皮，没必要将事情严重化。”
<br/>　　“我去吧。”萧逸淡淡道。
<br/>　　“既然慕容千军在陨星山脉，也省得我去慕容家了。”
<br/>　　“你有把握？”大长老沉声问道。
<br/>　　“有。”
<br/>　　“何时出发？”
<br/>　　“明日。”
<br/>　　“这么快？”大长老皱起了眉头。
<br/>　　萧逸眼中迸发出一股冷意，道，“既然他们总喜欢来算计我，我不介意让他们十倍奉还。”
<br/>　　大长老、二长老和三长老，感受到这股澎湃的冷意，不禁一愣，想不明白萧逸哪来的自信。
<br/>　　毕竟，慕容千军的名声和本事人所皆知。
<br/>　　忽然，二长老出言问道，“小子，你的自信，来自于你那些古怪的招式吧。”
<br/>　　“有问题吗？”萧逸反问一声。
<br/>　　“当然有。”二长老，道，“你太少看后天境武者了，别以为你打败了萧若狂就可以得意扬扬。我告诉你，你一个凡境九重和慕容千军的差距非常大。”
<br/>　　“罢了，先不说这个。”二长老道，“小子，你知道我和大长老今天为什么会在这里吗？”
<br/>　　“不是为了慕容千军的事吗？”萧逸疑惑地问道。
<br/>　　“那只是其一。”二长老说道。
<br/>　　“哦？”萧逸面露疑惑之色。
<br/>　　大长老出言道，“萧逸，我且问你，你的炼药术，是何人所教？”
<br/>　　萧逸耸耸肩，道，“没人教，自己炼啊炼的，忽然就炼会了。”
<br/>　　大长老继续问道，“那你的修为呢，怎么跳跃得这么快，还有那些古怪的招式。”
<br/>　　萧逸仍旧道，“也是练啊练的，忽然就突破了，忽然就悟道那些招式了。”
<br/>　　“练阿练的，你骗鬼吧你。”二长老满脸不信地喝道。
<br/>　　“我没必要骗你们。”萧逸双手一摊。
<br/>　　大长老笑笑，道，“罢了，你不说我们也不逼你。这次，我们主要是要交一些东西给你。”
<br/>　　说着，大长老从怀中拿出一本书，道，“这是我大半辈子的炼药心得，还有一些丹方，你拿去看，有不懂的，可以随时来问我。”
<br/>　　二长老也从怀中拿出了两本书，道，“这是我萧家最高深的两本武技，同样的，有看不懂，可以来问我。”
<br/>　　“武技？”萧逸眼前一亮，他对炼药知识不太感冒，反而对于武学极为热衷。
