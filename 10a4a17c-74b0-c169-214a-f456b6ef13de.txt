        “谢谢。”傅夜沉很客气地谢了声，向两位前辈颔首后，轻而易举地捞起了落以柔的身子，将她横抱入怀中，起身疾步离去。
<br/>
<br/>        此时，落以柔迷迷糊糊地睁开了双眼，但是她觉得自己的眼皮子很重，只能半眯着眼睛，打量着这个抱着自己的男人。
<br/>
<br/>        是大哥……
<br/>
<br/>        这是她第一次以这样的角度去看这个男人，从鹳骨到下巴再到喉结，棱角分明，线条俊逸，透着十足性感的男人味。
<br/>
<br/>        如果他是她的男朋友，那一定是男友力爆棚的那种！
<br/>
<br/>        落以柔只觉自己浑身一阵恶寒，但是却觉得被傅夜沉抱在怀中，让她心窝里暖暖的。
<br/>
<br/>        要是……
<br/>
<br/>        他能这样抱着她，抱一辈子，该多好啊？
<br/>
<br/>        落以柔下意识地将自己的脑袋往傅夜沉的左胸上靠，缓缓闭上眼睛，侧耳倾听着他那强劲有力的心跳声，让她觉得很有安全感，也很幸福。
<br/>
<br/>        去医院的途中，傅夜沉一个电话，让两台警车一前一后为他开路，带着他的车连闯了三个红灯，以最快的速度将落以柔送到了医院。
<br/>
<br/>        傅夜沉将落以柔送到医院后，医生和护士早早地就推着移动病床在门口等候了。
<br/>
<br/>        接下来，傅夜沉将落以柔放到病床上后，交由医护人员护送，将她推入了急诊室。
<br/>
<br/>        连华生也随后赶了过来，在医院急诊大楼的大厅门口，与傅夜沉打了个照面。
<br/>
<br/>        起初傅夜沉给警局里打完电话后，又给连华生打了通电话，让他过来照顾落以柔。
<br/>
<br/>        连华生刚赶过来，见傅夜沉急着要走，立即随口谩骂：“诶？阿沉！你这也忒没良心了吧！好歹人家是你的秘书啊！你也得等医生诊断出她的病情后再走吧！”
<br/>
<br/>        “我还有两位重要的顾客要见，十个亿的合同在等着我。所以，小落的事情，就交给你了！”傅夜沉拍了拍连华生的臂膀，抬起手来看了看手腕上手表上的时间，二话不说就直接转身离去。
<br/>
<br/>        连华生忍不住在他背后没好气地大声调侃道：“喂！阿沉！如果是童筱颖出了事，我看你别说是十个亿的合同，上千亿的合同，你也不会要了吧！”
<br/>
<br/>        可惜啊！
<br/>
<br/>        落以柔不是童筱颖。
<br/>
<br/>        傅夜沉这个没心没肺的男人，一看落以柔就知道，她肯定是为了他在工作上太拼命了，才会生病的。
<br/>
<br/>        连华生第一次为一个女人感到愤愤不平。
<br/>
<br/>        傅夜沉回到自己的总裁办后，原以为那两位大佬已经走了，却没料到他们居然还坐在待客区里等他回来。
<br/>
<br/>        这两位大佬手中各自拿着一份资料，见傅夜沉从电梯里出来，纷纷站起身。
<br/>
<br/>        为首的中年男子，先是向傅夜沉道歉说他未经允许就私自先看了资料，后是夸赞这份资料做得很漂亮，特别是里面的企划案，正是他们想要的那种。
<br/>
<br/>        傅夜沉感到欣喜的同时，看着顾客手中拿着的那份资料，才想起落以柔的事情来。
<br/>
<br/>        为首的大佬将资料递还给了傅夜沉后，向他友好地伸出手来，微笑说：“希望合作愉快。”
<br/>
<br/>        “谢谢，合作愉快。”傅夜沉礼貌性地握了握手，嘴角微扬。
<br/>
<br/>        很快，他就把这份十亿的合同签到手了。
<br/>
<br/>        这其中有大部分的功劳得归落以柔。
<br/>
<br/>        他依稀记得，他把资料批改得密密麻麻，没想到她会把他的条条框框梳理得一清二楚。
<br/>
<br/>        傅夜沉亲自送走了两位贵客，转身还未进大厦正厅大门，就接到了连华生打来的电话。
<br/>
<br/>        “小落高烧不退，需要人陪护。她是你的秘书，你自己亲自过来照顾！我还有我自己的事情要去做。”连华生的语气不是一般的凛冽。
<br/>
<br/>        傅夜沉用毫无起伏的声调回答：“你帮我请个护工过去。”
<br/>
<br/>        “喂！阿沉，你要不要搞得如此冷血无情啊？不会体恤下属的老板，可不是好老板。”连华生咬牙切齿道。
<br/>
<br/>        傅夜沉却不为所动地说道：“我现在要去视察子公司，挂电话了。”
<br/>
<br/>        他说完，就把电话给挂了。
<br/>
<br/>        但是，他的人，还是去了医院。
<br/>
<br/>        连华生已经不在病房里了，落以柔一个人安安静静地躺在病床上吊点滴无人照料。
<br/>
<br/>        傅夜沉只身走进病房时，就看到落以柔的双颊因为高烧不退而涨得通红。
<br/>
<br/>        她柳眉微微蹙着，嘴唇的有点儿干燥又偏白。
<br/>
<br/>        看到这里，他转身去了护士站，回来的时候，手里多了一个冰袋。
<br/>
<br/>        傅夜沉把冰袋裹在了自己随身携带的手帕里，然后轻轻地敷在了落以柔的额头上。
<br/>
<br/>        这来来去去，不知不觉中，就是一整晚。
