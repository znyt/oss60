　　第563章
<br/>　　程大雷在最短时间内打探到自己需要的消息，当然，书院故意放出风来，目的就是把消息传到程大雷耳中。
<br/>　　现在程大雷打探到的消息是：自己的身份已暴露，刘发财被扣住，并且书院已准备将刘发财转移到海岛上。
<br/>　　甚至连海岛的信息他也打探到了。
<br/>　　岛名白沙岛，离岸三十里，有些书院不听话的弟子，会派到海岛上苦修。也有些喜欢自虐的书院弟子，自愿到海岛上苦修。
<br/>　　想也知道，白沙岛已经是天罗地网，东海一地的高手齐聚，还有书院的七十二剑阵，目的，就是诛杀自己这条大鱼。
<br/>　　程大雷再狂妄，也没有自信能以一己之力迎战东海所有高手。张六、宋寒之、还有那个虽然没有出过手，怎么看怎么像是老怪物的夫子……任何一人，都有与自己捉对厮杀的实力。
<br/>　　最佳选择是放过这件事不管，刘发财是生是死和自己也没太大关系，忍他个把月，等风声过去后，带着华正则离开便是。至于自己在江湖上的名声……喂，自己的名声什么时候好过，自己又什么时候在乎过名声。
<br/>　　想是如此想，但刘发财毕竟是因自己陷入这漩涡，若让他因自己而死，程大雷当真过不去。
<br/>　　无论江湖有没有义气，程大雷本身还是有几分义气的。
<br/>　　但闯海岛，自投罗网的事想也不要想，那几乎等于送死。
<br/>　　可自己想要救人，未必要去海岛上呐。在白沙岛排兵布阵，转移刘发财总是需要时间。目前，刘发财还未被转移。
<br/>　　自己夜探万卷山不就好了，只要行动够快，抢了刘发财就走，量书院也无人能拦得住自己。
<br/>　　其实也无所谓夜探，现在书院肯定严防死守，白天晚上并没有什么区别。
<br/>　　想通之后，程大雷开始行动。黄昏时分，他出现在万卷山下一茶楼中，只等天黑展开行动。
<br/>　　“先生是第一次来琅琊城吧？”茶楼的伙计一边给程大雷倒水一边道。
<br/>　　“嗯呐。看出来了？”
<br/>　　所谓灯下黑，满城都在搜捕程大雷，倒是没人想到程大雷正好就在书院山下。当然，程大雷现在也不惧被人发现。如今，几乎已是打明牌，被人发现了，一剑杀上山便是。
<br/>　　“这还能看不出来，哪个入城的外地人，不先到书院看看。”天下茶楼的伙计都是碎嘴，他冲程大雷道：“公子人，肯定也想瞻仰这书山圣地的风采。”
<br/>　　程大雷也不知道，他怎么看出来自己像个读书人。虽然也穿着儒生长衫，腰间佩着剑，但程大雷的剑可不是装饰品，而是真能杀人。况且，程大雷这张脸实在有些……
<br/>　　程大雷坐在二楼，为了避免被人打扰，已经将二楼包下。从程大雷坐的角度，正好能看到书院山下的玉石牌坊。
<br/>　　他有一搭没一搭的和伙计聊着天，大战之前，他也有些忐忑，需要借着说话理清思绪。
<br/>　　看上去，程大雷像个土豪，伙计道：“公子，楼下有唱曲儿的，要不我请上来，给公子唱个小曲解乏。”
<br/>　　程大雷略顿了顿，点点头道：“也好。”
<br/>　　有一老一少沿着楼梯上来，老者五十有余，外表看上去年纪更老，少者是个丫头，不过十三四岁。
<br/>　　“客官想听什么？”老者手里捧着琵琶。
<br/>　　“来一曲……将军令吧？”程大雷抿着茶杯，天已快暗了。
<br/>　　“呃……什么？”
<br/>　　“那个，来一曲十面埋伏？”
<br/>　　“老朽不会啊。”
<br/>　　“那你会什么？”
<br/>　　“小的会小桃枝，三月柳，九星子……”老者一连报出十几个曲牌。
<br/>　　“停。”
<br/>　　程大雷及时制止他，他说得这些，自己根本就没有听说过。当然，在诗词歌赋上，程大雷也并没有什么造诣。
<br/>　　老者神情忐忑，他们爷孙俩就是指望着卖曲挣钱，今天看程大雷像个土豪，又傻又有钱那种，也是伙计好心照顾他们。
<br/>　　程大雷视线盯着程大雷，表情忽然顿住。只见书院山下，一人身背重剑，迈步走过大牌坊，拾阶上山。
<br/>　　虽然燕不归已经二次闯山，但书院和他照过面的人当真不多。所以此刻上山，倒也没人认得他。至于相貌有些奇怪，书院相貌奇怪的人太多了。
<br/>　　但别人认不出燕不归，程大雷可不会忘了他的模样，尤其是那柄重剑。
<br/>　　这货无端端，上山干什么。看来势汹汹的模样，来者不善呐。
<br/>　　“有点意思，有点意思。”程大雷忽然笑了。
<br/>　　“客官，客官……不如我让妮子给您唱一段玉堂春，您抬抬手，赏我们倆铜板。”
<br/>　　“赏什么赏，凭手艺吃饭，又不是沿街乞讨，把腰板挺直了。”程大雷板起脸：“破阵子会么？”
<br/>　　“会会会。”老者忙不迭道：“书院宋先生曾做过一曲破阵子，最是慷慨豪放，正应此时此景。”
<br/>　　“什么宋先生，没听说过。”程大雷道：“你唱我作的。”
<br/>　　说着，程大雷用手指蘸着茶水，写在茶桌上。老者和丫头凑过脑袋看，两人靠这行当吃饭，记忆力算是不错的，只是看一遍就记住了。
<br/>　　程大雷的注意力却完全在燕不归身上，这货背着一柄重剑，一步一步上山，在程大雷这里都能察觉到他身上的杀气。
<br/>　　终于，走到半山腰那座红门前时，他还是被人拦住了。一脸凶神恶煞，明显像找事来的，别人不注意他也是困难。
<br/>　　“喂，停下。”看门的大汉堵住门口：“你是做什么的？”
<br/>　　燕不归拔剑，一剑将大汉斩成两段。哇呀呀大叫，周围人四散逃开。燕不归近前一步，将那朱漆大门砍成稀巴烂。
<br/>　　抬起头，面对高高在上的大山。
<br/>　　“张六，出来。”
<br/>　　“啧。”程大雷叹了一声：“事情变得好玩了。”
<br/>　　这时候，卖曲的老汉已调好丝弦，他弹琵琶，那丫头击竹，声音倒是稚嫩好听。
<br/>　　『醉里挑灯看剑，梦回吹角连营』
<br/>　　丫头才唱了一句，偷眼打量程大雷，想要看他是否满意。
<br/>　　却忽然发现，程大雷不知何时已消失不见，只是桌上搁着一锭银子。
<br/>(本章完)
